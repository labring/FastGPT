### Fast GPT V4.6.7

1. 修改了知识库UI及新的导入交互方式。
2. 优化知识库和对话的数据索引，加快数据操作。

5. 修复 - API 对话时，chatId 冲突问题。
6. 修复 - Iframe 嵌入网页可能导致的 window.onLoad 冲突。
