### Fast GPT V4.6.8

1. 修改了知识库UI及新的导入交互方式。
2. 新增 - 知识库搜索合并模块。
3. 优化 - LLM 模型配置，不再区分对话、分类、提取模型。同时支持模型的默认参数，避免不同模型参数冲突，可通过`defaultConfig`传入默认的配置。
4. 优化 - HTTP 模块，支持输出字符串自动序列化（JSON可自动转成字符串）
5. 修复 - 语音输入文件无法上传。
6. 修复 - 对话框重新生成无法使用。
