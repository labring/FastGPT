# TD-2026041200016 修复文档

#### 【问题】
工作流完整响应弹窗中，知识库检索模块的 TOP2/TOP3 分块卡片，描述信息（综合分数、召回排名等）排版错乱，内容溢出到卡片内容区域。

#### 【问题原因】
`ChunkInfoCard.tsx` 的 Header Flex 高度固定为 `h={'40px'}`，但内部描述列表 Flex 设置了 `flexWrap={'wrap'}`，当综合分数、召回排名等多个描述项加上右侧文件名链接宽度超出一行时，描述列表触发换行，换行内容超出固定 40px 容器溢出显示，导致排版错乱。

同时，内层左侧 Flex `alignItems={'center'}` 在多行内容下会使图标和标题悬浮在描述列表高度的垂直中间，视觉对齐异常。

#### 【修复方案】
**文件**：`projects/app/src/components/core/chat/components/ChunkInfoCard.tsx`

| 行号 | 修改前 | 修改后 | 原因 |
|------|--------|--------|------|
| 61 | `h={'40px'}` | `minH={'40px'}` | 允许容器高度自适应内容，防止换行内容溢出 |
| 70 | `alignItems={'center'}` | `alignItems={'flex-start'}` | 多行时图标、标题、描述列表顶部对齐，视觉整洁 |

#### 【影响模块】
- `projects/app/src/components/core/chat/components/ChunkInfoCard.tsx`（直接修改）
- `projects/app/src/components/core/chat/components/AssistantDetailModal.tsx`（使用 ChunkInfoCard，正面受益）
- `projects/app/src/components/core/chat/ChatContainer/ChatBox/components/ChunkCardList.tsx`（使用 ChunkInfoCard，正面受益）
