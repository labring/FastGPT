## 技术栈规范
1. **项目框架**：基于 Next.js 开发，所有生成代码必须严格遵循：
   - Next.js 官方编码规范（ESLint+Prettier配置）
   - React 最佳实践（组件生命周期/状态管理）
   - 内存安全要求（避免闭包泄漏/事件监听未清理/定时器未清除、死循环）

2. **UI组件库**：
   - 使用 **Chakra UI v2.x** 作为唯一UI库
   - 严格遵循组件库 v2.x 版本API规范
   - 禁止混合使用其他UI库或原生HTML元素（Chakra未覆盖的除外）

## 国际化规范
### 支持语言版本
- 英文 (en)  - 新增词条使用中文即可。会有专门翻译同事完成翻译
- 简体中文 (zh-CN)
- 繁体中文 (zh-Hant)
