# FastGPT 代码索引

## 一、项目目录树（2层深度）

```
FastGPT/
├── .github/                  # CI/CD工作流自动化
├── .husky/                   # Git钩子
├── bin/                      # 可执行脚本
├── build/                    # 构建输出
├── deploy/                   # 部署配置（Docker、K8s）
│   ├── dev/                  # 开发环境docker-compose
│   └── docker/               # 生产部署配置
├── doc/                      # 知识库文档 [文档]
├── document/                 # 产品文档站点（NextJS）[文档]
├── packages/                 # 公共库
│   ├── diting-rag-ts/        # RAG向量库 [公共库]
│   ├── global/               # 全局常量、类型（跨端共享）[公共库]
│   ├── service/              # 后端服务核心业务库 [业务模块]
│   ├── templates/            # 应用模板库 [公共库]
│   └── web/                  # 前端共享组件库 [公共库]
├── plugins/                  # 外部插件（模型/爬虫等）[业务模块]
├── projects/                 # 独立应用项目
│   ├── app/                  # 主应用（Next.js前后端一体）[业务模块]
│   ├── agent-sandbox/        # Agent代码沙箱 [业务模块]
│   ├── code-sandbox/         # 代码执行沙箱（Bun+Hono）[业务模块]
│   ├── marketplace/          # 应用市场 [业务模块]
│   ├── mcp_server/           # Model Context Protocol服务器 [业务模块]
│   └── volume-manager/       # 卷存储管理 [业务模块]
├── scripts/                  # 构建和维护脚本
├── sdk/                      # 独立SDK
│   ├── logger/               # 日志SDK [公共库]
│   ├── storage/              # 存储SDK [公共库]
│   └── otel/                 # OpenTelemetry SDK [公共库]
├── test/                     # 集中测试目录 [测试]
├── package.json              # Monorepo根配置
├── vitest.config.mts         # 全局测试配置
├── Dockerfile.app            # 主应用Docker构建
└── Makefile                  # Make脚本
```
