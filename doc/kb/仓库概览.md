### 项目简介

FastGPT 是一个 AI Agent 构建平台，通过可视化工作流编排、知识库（RAG）和插件系统，帮助用户快速构建基于大语言模型的应用程序。支持多模型接入，提供 OpenAI 兼容的 API 接口。

### 技术栈

TypeScript + Next.js + React + Chakra UI + MongoDB + PostgreSQL (pgvector) + Milvus + Bun + Hono + LangGraph

### 仓库结构

pnpm workspaces monorepo，分为 packages（共享库）、projects（应用项目）、sdk、plugins、deploy、test、document、doc 等目录。

## 八、开发入门

### 环境要求
- Node.js >=20.x
- pnpm >=9.x
- 支持 MongoDB、PostgreSQL (pgvector) 或 Milvus 向量存储

### 基本命令

```bash
# 安装依赖
pnpm install

# 启动开发服务器（主应用）
cd projects/app && pnpm dev

# 构建生产版本
cd projects/app && pnpm build

# 运行所有测试
pnpm test

# 代码规范检查和自动修复
pnpm lint

# 初始化图标资源
pnpm initIcon

# 生成 Chakra UI 主题类型定义
pnpm gen:theme-typings
```

### 其他项目启动

```bash
# 启动代码沙箱（Bun）
cd projects/code-sandbox && pnpm dev

# 启动 MCP 服务器
cd projects/mcp_server && bun dev

# 启动产品文档
cd document && pnpm dev
```
