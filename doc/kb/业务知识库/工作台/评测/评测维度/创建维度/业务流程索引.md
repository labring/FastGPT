# 创建维度 — 业务流程索引

## 填写并创建评估维度

- **业务描述**: 用户填写评估维度的名称、描述和提示词，提交后创建一条新的评估维度记录，系统跳转回评测首页并切换到维度列表页。
- **触发入口**: 评测首页 → 维度列表 → 点击"创建维度"按钮
- **路由路径**: `/dashboard/evaluation/dimension/create`
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dimension/create.tsx
- **涉及角色**: admin, team_member
- **前置条件**: 用户已登录，具有评测模块访问权限

## 引用模板快速填充

- **业务描述**: 在填写提示词时，用户可通过"引用模板"按钮打开模板选择弹窗，从 8 种内置评估维度模板中选择一种，系统将对应的评分提示词模板填充到提示词输入框中。
- **触发入口**: 创建维度页 → 提示词区域 → 点击"引用模板"按钮
- **路由路径**: `/dashboard/evaluation/dimension/create`（同页面内弹窗）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/evaluation/dimension/CitationTemplate.tsx
- **涉及角色**: admin, team_member
- **前置条件**: 已在创建维度页面

## 试运行测试维度效果

- **业务描述**: 填写完毕表单后，用户可点击"试运行"按钮打开测试弹窗，输入测试问题、参考答案和实际回答，选择评估模型，运行测试查看维度在当前配置下的评分结果和反馈。
- **触发入口**: 创建维度页 → 页面底部 → 点击"试运行"按钮
- **路由路径**: `/dashboard/evaluation/dimension/create`（同页面内弹窗）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/evaluation/dimension/TestRun.tsx
- **涉及角色**: admin, team_member
- **前置条件**: 表单必填字段（名称、提示词）已填写有效，表单验证通过

## 场景关系

- **填写并创建评估维度** ← **引用模板快速填充**：引用模板是填写提示词的辅助操作，可缩短提示词编写时间
- **填写并创建评估维度** ← **试运行测试维度效果**：试运行是提交前的可选验证步骤，用于确认维度评分效果符合预期
