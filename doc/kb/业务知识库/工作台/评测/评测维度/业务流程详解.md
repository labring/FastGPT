## 公共能力概述

三个子能力共享以下公共能力，不在此分组节点展开：

- **表单组件**：EditForm 组件同时被创建和编辑页面使用，提供名称（必填）、描述（选填）、提示词（必填）三个字段的输入，支持实时校验和内置模板引用
- **试运行组件**：TestRun 组件同时被创建和编辑页面使用，提供问题、参考答案、实际回答的输入和评测模型选择，调用调试接口获取评分结果
- **API 层**：子能力共享 `@/web/core/evaluation/dimension.ts` 中的 6 个 API 函数（列表、详情、创建、更新、删除、调试）
