# 评测维度 — 业务流程索引

## 查看维度列表

- **业务描述**: 用户在评测首页切换到「评测维度」Tab，查看系统内置维度和团队自定义维度的完整列表，按名称或描述搜索过滤。
- **触发入口**: 工作台 → 评测 → 点击「评测维度」Tab
- **路由路径**: `/dashboard/evaluation?evaluationTab=dimensions`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**: 有评测权限的团队成员
- **前置条件**: `feConfigs.show_evaluation` 为 `true`，用户拥有评测访问权限

## 创建自定义维度

- **业务描述**: 用户在维度列表页点击「创建维度」按钮，填写维度名称、描述和评分提示词，可选择引用内置模板作为提示词起点，提交前可试运行验证评分效果。
- **触发入口**: 维度列表页 → 「创建维度」按钮
- **路由路径**: `/dashboard/evaluation/dimension/create`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/create.tsx`
- **涉及角色**: 有评测权限的团队成员
- **前置条件**: 已登录，有评测权限，维度名称和提示词均为必填

## 编辑自定义维度

- **业务描述**: 用户点击列表中某个自定义维度的行，进入编辑页面修改维度名称、描述或评分提示词，支持试运行验证修改后的效果。
- **触发入口**: 维度列表页 → 点击自定义维度行（内置维度不可点击编辑）
- **路由路径**: `/dashboard/evaluation/dimension/edit?id={dimensionId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/edit.tsx`
- **涉及角色**: 有评测权限的团队成员
- **前置条件**: 维度类型为自定义（Custom），`dimensionId` 有效，维度数据加载成功

## 删除自定义维度

- **业务描述**: 用户在维度列表中点击自定义维度行的删除图标，确认后删除该维度。
- **触发入口**: 维度列表页 → 点击行末删除图标 → 确认弹窗
- **路由路径**: `/dashboard/evaluation?evaluationTab=dimensions`（无页面跳转）
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`（内联操作）
- **涉及角色**: 有评测权限的团队成员
- **前置条件**: 维度类型为自定义，内置维度不可删除（行末无删除按钮）

## 试运行维度评分

- **业务描述**: 在创建或编辑维度时，用户点击「试运行」按钮，在弹窗中输入测试问题、参考答案和实际回答，选择评测模型后运行，查看维度评分结果。
- **触发入口**: 创建维度/编辑维度页 → 「试运行」按钮
- **路由路径**: 复用当前页面路由（弹窗形式）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dimension/TestRun.tsx`
- **涉及角色**: 有评测权限的团队成员
- **前置条件**: 表单必填字段（名称、提示词）已填写，已选择评测模型，问题/参考答案/实际回答均已输入

## 场景关系

- **查看维度列表** → **创建自定义维度**: 在列表页点击「创建维度」按钮跳转，创建成功后自动返回列表页
- **查看维度列表** → **编辑自定义维度**: 在列表页点击自定义维度行跳转，编辑成功后自动返回列表页
- **查看维度列表** → **删除自定义维度**: 在列表页行内操作，删除成功后刷新列表
- **创建/编辑维度** → **试运行维度评分**: 在表单页面点击「试运行」按钮打开弹窗，不跳转页面
- 创建、编辑、删除、试运行各场景独立，无严格先后依赖关系
