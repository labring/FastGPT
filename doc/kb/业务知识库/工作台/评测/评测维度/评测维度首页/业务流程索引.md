# 评测维度首页 — 业务流程索引

## 查看维度列表

- **业务描述**：进入评测维度页面后，系统自动加载并展示当前团队下所有的评测维度（包含内置维度和自定义维度），以表格形式呈现维度名、介绍、创建/更新时间、创建人等字段。
- **触发入口**：评测工作台 → 点击「评测维度」标签页
- **路由路径**：`/dashboard/evaluation?evaluationTab=dimensions`
- **页面组件位置**：`projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**：全部可访问评测功能的用户
- **前置条件**：用户已登录，评测功能开关 `show_evaluation` 为开启状态

## 搜索维度

- **业务描述**：在搜索框中输入关键词后，系统在前端对维度名称和介绍进行模糊匹配过滤，实时展示匹配结果。
- **触发入口**：评测维度列表页 → 顶部搜索框
- **路由路径**：`/dashboard/evaluation?evaluationTab=dimensions`
- **页面组件位置**：`projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**：全部可访问评测功能的用户
- **前置条件**：维度列表已加载完成

## 创建自定义维度

- **业务描述**：用户点击「新建」按钮后，页面跳转至维度创建页面，填写维度名称、介绍和提示词后保存，生成一个新的自定义评测维度。
- **触发入口**：评测维度列表页 → 顶部「新建」按钮
- **路由路径**：`/dashboard/evaluation/dimension/create`
- **页面组件位置**：`projects/app/src/pages/dashboard/evaluation/dimension/create.tsx`
- **涉及角色**：全部可访问评测功能的用户
- **前置条件**：用户已登录

## 编辑自定义维度

- **业务描述**：用户点击自定义维度所在行后，页面跳转至维度编辑页面，可修改维度的名称、介绍和提示词并保存。
- **触发入口**：评测维度列表页 → 点击自定义维度行
- **路由路径**：`/dashboard/evaluation/dimension/edit?id={dimensionId}`
- **页面组件位置**：`projects/app/src/pages/dashboard/evaluation/dimension/edit.tsx`
- **涉及角色**：全部可访问评测功能的用户
- **前置条件**：目标维度为自定义类型（非内置维度）

## 删除自定义维度

- **业务描述**：用户点击自定义维度行末的删除图标，在弹出的确认对话框中确认后，系统调用后端接口删除该维度并刷新列表。
- **触发入口**：评测维度列表页 → 自定义维度行末删除图标
- **路由路径**：`/dashboard/evaluation?evaluationTab=dimensions`（当前页操作）
- **页面组件位置**：`projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**：全部可访问评测功能的用户
- **前置条件**：目标维度为自定义类型，且通过删除确认

## 场景关系

- **查看维度列表** → **搜索维度**：搜索是列表的筛选操作，依赖列表数据已加载
- **查看维度列表** → **编辑自定义维度**：编辑通过点击列表中的维度行触发
- **查看维度列表** → **删除自定义维度**：删除通过点击列表中的删除图标触发
- **创建自定义维度** → **查看维度列表**：创建成功后需手动返回列表查看新维度（当前为路由跳转，创建页面为独立路由）
