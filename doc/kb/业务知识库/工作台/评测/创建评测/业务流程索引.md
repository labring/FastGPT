# 创建评测 — 业务流程索引

## 创建评测任务

- **业务描述**: 管理员选择一个已创建的应用，指定评测模型，上传按模板填写的 CSV 评测文件，创建一条评测任务。任务创建后自动跳转至评测首页。
- **触发入口**: 工作台侧边栏 → 评测 → 评测首页 → "新建任务"按钮
- **路由路径**: /dashboard/evaluation/create
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/create.tsx
- **涉及角色**: admin
- **前置条件**: 已登录且具有团队管理员权限；目标应用已存在

## 场景关系

创建评测任务是独立的原子操作，无上下游场景依赖。创建完成后跳转至评测首页查看任务列表。
