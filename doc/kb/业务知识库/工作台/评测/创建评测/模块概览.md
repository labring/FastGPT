# 创建评测 — 模块概览

## 1. 模块业务概述

创建评测是 FastGPT 应用评测体系的入口功能，为团队管理员提供创建评测任务的表单页面。用户在此页面选择待评测的应用、指定评测模型、上传符合模板的 CSV 评测文件，提交后系统将自动解析文件并执行评测流程。

本模块属于"工作台 → 评测"的子能力，是评测工作流的起点。评测任务创建成功后，用户将被引导至评测首页查看任务进度和结果。

本模块无子能力。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| admin | 创建评测页面 | 选择应用、下载模板、上传评测文件、创建评测任务 | 团队管理员可完整使用全部功能 |

**权限模型**: 基于 DashboardContainer 的团队权限控制，仅团队管理员可访问工作台下的评测相关页面。本页面无额外的角色判断逻辑，权限由容器层统一管控。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 应用管理 | 获取应用详情（含变量配置），用于生成评测 CSV 模板 | API |
| 评测模块 | 调用创建评测任务接口，上传 CSV 文件并提交任务参数 | API |
| 全局系统 | LLM 模型列表，用于评测模型选择器的数据源 | Store |
| 文件工具 | 浏览器端文件下载能力，用于下载 CSV 模板 | 工具函数 |
