# 评测任务首页 — 模块概览

## 1. 模块业务概述

评测任务首页是 FastGPT 评测系统的任务管理入口，提供评测任务的全生命周期管理视图。用户在此页面可以集中查看和搜索所有已创建的评测任务，了解每个任务的执行进度、评测结果和状态，并对任务执行重命名、删除、重试失败项等管理操作。

本模块服务于需要进行 AI 应用质量评估的团队成员和管理员，是"评测"大模块下的子能力页面。评测任务首页本身为单页列表视图，无子 Tab，新任务创建通过弹窗完成，任务详情通过行点击跳转至独立详情页。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| admin（管理员） | 评测任务首页、创建任务弹窗 | 查看所有任务、创建任务、重命名、删除、重试失败项 | 可操作团队内所有评测任务 |
| team_member（团队成员） | 评测任务首页、创建任务弹窗 | 查看权限范围内的任务、创建任务、重命名、删除、重试失败项 | 仅能查看和操作自己有权限的任务资源 |

**权限模型**: 基于团队（teamId）的资源隔离模型。任务列表 API 根据当前用户身份（是否为团队 owner）过滤可访问的任务：owner 可见全部任务，非 owner 仅可见分配给自己的任务和有权访问的资源。删除操作在后端校验团队归属。

## 4. 对外共享

本模块为叶节点页面组件，不对外提供共享能力。组件 `EvaluationTasks` 仅被父页面 `evaluation/index.tsx` 引用，属于同一模块组内部关系。
