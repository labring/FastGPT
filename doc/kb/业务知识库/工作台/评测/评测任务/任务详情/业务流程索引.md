# 任务详情 — 业务流程索引

## 1. 业务场景列表

### 查看评测结果

- **业务描述**: 进入评测任务详情页，浏览全部评测数据的评分结果及各维度的得分统计。
- **触发入口**: 评测任务列表 → 点击任务名称进入
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 用户已登录并属于该评测任务所在团队；评测任务已创建

### 过滤查看问题数据

- **业务描述**: 切换至"问题数据"标签页，筛选出评分低于阈值的评测项，快速定位需要关注的异常数据。
- **触发入口**: 详情页顶部 → 点击"问题数据"标签
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}&currentTab=questionData`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 统计数据显示存在低于阈值的数据项（标签仅在 `belowThreshold > 0` 时显示）

### 过滤查看错误数据

- **业务描述**: 切换至"错误数据"标签页，筛选出执行异常的评测项，可进行批量重试。
- **触发入口**: 详情页顶部 → 点击"错误数据"标签
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}&currentTab=errorData`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 统计数据显示存在错误数据项（标签仅在 `error > 0` 时显示）

### 搜索评测数据

- **业务描述**: 通过搜索框输入关键词，按用户问题内容筛选评测项列表。
- **触发入口**: 数据列表区域上方的搜索框
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测数据列表已加载

### 编辑评测数据

- **业务描述**: 对处于错误状态的评测项，修改问题和参考答案内容后重新保存，可选择是否同步修改数据集。
- **触发入口**: 选中错误状态的数据项 → 点击编辑按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 选中的数据项状态为"错误"（`EvaluationStatusEnum.error`）

### 重试单条评测

- **业务描述**: 对执行失败的单条评测项发起重新评测请求。
- **触发入口**: 选中错误状态的数据项 → 点击重试按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 选中的数据项状态为"错误"

### 删除评测数据

- **业务描述**: 从评测任务中删除单条评测数据，删除后自动选中下一条。
- **触发入口**: 选中数据项 → 点击删除按钮 → 确认弹窗
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 选中的数据项状态为"错误"或"已完成"

### 批量重试失败项

- **业务描述**: 在错误数据标签页下，一键重试所有执行失败的评测项，完成后自动切换到全部数据标签页。
- **触发入口**: 错误数据标签页 → 点击"重试"按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}&currentTab=errorData`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 当前位于错误数据标签页

### 导出评测数据

- **业务描述**: 将当前筛选条件下的评测数据导出为 CSV 文件，包含问题、参考答案、实际回答、各维度评分、状态等信息。
- **触发入口**: 详情页顶部 → 点击"导出"按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测任务已加载

### 刷新评分

- **业务描述**: 手动触发重新计算综合评分和各维度得分。
- **触发入口**: 右侧评分面板 → 点击刷新按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测任务不在排队或执行中状态

### 配置评分参数

- **业务描述**: 调整评测维度的判定阈值、权重，以及分数聚合方式（均值/中位数）。
- **触发入口**: 右侧评分面板 → 点击设置按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`（通过 ConfigParams 弹窗）
- **涉及角色**: 团队成员
- **前置条件**: 评测任务已加载完成

### 查看完整响应

- **业务描述**: 查看评测项对应的 AI 对话完整响应内容。
- **触发入口**: 选中已完成状态的数据项 → 点击"查看完整响应"按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={taskId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx`（通过 DetailedResponseModal 弹窗）
- **涉及角色**: 团队成员
- **前置条件**: 选中的数据项状态为"已完成"；数据项有关联的 AI 对话数据

## 2. 场景关系说明

- **查看评测结果** → **过滤查看问题数据/错误数据**: 用户先进入详情页查看全部数据，发现异常后切换到问题数据或错误数据标签进行针对性分析。
- **过滤查看错误数据** → **批量重试失败项**: 在错误数据标签下可一键重试所有失败项。
- **过滤查看错误数据** → **编辑评测数据** 或 **重试单条评测** 或 **删除评测数据**: 针对单条错误数据执行操作。
- **查看评测结果** → **导出评测数据**: 查看结果后可导出当前筛选条件下的数据。
- **查看评测结果** → **刷新评分** 或 **配置评分参数**: 查看评分后可能需要调整参数或刷新统计。

其他场景独立执行，无严格先后关系。
