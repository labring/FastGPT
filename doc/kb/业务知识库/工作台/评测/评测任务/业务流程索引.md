# 评测任务 — 业务流程索引

## 评测任务首页

- **业务描述**: 查看和管理所有评测任务，按应用筛选、按名称搜索任务，监控任务执行进度和评测结果
- **触发入口**: 工作台 → 评测 → 评测任务 Tab
- **路由路径**: /dashboard/evaluation/task
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/task/index.tsx
- **涉及角色**: 团队成员、团队管理员
- **前置条件**: 已登录，属于某个团队

## 创建评测任务

- **业务描述**: 创建新的评测任务，选择目标应用和版本、评测数据集、配置评测维度（内置维度或自定义维度），提交后任务进入排队状态等待执行
- **触发入口**: 评测任务首页 → "创建新任务"按钮
- **路由路径**: /dashboard/evaluation/task（弹窗形式）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/evaluation/task/CreateModal.tsx
- **涉及角色**: 团队管理员
- **前置条件**: 已选择目标应用，已有评测数据集，已配置评测维度

## 重命名评测任务

- **业务描述**: 修改已有评测任务的名称
- **触发入口**: 评测任务首页 → 任务行操作菜单 → "重命名"
- **路由路径**: /dashboard/evaluation/task
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/task/index.tsx
- **涉及角色**: 团队管理员
- **前置条件**: 任务存在

## 删除评测任务

- **业务描述**: 删除指定的评测任务及其全部评测结果数据
- **触发入口**: 评测任务首页 → 任务行操作菜单 → "删除"
- **路由路径**: /dashboard/evaluation/task
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/task/index.tsx
- **涉及角色**: 团队管理员
- **前置条件**: 任务存在，需在确认弹窗中再次确认

## 重试异常数据

- **业务描述**: 对任务中状态为异常的评测项重新提交评测请求
- **触发入口**: 评测任务首页 → 有异常数据的任务行操作菜单 → "重试异常数据"
- **路由路径**: /dashboard/evaluation/task
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/task/index.tsx
- **涉及角色**: 团队管理员
- **前置条件**: 任务存在异常数据项（error > 0）

## 查看任务详情

- **业务描述**: 进入单个评测任务的详情页面，查看全部评测数据的列表、各维度得分、综合评分、错误信息和评测总结报告
- **触发入口**: 评测任务首页 → 点击任务行
- **路由路径**: /dashboard/evaluation/task/detail
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/task/detail/index.tsx
- **涉及角色**: 团队成员、团队管理员
- **前置条件**: 任务存在

## 场景关系

- **评测任务首页** → **创建评测任务**：必须先有任务才能查看详情和执行评测
- **评测任务首页** → **查看任务详情**：从列表页点击进入详情
- **测评任务首页** → **重试异常数据**：任务执行完成且存在异常数据时可触发
- **任务详情** ↔ **重试异常数据**（详情页内）：在详情页的异常数据 Tab 中也可操作重试
