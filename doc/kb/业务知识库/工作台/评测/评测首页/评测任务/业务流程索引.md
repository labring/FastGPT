# 评测任务 — 业务流程索引

## 任务列表页

- **业务描述**: 管理员查看和管理所有评测任务，支持创建新任务、搜索、按应用筛选，以及单个任务的重命名、删除、重试失败数据等操作
- **触发入口**: 评测首页 → 评测任务 Tab（默认激活）
- **路由路径**: `/dashboard/evaluation?evaluationTab=tasks`
- **页面组件位置**: `src/pages/dashboard/evaluation/task/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 已登录，`feConfigs.show_evaluation` 为 true（否则重定向到工作台）

### 创建评测任务

- **业务描述**: 创建新的评测任务，选择目标应用和版本、评测数据集和评测维度，配置评测模型和向量模型
- **触发入口**: 任务列表页右上角「创建新任务」按钮
- **路由路径**: 无独立路由，以弹窗形式在列表页内打开
- **页面组件位置**: `src/pageComponents/dashboard/evaluation/task/CreateModal.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 已登录，有可选的 AI 应用和评测数据集

### 查看任务详情

- **业务描述**: 进入评测任务详情页，查看任务的基本信息、进度统计、各维度得分和评测总结，浏览逐条评测数据并进行编辑、删除、重试、导出等操作
- **触发入口**: 任务列表 → 点击任务行
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={id}`
- **页面组件位置**: `src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 任务已创建

### 编辑评测数据

- **业务描述**: 在任务详情页编辑单条评测数据的问题和期望回答，可同步修改数据集
- **触发入口**: 详情页 → 数据列表 → 选中数据 → 点击编辑按钮
- **路由路径**: `/dashboard/evaluation/task/detail?taskId={id}`（详情页内操作）
- **页面组件位置**: `src/pages/dashboard/evaluation/task/detail/index.tsx`（内联编辑）
- **涉及角色**: 管理员、团队成员
- **前置条件**: 任务详情已加载，选中的数据项存在

### 配置评分参数

- **业务描述**: 配置评测任务的分数聚合方式（均值/中位数）、各维度的判断阈值和综合评分权重
- **触发入口**: 详情页 → 右侧面板 → 设置按钮
- **路由路径**: 无独立路由，弹窗形式
- **页面组件位置**: `src/pageComponents/dashboard/evaluation/task/detail/ConfigParams.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 任务详情已加载，总结数据可用

### 导出评测数据

- **业务描述**: 将评测数据导出为 CSV 文件，包含用户输入、期望输出、实际输出、各维度得分等字段
- **触发入口**: 详情页 → 顶部导航栏 → 导出按钮
- **路由路径**: 调用 `/api/core/evaluation/task/item/export` API
- **页面组件位置**: 操作在 `src/pages/dashboard/evaluation/task/detail/index.tsx` 中触发
- **涉及角色**: 管理员、团队成员
- **前置条件**: 任务详情已加载，有可导出的数据

### 重试失败数据

- **业务描述**: 重试评测任务中所有状态为失败的数据项
- **触发入口**: 详情页 → 错误数据标签页 → 重试按钮
- **路由路径**: 调用 `/core/evaluation/task/retryFailed` API
- **页面组件位置**: `src/pages/dashboard/evaluation/task/detail/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 任务存在错误状态的数据项

## 场景关系

- **任务列表 → 创建任务**: 从列表页发起新任务创建
- **任务列表 → 任务详情**: 从列表页点击任务行进入详情
- **任务详情 → 编辑数据**: 在详情页选中数据后编辑
- **任务详情 → 配置参数**: 在详情页打开评分参数配置
- **任务详情 → 导出数据**: 在详情页导出当前筛选条件下的数据
- **创建任务 → 任务列表**: 创建成功后自动刷新列表
