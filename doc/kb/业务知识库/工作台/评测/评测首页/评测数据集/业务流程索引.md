# 评测数据集 — 业务流程索引

## 业务场景列表

### 查看数据集列表

- **业务描述**: 以分页表格形式浏览当前团队创建的所有评测数据集，支持名称搜索和状态概览。
- **触发入口**: 评测首页 → 切换到"评测数据集"Tab
- **路由路径**: `/dashboard/evaluation?evaluationTab=datasets`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 已登录且有团队，评测功能已开启（`feConfigs.show_evaluation` 不为 false）

### 智能生成数据集

- **业务描述**: 基于已有知识库，通过 AI 模型自动生成一批包含问题和参考答案的评测数据，组成新数据集。
- **触发入口**: 数据集列表页 → 点击"新建"按钮 → 选择"智能生成"
- **路由路径**: `/dashboard/evaluation?evaluationTab=datasets`（弹窗形式）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/IntelligentGeneration.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 存在可用的知识库，有评测模型可用

### 文件导入数据集

- **业务描述**: 通过下载 CSV 模板、填写数据后上传文件，批量创建数据集数据。
- **触发入口**: 数据集列表页 → 点击"新建"按钮 → 选择"文件导入"
- **路由路径**: `/dashboard/evaluation/dataset/fileImport`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/fileImport.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 准备好符合模板格式的 CSV 文件

### 管理数据集

- **业务描述**: 对已有数据集进行重命名或删除操作。
- **触发入口**: 数据集列表 → 行内操作菜单（⋮ 图标）
- **路由路径**: `/dashboard/evaluation?evaluationTab=datasets`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 存在已创建的数据集

### 查看异常详情

- **业务描述**: 当数据集状态为"生成异常"时，点击状态标签查看异常的详细任务列表，并可重试或删除单个/全部异常任务。
- **触发入口**: 数据集列表 → 点击状态为"生成异常"的状态标签
- **路由路径**: `/dashboard/evaluation?evaluationTab=datasets`（弹窗形式）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/errorModal.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 存在状态为异常的数据集

### 进入数据集详情

- **业务描述**: 进入数据集详情页，查看和管理数据集内包含的具体数据条目，执行质量评测等操作。
- **触发入口**: 数据集列表 → 点击数据集的表格行
- **路由路径**: `/dashboard/evaluation/dataset/detail?collectionId={id}&collectionName={name}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 数据集已创建成功

## 场景关系

- **查看数据集列表** → **智能生成数据集** / **文件导入数据集**: 列表为空或需要新数据集时，通过新建入口创建数据集。
- **查看数据集列表** → **进入数据集详情**: 点击列表行进入详情页查看和管理具体数据。
- **查看数据集列表** → **管理数据集**: 通过列表行内菜单进行重命名或删除。
- **查看数据集列表** → **查看异常详情**: 状态为异常时点击状态标签查看错误信息。
- **智能生成数据集** / **文件导入数据集** → **查看数据集列表**: 创建成功后自动刷新列表。
