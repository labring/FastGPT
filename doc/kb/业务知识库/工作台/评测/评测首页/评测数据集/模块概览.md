# 评测数据集 — 模块概览

## 1. 模块业务概述

评测数据集是 FastGPT 评测系统的数据管理模块，为团队成员提供评测数据集的创建、查看、管理和维护能力。数据集是应用评测任务的核心输入，包含一组用于测试 AI 应用的问题和参考答案，用户在创建评测任务时选择数据集作为测试用例。

模块提供两种数据集创建方式：**智能生成**（基于知识库 AI 自动生成评测数据）和**文件导入**（通过 CSV 模板批量导入数据），并支持对数据集进行重命名、删除等管理操作。数据集详情页进一步支持数据的增删改和质量评测。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队成员 | 数据集列表、数据集详情、文件导入 | 查看列表、创建数据集、重命名、删除、查看详情、追加数据 | 所有团队成员操作权限一致，无角色区分 |

**权限模型**: 本模块不单独设置权限位，依赖工作台 Dashboard 层级的团队准入控制。评测功能整体受 `feConfigs.show_evaluation` 开关控制，父级评测首页在入口处检查该配置，关闭时重定向至 Dashboard 首页。

## 4. 对外共享

本模块为评测系统的数据基础层，以下能力被其他模块消费：

| 共享能力 | 消费模块 | 能力类型 |
|---------|---------|---------|
| 评测数据集列表查询 API | 评测任务创建（选择数据集）、对话中评价数据加入数据集 | API |
| 智能生成数据集弹窗组件 | 评测任务创建、数据集详情页、对话评价数据集选择器 | 组件 |
| 数据集详情页上下文 | 数据集详情页内各子组件 | Context |
