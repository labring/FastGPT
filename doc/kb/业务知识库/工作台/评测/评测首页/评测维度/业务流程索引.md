# 评测维度 — 业务流程索引

## 查看维度列表

- **业务描述**: 在评测维度 Tab 中浏览系统内置维度和自定义维度的完整列表，查看每个维度的名称、描述、创建/更新时间、创建者信息。
- **触发入口**: 工作台 → 评测 → 切换到「评测维度」Tab
- **路由路径**: `/dashboard/evaluation?evaluationTab=dimensions`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 系统功能开关 `feConfigs.show_evaluation` 为 `true`（否则自动重定向到工作台首页）

## 搜索维度

- **业务描述**: 在搜索框中输入关键词，按维度名称或描述进行前端过滤，快速定位目标维度。
- **触发入口**: 维度列表页顶部搜索框
- **路由路径**: `/dashboard/evaluation?evaluationTab=dimensions`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 已进入维度列表页面

## 创建自定义维度

- **业务描述**: 填写维度名称、描述和评估提示词，创建一个新的自定义评估维度。支持引用内置模板快速填充提示词内容，创建前可测试运行维度效果。
- **触发入口**: 维度列表页右上角「创建维度」按钮
- **路由路径**: `/dashboard/evaluation/dimension/create`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/create.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 已进入维度列表页面

## 编辑自定义维度

- **业务描述**: 点击自定义维度行，进入编辑页面修改维度名称、描述和提示词，保存后更新维度配置。支持修改前测试运行查看效果。
- **触发入口**: 维度列表 → 点击自定义维度行（内置维度不可点击编辑）
- **路由路径**: `/dashboard/evaluation/dimension/edit?id={dimensionId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/edit.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 目标维度类型为自定义（非内置）

## 删除自定义维度

- **业务描述**: 删除一个不再需要的自定义评估维度。内置维度不支持删除。
- **触发入口**: 维度列表中自定义维度行的删除图标按钮
- **路由路径**: `/dashboard/evaluation?evaluationTab=dimensions`（弹窗确认，不跳转）
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 目标维度类型为自定义；用户确认删除弹窗

## 测试运行维度

- **业务描述**: 在创建或编辑维度页面中，使用示例问答数据测试维度的评估效果。输入问题、参考答案和实际回答，选择评估模型后运行，查看评分结果和反馈。
- **触发入口**: 创建/编辑维度页面中的「测试运行」按钮
- **路由路径**: `/dashboard/evaluation/dimension/create` 或 `/dashboard/evaluation/dimension/edit`（弹窗模式，不跳转）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dimension/TestRun.tsx`
- **涉及角色**: 管理员、团队成员
- **前置条件**: 已在创建或编辑维度页面；表单已通过基本验证（名称和提示词已填写）

## 场景关系

- **查看维度列表** → **搜索维度**: 在列表页中进行前端搜索过滤
- **查看维度列表** → **创建自定义维度**: 通过「创建维度」按钮跳转到创建页面
- **查看维度列表** → **编辑自定义维度**: 点击自定义维度行跳转到编辑页面
- **查看维度列表** → **删除自定义维度**: 通过列表中的删除按钮触发
- **创建/编辑自定义维度** → **测试运行维度**: 在创建或编辑页面中打开测试运行弹窗
- **创建自定义维度** 成功后 → 自动跳转回**维度列表**
- **编辑自定义维度** 保存后 → 自动跳转回**维度列表**
