## 对外暴露的 Hook/API

| 导出项 | 类型 | 用途 | 使用者 | 使用场景 |
|--------|------|------|--------|---------|
| `getBuiltinDimensionInfo` | 工具函数 | 根据维度名称查询内置维度的国际化显示信息 | 评测任务（CreateModal、ManageDimension、ConfigParams、EvaluationSummaryCard） | 将内置维度 ID 转换为用户可读的名称和描述 |
| `getBuiltinDimensionNameFromId` | 工具函数 | 从维度 ID 中提取内置维度名称（去除 `builtin_` 前缀） | 评测任务（CreateModal、ManageDimension）、评测维度列表 | 识别和解析内置维度 ID |
| `getMetricList` | API 函数 | 获取评估维度完整列表 | 评测任务（ManageDimension） | 在评测任务中选择维度时获取可选维度列表 |
| `getBuiltinDimensionIdFromName` | 工具函数 | 根据维度名称生成内置维度 ID（添加 `builtin_` 前缀） | 评测任务模块 | 构造内置维度的标准 ID |
| `getBuiltinDimensionEnglishInfo` | 工具函数 | 获取内置维度的英文名称和描述（用于下发给后台） | 评测任务模块 | 创建评测任务时需提交英文标识给后端 |
| `BUILTIN_DIMENSION_MAP` | 常量 | 内置维度信息映射表（含 name / description 的 i18n key） | 评测任务模块 | 配置内置维度的显示信息 |
