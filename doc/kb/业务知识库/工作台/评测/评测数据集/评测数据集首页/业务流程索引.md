# 评测数据集首页 — 业务流程索引

## 业务场景列表

### 查看数据集列表

- **业务描述**: 用户进入评测数据集首页，查看所有已创建的评测数据集，包括名称、数据量、创建/更新时间、状态和创建者信息。
- **触发入口**: 工作台 → 评测 → 评测数据集菜单
- **路由路径**: /dashboard/evaluation/dataset
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/index.tsx
- **涉及角色**: 团队成员
- **前置条件**: 用户已登录并有团队权限

### 搜索数据集

- **业务描述**: 用户通过数据集名称关键词搜索，系统在 500ms 防抖后自动发起筛选查询，列表实时更新匹配结果。
- **触发入口**: 数据集列表页顶部搜索框
- **路由路径**: /dashboard/evaluation/dataset
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/index.tsx
- **涉及角色**: 团队成员
- **前置条件**: 已在数据集列表页

### 智能生成数据集

- **业务描述**: 用户通过选择知识库作为数据源，配置生成模型和数据量，由 AI 自动生成评测数据集。
- **触发入口**: 数据集列表页 → "创建新数据集"按钮 → "智能生成"选项
- **路由路径**: /dashboard/evaluation/dataset（弹窗形式）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/evaluation/dataset/IntelligentGeneration.tsx
- **涉及角色**: 团队成员
- **前置条件**: 系统中有可用知识库；用户有权限访问知识库

### 文件导入数据集

- **业务描述**: 用户通过上传文件批量导入数据创建评测数据集，系统跳转到文件导入页面进行处理。
- **触发入口**: 数据集列表页 → "创建新数据集"按钮 → "文件导入"选项
- **路由路径**: /dashboard/evaluation/dataset/fileImport
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/fileImport.tsx
- **涉及角色**: 团队成员
- **前置条件**: 准备好转悠格式的数据文件

### 重命名数据集

- **业务描述**: 用户修改已有评测数据集的名称，提交后系统更新名称并刷新列表。
- **触发入口**: 数据集列表行 → 操作菜单（更多按钮）→ "重命名"
- **路由路径**: /dashboard/evaluation/dataset（内联弹窗）
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/index.tsx
- **涉及角色**: 团队成员
- **前置条件**: 数据集已存在

### 删除数据集

- **业务描述**: 用户删除不再需要的评测数据集，系统弹出确认对话框，确认后执行删除并刷新列表。
- **触发入口**: 数据集列表行 → 操作菜单（更多按钮）→ "删除"
- **路由路径**: /dashboard/evaluation/dataset
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/index.tsx
- **涉及角色**: 团队成员
- **前置条件**: 数据集已存在

### 查看数据集详情

- **业务描述**: 用户点击数据集行，跳转到数据集详情页查看数据项列表和进行数据管理。
- **触发入口**: 数据集列表行点击
- **路由路径**: /dashboard/evaluation/dataset/detail
- **页面组件位置**: projects/app/src/pages/dashboard/evaluation/dataset/detail/index.tsx
- **涉及角色**: 团队成员
- **前置条件**: 数据集已存在

### 查看异常任务

- **业务描述**: 当数据集状态为"生成异常"时，用户点击状态标签查看失败任务的详细错误信息，可对单个任务执行重试或删除操作。
- **触发入口**: 数据集列表 → 状态为"生成异常"的状态标签点击
- **路由路径**: /dashboard/evaluation/dataset（弹窗形式）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/evaluation/dataset/errorModal.tsx
- **涉及角色**: 团队成员
- **前置条件**: 数据集状态为 error（生成异常）

## 场景关系

- **查看数据集列表** → **搜索数据集**: 列表页内部操作，筛选已有列表
- **查看数据集列表** → **智能生成数据集** / **文件导入数据集**: 从列表页创建新数据集
- **查看数据集列表** → **查看数据集详情**: 点击列表行导航到详情页
- **查看数据集列表** → **重命名/删除数据集**: 行内操作，执行后刷新列表
- **查看数据集列表** → **查看异常任务**: 仅 error 状态的数据集可触发
