# 评测数据集 — 业务流程索引

## 评测数据集首页

- **业务描述**: 浏览和管理所有评测数据集，支持搜索、创建、重命名和删除操作
- **触发入口**: 工作台 → 评测 → 评测数据集 Tab
- **路由路径**: `/dashboard/evaluation?evaluationTab=datasets`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测功能可见（`feConfigs.show_evaluation !== false`）

## 文件导入

- **业务描述**: 通过上传 CSV 文件批量创建评测数据集，或向已有数据集追加数据
- **触发入口**: 评测数据集列表 → 创建数据集 → 文件导入（或数据集详情页 → 追加数据 → 文件导入）
- **路由路径**: `/dashboard/evaluation/dataset/fileImport`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/fileImport.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测功能可见；追加模式下需已有数据集

## 评测数据集详情

- **业务描述**: 查看数据集内的数据条目，支持编辑、删除、手动添加、智能追加、质量评估等操作
- **触发入口**: 评测数据集列表 → 点击数据集行
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/detail/index.tsx`
- **涉及角色**: 团队成员
- **前置条件**: 评测功能可见；需传入 `collectionId` 参数

## 场景关系

- **评测数据集首页** → **文件导入**: 用户通过列表页创建菜单进入文件导入页创建新数据集
- **评测数据集首页** → **评测数据集详情**: 用户点击列表中的数据集进入详情页
- **评测数据集详情** → **文件导入**: 用户通过详情页追加数据菜单以追加模式进入文件导入页
- **评测数据集首页** → **智能生成**: 用户通过创建菜单打开智能生成弹窗
- **评测数据集详情** → **智能生成**: 用户通过追加数据菜单打开智能生成弹窗
