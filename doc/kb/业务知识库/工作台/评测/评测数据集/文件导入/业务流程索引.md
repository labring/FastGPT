# 文件导入 — 业务流程索引

## 文件导入

- **业务描述**: 用户通过上传 CSV 文件，批量创建或追加评测数据集数据。支持在导入后自动触发质量评测。
- **触发入口**: 
  - 评测数据集首页 → 点击"导入文件"按钮 → 进入此页面（新建模式）
  - 评测数据集详情页 → 点击"追加数据" → 进入此页面（追加模式，携带 `collectionId`）
- **路由路径**: `/dashboard/evaluation/dataset/fileImport`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/fileImport.tsx`
- **涉及角色**: 具有评测创建权限的团队成员
- **前置条件**: 
  - 团队已开启评测功能
  - 用户具有 `hasEvaluationCreatePer` 权限
  - 追加模式下需目标数据集已存在

## 场景关系

- **文件导入（新建模式）** → **数据集详情**: 导入完成后自动跳转到数据集详情页
- **文件导入（追加模式）** → **数据集详情**: 导入完成后自动跳转回数据集详情页
- **文件导入（新建模式）** → **数据集列表**: 导入完成后可跳转到数据集列表

各场景按入口参数（`scene`）分流，互不交叉。
