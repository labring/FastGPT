# 评测数据集详情 — 业务流程索引

## 业务场景列表

### 查看数据集数据列表

- **业务描述**: 进入评测数据集详情页，查看该数据集下所有数据条目，通过滚动分页加载更多数据。
- **触发入口**: 评测数据集列表 → 点击某个数据集名称进入详情
- **路由路径**: `/dashboard/evaluation/dataset/detail?collectionId={id}&collectionName={name}`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/detail/index.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 已登录系统，已有评测数据集

### 搜索过滤数据

- **业务描述**: 在数据集详情页通过搜索框按关键字过滤数据条目，或通过状态下拉筛选器按质量状态筛选数据。
- **触发入口**: 数据集详情页顶部的搜索输入框或状态筛选下拉
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pages/dashboard/evaluation/dataset/detail/index.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 已在数据集详情页

### 编辑数据条目

- **业务描述**: 点击某条数据后打开编辑弹窗，修改该条数据的用户输入问题和期望输出回答。
- **触发入口**: 数据集详情页 → 点击某条数据卡片
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataList.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 数据集中有数据条目

### 删除数据条目

- **业务描述**: 在数据卡片悬停时显示删除按钮，点击后弹出确认弹窗，确认后删除该数据条目。
- **触发入口**: 数据集详情页 → 悬停数据卡片 → 点击删除图标
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataList.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 数据集中有数据条目

### 批量质量评估

- **业务描述**: 对该数据集中的全部数据条目发起批量质量评估，由评测模型自动评估每条数据的质量。
- **触发入口**: 数据集详情页 → 点击「质量评估」按钮 → 确认弹窗
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataListModals.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 数据集中有数据条目

### 追加数据 — AI 智能生成

- **业务描述**: 通过 AI 智能生成方式从知识库中为当前数据集追加新的评测数据。
- **触发入口**: 数据集详情页 → 点击「追加数据」按钮 → 选择「AI生成」
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataList.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 已配置知识库数据源

### 追加数据 — 文件导入

- **业务描述**: 通过上传文件将外部数据导入到当前数据集中。
- **触发入口**: 数据集详情页 → 点击「追加数据」按钮 → 选择「文件导入」
- **路由路径**: `/dashboard/evaluation/dataset/fileImport`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataList.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 有可导入的数据文件

### 追加数据 — 手动录入

- **业务描述**: 手动填写问题和参考回答，向当前数据集追加单条评测数据。
- **触发入口**: 数据集详情页 → 点击「追加数据」按钮 → 选择「手动录入」
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataList.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 无

### 配置评测模型

- **业务描述**: 在设置弹窗中为当前数据集选择用于质量评估的 AI 模型。
- **触发入口**: 数据集详情页 → 点击「设置」按钮
- **路由路径**: `/dashboard/evaluation/dataset/detail`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/detail/DataListModals.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 系统已配置可用的评测模型

## 场景关系

- **查看数据列表** → **编辑数据条目**: 通过点击数据卡片进入编辑
- **查看数据列表** → **搜索过滤数据**: 通过搜索框或筛选器实时过滤
- **查看数据列表** → **删除数据条目**: 悬停后点击删除
- **查看数据列表** → **批量质量评估**: 点击质量评估按钮（场景独立）
- **查看数据列表** → **追加数据**: 通过追加数据按钮打开子菜单（AI生成/文件导入/手动录入）
- **查看数据列表** → **配置评测模型**: 点击设置按钮（场景独立）

各场景之间无严格先后依赖关系，可独立执行。
