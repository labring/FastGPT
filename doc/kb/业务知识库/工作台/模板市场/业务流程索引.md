# 模板市场 — 业务流程索引

## 浏览模板市场

- **业务描述**: 用户进入模板市场页面，查看所有可用的预构建 AI 应用模板卡片列表，了解每个模板的名称、简介、作者和分类标签。
- **触发入口**: 工作台侧边栏 → 点击"模板市场"导航项
- **路由路径**: /dashboard/templateMarket
- **页面组件位置**: projects/app/src/pages/dashboard/templateMarket/index.tsx
- **涉及角色**: 所有登录用户
- **前置条件**: 用户已登录

## 按分类筛选模板

- **业务描述**: 用户点击顶部分类标签（如"对话"、"工具"、"知识库"等），模板卡片列表按所选分类过滤，仅展示属于该类别的模板。
- **触发入口**: 模板市场页面 → 点击分类标签 Tab
- **路由路径**: /dashboard/templateMarket
- **页面组件位置**: projects/app/src/pages/dashboard/templateMarket/index.tsx
- **涉及角色**: 所有登录用户
- **前置条件**: 已进入模板市场页面，模板标签数据已加载

## 搜索模板

- **业务描述**: 用户在搜索框中输入关键词，模板卡片列表实时按名称和简介进行模糊匹配过滤。
- **触发入口**: 模板市场页面 → 在搜索框中输入关键词
- **路由路径**: /dashboard/templateMarket
- **页面组件位置**: projects/app/src/pages/dashboard/templateMarket/index.tsx
- **涉及角色**: 所有登录用户
- **前置条件**: 已进入模板市场页面

## 使用模板创建应用

- **业务描述**: 用户点击模板卡片上的"使用"按钮，系统获取模板详情，解析工作流结构，创建新应用，成功后跳转到应用详情页。
- **触发入口**: 模板市场页面 → 点击模板卡片上的"使用"按钮
- **路由路径**: /dashboard/templateMarket → 创建成功后跳转 /app/detail?appId={id}
- **页面组件位置**: projects/app/src/pages/dashboard/templateMarket/index.tsx
- **涉及角色**: 所有登录用户
- **前置条件**: 已登录，有创建应用的权限

## 场景关系

- **浏览模板市场** → **按分类筛选模板**: 浏览是筛选的前置场景，筛选在浏览页面内完成
- **浏览模板市场** → **搜索模板**: 浏览是搜索的前置场景，搜索在浏览页面内完成
- **浏览模板市场** → **使用模板创建应用**: 使用模板是浏览的后续操作

筛选和搜索可组合使用（先筛选分类再搜索，或先搜索再筛选分类），二者无先后依赖。
