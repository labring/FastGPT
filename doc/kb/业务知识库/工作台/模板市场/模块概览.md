# 模板市场 — 模块概览

## 1. 模块业务概述

模板市场是 FastGPT 工作台中的应用模板中心，为用户提供预构建的 AI 应用模板浏览和使用功能。用户可以在模板市场中按类别筛选、搜索模板，并一键将模板创建为自己的应用，快速启动 AI 应用开发。

本模块是独立叶子页面，无子能力。模板数据由父容器 `DashboardContainer` 统一获取后通过 render props 注入，模板市场本身不负责数据拉取。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 所有登录用户 | 模板市场浏览页 | 浏览模板、按分类筛选、搜索模板、使用模板创建应用 | 基础功能，所有用户可用 |
| 企业微信团队用户 | 同上 | 同上 | 推荐分类标签名称变为"企业微信专区" |

**权限模型**: 模板市场页面通过侧边栏导航入口控制可见性，无独立页面级权限校验。模板使用操作依赖应用创建权限（后端校验）。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| DashboardContainer（工作台容器） | 提供模板列表数据、模板标签数据、侧边栏菜单图标 | 组件 |
| 应用 API 层 | 获取模板列表、模板详情、创建应用 | API |
| 应用详情页 | 使用模板创建应用成功后跳转到应用详情 | 页面路由 |
| SimpleApp 工具函数 | form2AppWorkflow：将简单应用表单转换为工作流结构 | 工具函数 |
| 埋点追踪 | webPushTrack：记录模板使用事件 | 工具函数 |

## 4. 对外共享

本模块为消费型页面，组件均为内部使用，无对外共享能力。`AppMarketCard` 仅在模板市场页面内部使用。
