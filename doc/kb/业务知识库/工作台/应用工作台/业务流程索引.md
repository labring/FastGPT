# 应用工作台 — 业务流程索引

## 场景关系

- **浏览应用列表** → **筛选/搜索应用** → **进入文件夹** → **查看文件夹详情** — 逐层深入的导航流程
- **浏览应用列表** → **创建应用** — 从列表页直接创建
- **浏览应用列表** → **导入 JSON 配置** — 通过菜单导入
- **浏览应用列表** → **从模板创建** — 跳转至模板市场
- **查看应用卡片** → **编辑信息 / 复制 / 删除 / 移动 / 导出 Skill / 权限管理** — 卡片右键菜单操作
- **查看应用卡片** → **进入应用详情** — 点击有写权限的应用卡片
- **查看应用卡片** → **进入聊天** — 点击只读应用或通过菜单"去聊天"

各场景独立执行，无严格先后关系。

## 业务场景列表

### 浏览和筛选应用列表

- **业务描述**: 用户进入应用工作台后，默认展示当前文件夹下的所有应用卡片，支持按应用类型筛选和按名称搜索。
- **触发入口**: 侧边栏 → "应用构建" → "应用"
- **路由路径**: `/dashboard/agent`
- **页面组件位置**: `projects/app/src/pages/dashboard/agent/index.tsx`
- **涉及角色**: 所有角色
- **前置条件**: 用户已登录，团队空间已初始化

### 文件夹导航

- **业务描述**: 用户点击文件夹卡片进入子文件夹，通过面包屑路径在各层级间导航，支持点击面包屑返回上级。
- **触发入口**: 应用列表中的文件夹卡片
- **路由路径**: `/dashboard/agent?parentId={folderId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/agent/index.tsx`（FolderPath 组件）
- **涉及角色**: 所有角色
- **前置条件**: 当前文件夹下存在子文件夹

### 查看文件夹详情

- **业务描述**: 进入文件夹后，右侧滑出文件夹详情面板，展示名称、描述、权限信息，支持编辑、移动、删除文件夹操作。
- **触发入口**: 进入任意文件夹
- **路由路径**: `/dashboard/agent?parentId={folderId}`
- **页面组件位置**: `projects/app/src/components/common/folder/SlideCard.tsx`
- **涉及角色**: 具有写权限的用户（编辑/删除）或所有者（权限管理）
- **前置条件**: 当前处于文件夹内部（parentId 非空）

### 创建应用

- **业务描述**: 用户通过"新建应用"按钮选择应用类型（智能问答/Chat Agent/工作流机器人），弹出创建弹窗选择模板并创建。
- **触发入口**: 应用列表页顶部"新建应用"下拉菜单 → 选择类型
- **路由路径**: `/dashboard/agent`（弹窗模式，不跳转）
- **页面组件位置**: `projects/app/src/pages/dashboard/create/CreateModal.tsx`
- **涉及角色**: 具有 hasAppCreatePer 权限的用户
- **前置条件**: 用户团队具有应用创建权限；若在文件夹内，需文件夹具有写权限且非 HTTP 插件类型文件夹

### 创建文件夹

- **业务描述**: 用户创建新文件夹用于组织应用，输入文件夹名称和描述后提交。
- **触发入口**: 应用列表页顶部"新建文件夹"按钮
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `@fastgpt/web/components/common/MyModal/EditFolderModal`
- **涉及角色**: 具有写权限的用户
- **前置条件**: 若在文件夹内，需该文件夹具有写权限

### 搜索应用

- **业务描述**: 用户在搜索框中输入应用名称，列表实时过滤匹配的应用。
- **触发入口**: 应用列表页顶部搜索框
- **路由路径**: `/dashboard/agent`（不跳转）
- **页面组件位置**: `@fastgpt/web/components/common/Input/SearchInput`
- **涉及角色**: 所有角色
- **前置条件**: 无

### 编辑应用信息

- **业务描述**: 用户通过应用卡片的菜单"编辑信息"修改应用的名称、头像、描述。
- **触发入口**: 应用卡片 → 更多菜单 → "编辑信息"
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `projects/app/src/components/common/Modal/EditResourceModal`
- **涉及角色**: 具有 hasWritePer 或 hasReadChatLogPer 权限的用户
- **前置条件**: 应用卡片菜单按钮可见（需有相应权限）

### 复制应用

- **业务描述**: 用户复制一个已有应用，系统创建副本并跳转至新应用的详情页。
- **触发入口**: 应用卡片 → 更多菜单 → "复制应用"
- **路由路径**: `/dashboard/agent` → 成功后跳转 `/app/detail?appId={newAppId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/agent/AppCard.tsx`（菜单触发）
- **涉及角色**: 具有写权限的用户
- **前置条件**: 应用类型非文件夹、MCP工具集、HTTP工具集、HTTP插件

### 删除应用/文件夹

- **业务描述**: 用户删除应用或文件夹，需输入确认文本。文件夹删除会级联删除所有子内容。
- **触发入口**: 应用卡片 → 更多菜单 → "删除"（红色危险按钮）
- **路由路径**: `/dashboard/agent`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/agent/AppCard.tsx`（菜单触发）
- **涉及角色**: 应用所有者
- **前置条件**: 工具类应用需关联应用数为 0；文件夹删除无引用限制但需确认

### 移动应用

- **业务描述**: 用户将应用/文件夹移动到其他文件夹中。
- **触发入口**: 应用卡片 → 更多菜单 → "移动到"
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `projects/app/src/components/common/folder/MoveModal`
- **涉及角色**: 具有管理权限的用户
- **前置条件**: 应用存在管理权限

### 导出应用为 Skill

- **业务描述**: 将 Agent 类应用导出为 Skill 压缩包文件，可用于分享或导入到其他团队。
- **触发入口**: 应用卡片 → 更多菜单 → "导出为 Skill"
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `projects/app/src/components/core/app/ExportSkillModal`
- **涉及角色**: 具有写权限的用户
- **前置条件**: 应用类型属于 Agent 类（非文件夹、非工具类）

### 导入 JSON 配置

- **业务描述**: 用户导入 JSON 格式的应用配置文件，快速创建工作流应用。若会话存储中有工作流 URL，页面加载时自动弹出导入弹窗。
- **触发入口**: "新建应用"下拉菜单 → "导入 JSON 配置"；或页面加载时自动触发
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/agent/JsonImportModal.tsx`
- **涉及角色**: 具有应用创建权限的用户
- **前置条件**: 用户团队具有应用创建权限

### 从模板创建

- **业务描述**: 用户跳转到模板市场，从模板库中选择模板来创建应用。
- **触发入口**: "新建应用"下拉菜单 → "从模板创建"
- **路由路径**: `/dashboard/agent` → 跳转到 `/dashboard/templateMarket`
- **页面组件位置**: 路由跳转
- **涉及角色**: 所有角色
- **前置条件**: 无

### 权限管理

- **业务描述**: 应用/文件夹所有者管理协作者及其角色权限，支持添加、修改、删除协作者。
- **触发入口**: 应用卡片 → 更多菜单 → "权限管理"；文件夹详情面板 → 权限设置
- **路由路径**: `/dashboard/agent`（弹窗模式）
- **页面组件位置**: `projects/app/src/components/support/permission/ConfigPerModal`
- **涉及角色**: 具有管理权限的用户
- **前置条件**: 应用/文件夹存在管理权限
