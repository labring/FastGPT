# 技能管理 — 业务流程索引

## 浏览技能列表

- **业务描述**: 在工作台技能管理页面浏览当前文件夹下的所有技能和子文件夹，支持卡片网格布局展示。
- **触发入口**: 工作台左侧导航 → 点击"技能管理" Tab
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx`
- **涉及角色**: 所有团队成员
- **前置条件**: 已登录且属于某个团队

## 搜索技能

- **业务描述**: 通过关键词搜索当前文件夹下的技能和文件夹，输入即触发搜索（500ms 防抖）。
- **触发入口**: 技能管理列表页顶部搜索框
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx`
- **涉及角色**: 所有团队成员
- **前置条件**: 已进入技能管理页面

## 进入子文件夹

- **业务描述**: 点击文件夹卡片进入其子目录，查看该文件夹内的技能和子文件夹，面包屑导航支持返回上级目录。
- **触发入口**: 技能管理列表 → 点击文件夹卡片
- **路由路径**: `/dashboard/skill?parentId={folderId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx`
- **涉及角色**: 所有团队成员
- **前置条件**: 当前目录存在文件夹

## 创建文件夹

- **业务描述**: 在当前目录下创建一个新的技能文件夹，用于分类管理技能。
- **触发入口**: 技能管理列表 → 点击"创建文件夹"按钮（仅具有技能创建权限的成员可见）
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 用户具有 `hasSkillCreatePer` 权限

## 创建技能

- **业务描述**: 创建新的自定义 Agent 技能，可设定名称、描述、图标等信息，创建后跳转到技能详情页进行编辑。
- **触发入口**: 技能管理列表 → 点击"创建技能"下拉菜单 → 选择"自定义技能"
- **路由路径**: `/dashboard/skill` → 创建成功后跳转 `/skill/detail?skillId={newSkillId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx` → `CreateSkillModal`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 用户具有 `hasSkillCreatePer` 权限

## 导入技能

- **业务描述**: 从 ZIP/TAR 压缩包导入技能，支持拖拽上传或点击选择文件，最大支持 50MB。
- **触发入口**: 技能管理列表 → 点击"创建技能"下拉菜单 → 选择"导入压缩包"
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx` → `ImportSkillModal`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 用户具有 `hasSkillCreatePer` 权限；上传文件为 .zip/.tar/.tar.gz 格式，大小不超过 50MB

## 进入技能详情

- **业务描述**: 点击技能卡片进入技能详情页查看和编辑技能配置（SKILL.md、文件树等）。
- **触发入口**: 技能管理列表 → 点击技能卡片
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pages/dashboard/skill/index.tsx` → 跳转到技能详情页（`projects/app/src/pages/skill/detail.tsx`）
- **涉及角色**: 所有团队成员
- **前置条件**: 技能存在

## 编辑技能信息

- **业务描述**: 修改个人技能的名称、头像和描述。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"编辑信息"（仅个人技能可见）
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 技能来源为个人（source=personal）

## 移动技能/文件夹

- **业务描述**: 将技能或文件夹移动到其他文件夹中，支持选择目标文件夹。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"移动到"
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 技能来源为个人

## 权限设置

- **业务描述**: 配置技能的协作者权限、转让所有者、设置权限继承等。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"权限设置"
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 技能所有者/具有管理权限的成员
- **前置条件**: 技能来源为个人

## 导出技能

- **业务描述**: 将技能的配置和文件打包为 ZIP 文件下载。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"导出配置"（仅非文件夹的技能可见）
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 技能来源为个人，且非文件夹类型

## 复制技能

- **业务描述**: 创建技能的完整副本。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"复制技能"（仅非文件夹的技能可见）
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 技能来源为个人，且非文件夹类型；操作需二次确认

## 删除技能/文件夹

- **业务描述**: 删除个人技能或文件夹。文件夹删除会递归删除所有子项；有应用引用的技能不可删除。
- **触发入口**: 技能卡片悬停 → 更多菜单 → 选择"删除"
- **路由路径**: `/dashboard/skill`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/List.tsx`
- **涉及角色**: 具有技能创建权限的成员
- **前置条件**: 技能来源为个人；技能未被任何应用引用（关联应用数=0）；操作需输入名称二次确认

## 场景关系

- **浏览技能列表** → **搜索技能**: 在列表页内随时使用搜索框过滤
- **浏览技能列表** → **进入子文件夹**: 点击文件夹卡片钻取子目录
- **浏览技能列表** → **创建文件夹**: 在当前目录创建后回到列表
- **浏览技能列表** → **创建技能**: 创建后跳转到技能详情页
- **浏览技能列表** → **导入技能**: 导入后刷新当前列表
- **浏览技能列表** → **进入技能详情**: 点击技能卡片跳转
- 编辑、移动、权限、导出、复制、删除操作均从技能卡片更多菜单触发，相互独立
