# 系统工具 — 业务流程索引

本模块为单页面工具体验，无 Tab 分页。以下为页面中可执行的业务场景。

## 浏览系统工具列表

- **业务描述**: 用户进入系统工具市场，查看所有可安装的系统预置工具，了解每个工具的名称、描述、作者等信息。
- **触发入口**: 工作台侧边栏 → 点击「工具」导航 → 选择「系统工具」标签
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录

## 按标签筛选工具

- **业务描述**: 用户按预定义的标签分类筛选系统工具，快速定位特定类型的功能工具。
- **触发入口**: 系统工具页面 → 点击标签筛选栏中的标签
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录，工具标签数据已加载

## 搜索工具

- **业务描述**: 用户通过搜索框输入关键词，按工具名称或简介进行模糊匹配，快速找到目标工具。
- **触发入口**: 系统工具页面 → 右上角搜索框输入
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录

## 安装系统工具

- **业务描述**: 用户将系统预置工具安装到团队工作区，安装后可在应用工作流中使用该工具。
- **触发入口**: 系统工具页面 → 工具卡片的「安装」按钮
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录，工具未安装

## 卸载系统工具

- **业务描述**: 用户卸载已安装的系统工具，从团队工作区移除该工具。
- **触发入口**: 系统工具页面 → 已安装工具卡片的「卸载」按钮
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录，工具已安装

## 查看工具详情

- **业务描述**: 用户点击工具卡片查看该工具的详细信息，包括工具版本、输入输出参数、使用说明（readme）等。
- **触发入口**: 系统工具页面 → 点击工具卡片
- **路由路径**: `/dashboard/systemTool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录

## 跳转工具配置（仅 root）

- **业务描述**: 当系统工具列表为空时，root 管理员可通过「点击配置」按钮跳转至工具配置页面进行工具配置。
- **触发入口**: 系统工具页面 → 工具列表为空时 → 「点击配置」按钮
- **路由路径**: `/config/tool`
- **页面组件位置**: `projects/app/src/pages/dashboard/systemTool/index.tsx`
- **涉及角色**: root管理员
- **前置条件**: 系统工具列表为空，当前用户为 root

## 场景关系

- **浏览系统工具列表** → **按标签筛选工具**：标签筛选在列表基础上进一步过滤
- **浏览系统工具列表** → **搜索工具**：搜索在列表基础上进行文本匹配过滤
- **浏览系统工具列表** → **查看工具详情**：点击工具卡片弹出详情抽屉
- **查看工具详情** → **安装/卸载系统工具**：详情抽屉中也可触发安装/卸载操作
- 标签筛选与搜索可叠加使用，进一步缩小结果范围
