# 创建应用 — 业务流程索引

## 创建 Agent 应用（空白模板）

- **业务描述**: 用户在应用工作台选择创建 Agent 类应用（Agent 对话助手/简单对话机器人/工作流编排/工作流工具），填写图标和名称后从空白模板创建。
- **触发入口**: 工作台页面 → 创建按钮 → 创建应用页面（Agent 类型）
- **路由路径**: `/dashboard/create?parentId={id}`
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx`
- **涉及角色**: 全体团队用户
- **前置条件**: 用户已登录并加入团队

## 创建工具应用（空白模板）

- **业务描述**: 用户在工具管理页面选择创建工具类应用（MCP 工具集/HTTP 工具集/工作流工具），配置相应参数后创建。
- **触发入口**: 工具管理页面 → 创建工具按钮 → 创建应用页面（工具类型）
- **路由路径**: `/dashboard/create?parentId={id}&appType={type}`
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx`
- **涉及角色**: 全体团队用户
- **前置条件**: 用户已登录并加入团队

## 创建 MCP 工具集

- **业务描述**: 选择 MCP 工具集类型，配置 MCP 服务地址和鉴权信息，解析并选择 MCP 工具列表后创建。
- **触发入口**: 创建应用页面 → 选择 MCP 工具集卡片
- **路由路径**: `/dashboard/create`（页面内交互）
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx` → MCP 配置区域
- **涉及角色**: 全体团队用户
- **前置条件**: 有可访问的 MCP 服务 URL

## 创建 HTTP 工具集

- **业务描述**: 选择 HTTP 工具集类型，选择批量或手动创建模式后创建。
- **触发入口**: 创建应用页面 → 选择 HTTP 工具集卡片
- **路由路径**: `/dashboard/create`（页面内交互）
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx` → HTTP 配置区域
- **涉及角色**: 全体团队用户
- **前置条件**: 无特殊前置条件

## 从快速模板创建

- **业务描述**: 浏览当前类型的快速模板列表，点击对应模板卡片一键创建应用。模板包含预设的工作流节点、边和对话配置。
- **触发入口**: 创建应用页面 → 模板区域 → 点击模板卡片
- **路由路径**: `/dashboard/create`（页面内交互）
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx` → 模板列表区域
- **涉及角色**: 全体团队用户
- **前置条件**: 当前类型有可用快速模板（由 `getTemplateMarketItemList` 返回）

## 智能生成工作流

- **业务描述**: 选择工作流类型后，用户输入业务需求描述（业务背景、核心流程、具体要求），选择 AI 模型，一键智能生成工作流应用。生成任务存入 sessionStorage，跳转至详情页后由 Copilot 执行。
- **触发入口**: 创建应用页面 → 选择工作流类型 → 智能生成区域
- **路由路径**: `/dashboard/create`（页面内交互）
- **页面组件位置**: `projects/app/src/pages/dashboard/create/index.tsx` → 智能生成区域
- **涉及角色**: 全体团队用户
- **前置条件**: 需选择工作流类型（`AppTypeEnum.workflow`），已配置 AI 模型

## 场景关系

- **创建 Agent 应用（空白模板）** 和 **创建工具应用（空白模板）** 是基础路径，所有类型共用图标/名称表单。
- **创建 MCP 工具集** 是工具创建的特殊分支，需要额外配置 MCP 服务地址和鉴权。
- **创建 HTTP 工具集** 是工具创建的特殊分支，需选择批量/手动模式。
- **从快速模板创建** 依赖模板市场 API，在 Agent 类和应用类类型下有不同模板列表。
- **智能生成工作流** 仅在工作流类型下可用，依赖 AI 模型选择器和业务需求描述。
- 所有创建路径成功后均跳转至 `/app/detail?appId={id}`。
