# 工具管理 — 业务流程索引

## 查看工具列表

- **业务描述**: 浏览当前文件夹下的所有工具，通过类型筛选和关键词搜索快速定位目标工具。列表采用无限滚动加载，支持分页获取。
- **触发入口**: 侧边导航栏 → 工作台 → 工具管理
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pages/dashboard/tool/index.tsx
- **涉及角色**: 全部角色
- **前置条件**: 已登录，具有团队访问权限

## 切换工具类型筛选

- **业务描述**: 按工具类型（全部/工作流工具/HTTP工具集/MCP工具集）筛选当前列表中的工具卡片。
- **触发入口**: 工具列表页 → 顶部类型筛选标签栏（MyTabBar）
- **路由路径**: /dashboard/tool?type={all|plugin|httpToolSet|toolSet}
- **页面组件位置**: projects/app/src/pages/dashboard/tool/index.tsx
- **涉及角色**: 全部角色
- **前置条件**: 已在工具列表页面

## 搜索工具

- **业务描述**: 通过名称关键词搜索工具，输入后自动触发服务端搜索（500ms 防抖），结果在列表中实时更新。
- **触发入口**: 工具列表页 → 顶部搜索框
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pages/dashboard/tool/index.tsx
- **涉及角色**: 全部角色
- **前置条件**: 已在工具列表页面

## 创建文件夹

- **业务描述**: 在当前位置下创建新文件夹，用于组织管理工具。需要填写文件夹名称和简介，支持上传自定义头像。
- **触发入口**: 工具列表页 → "新建文件夹"按钮
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pages/dashboard/tool/index.tsx（EditFolderModal）
- **涉及角色**: 团队所有者、管理员、编辑者（需有文件夹写入权限）
- **前置条件**: 当前文件夹有写入权限，或为根目录且有团队创建权限

## 创建工具

- **业务描述**: 创建新的 AI 工具，支持三种类型：工作流工具（基于可视化工作流编排）、HTTP 工具集（配置 HTTP 请求参数调用外部 API）、MCP 工具集（连接 MCP 服务器并选择工具）。工作流工具支持从模板市场创建或空白创建。
- **触发入口**: 工具列表页 → "新建工具"下拉菜单 → 选择工具类型
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pages/dashboard/create/ToolModal.tsx
- **涉及角色**: 团队所有者、管理员、编辑者（需有文件夹写入权限）
- **前置条件**: 当前文件夹有写入权限，或为根目录且有团队创建权限

## 编辑工具/文件夹信息

- **业务描述**: 修改工具或文件夹的名称、简介、头像。
- **触发入口**: 工具卡片 → 更多操作菜单 → 编辑；或文件夹详情面板 → 编辑按钮
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pages/dashboard/tool/index.tsx（EditFolderModal）、projects/app/src/pageComponents/dashboard/agent/List.tsx（EditResourceModal）
- **涉及角色**: 团队所有者、管理员、编辑者（需有该资源的编辑权限）
- **前置条件**: 对目标工具有编辑权限

## 移动工具/文件夹

- **业务描述**: 将工具或文件夹移动到另一个文件夹中。
- **触发入口**: 工具卡片 → 更多操作菜单 → 移动；或文件夹详情面板 → 移动按钮
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pageComponents/dashboard/agent/context.tsx（MoveModal）
- **涉及角色**: 团队所有者、管理员、编辑者
- **前置条件**: 对目标资源有管理权限

## 删除工具/文件夹

- **业务描述**: 删除指定的工具或文件夹，删除前弹出确认提示。删除后自动清理关联的本地存储数据，并跳转回上级文件夹。
- **触发入口**: 工具卡片 → 更多操作菜单 → 删除；或文件夹详情面板 → 删除按钮
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pageComponents/dashboard/agent/List.tsx（DelConfirmModal）、projects/app/src/pages/dashboard/tool/index.tsx（PopoverConfirm）
- **涉及角色**: 团队所有者（文件夹删除）；资源拥有者或有管理权限的用户
- **前置条件**: 对目标资源有管理权限

## 管理协作者权限

- **业务描述**: 为文件夹或工具添加/移除协作者，设置各协作者的权限级别（只读/编辑/管理），或恢复权限继承。
- **触发入口**: 文件夹详情面板 → 协作者区域 → 设置按钮
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/components/common/folder/SlideCard.tsx（ConfigPerModal）
- **涉及角色**: 团队所有者、管理员
- **前置条件**: 对目标资源有管理权限，团队启用了 Plus 功能

## 复制工具

- **业务描述**: 创建工具的完整副本，包含工具的所有配置。
- **触发入口**: 工具卡片 → 更多操作菜单 → 复制
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pageComponents/dashboard/agent/List.tsx（ConfirmCopyModal）
- **涉及角色**: 团队所有者、管理员、编辑者
- **前置条件**: 对目标工具有读取权限

## 导入 JSON 配置

- **业务描述**: 从 JSON 配置文件导入工作流创建新工具。若会话存储中存在工作流 URL 链接，页面加载时自动触发导入弹窗。
- **触发入口**: "新建工具"下拉菜单 → "导入 JSON 配置"
- **路由路径**: /dashboard/tool
- **页面组件位置**: projects/app/src/pageComponents/dashboard/agent/JsonImportModal.tsx
- **涉及角色**: 团队所有者、管理员、编辑者
- **前置条件**: 需要有创建权限

## 导航到文件夹

- **业务描述**: 点击文件夹路径中的某一级面包屑，导航到对应层级查看该文件夹下的工具。
- **触发入口**: 顶部面包屑导航路径 → 点击某级文件夹名称
- **路由路径**: /dashboard/tool?parentId={folderId}
- **页面组件位置**: projects/app/src/components/common/folder/Path.tsx
- **涉及角色**: 全部角色
- **前置条件**: 已进入某个文件夹

## 场景关系

- **查看工具列表** → **切换工具类型筛选** / **搜索工具** / **导航到文件夹**：筛选、搜索和文件夹导航均为列表的过滤方式，可组合使用
- **查看工具列表** → **创建工具** / **创建文件夹** / **导入 JSON 配置**：创建操作从列表页触发，成功后自动刷新列表
- **查看工具列表** → **编辑/移动/删除/复制**：针对单个工具或文件夹的操作，编辑成功后自动刷新列表
- **创建工具（工作流）** → **应用详情**：创建成功后自动跳转到应用详情页进行工作流编排
