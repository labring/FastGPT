# MCP 服务器 — 业务流程索引

## 业务场景列表

MCP 服务器模块是单页面应用，所有操作均在 `/dashboard/mcpServer` 路由下完成，无 Tab 子结构。

### 查看 MCP 服务器列表

- **业务描述**: 进入页面后查看当前团队已创建的所有 MCP 服务器，每行展示服务器名称、关联应用数量和操作入口。
- **触发入口**: 左侧导航栏「应用构建」→「MCP 服务器」
- **路由路径**: /dashboard/mcpServer
- **页面组件位置**: projects/app/src/pages/dashboard/mcpServer/index.tsx
- **涉及角色**: 所有团队成员
- **前置条件**: 用户已登录

### 创建 MCP 服务器

- **业务描述**: 创建一个新的 MCP 服务器，配置名称并从已有应用中选择关联的应用工具，可自定义工具名称和描述。
- **触发入口**: MCP 服务器列表页顶部「创建 MCP 服务器」按钮
- **路由路径**: /dashboard/mcpServer（弹窗内操作）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/mcp/EditModal.tsx
- **涉及角色**: 有 API 密钥创建权限的团队成员
- **前置条件**: 用户具有 `hasApikeyCreatePer` 权限

### 编辑 MCP 服务器

- **业务描述**: 修改已有 MCP 服务器的名称和关联应用列表，可增删工具并修改工具名称和描述。
- **触发入口**: MCP 服务器列表每行的「编辑」按钮
- **路由路径**: /dashboard/mcpServer（弹窗内操作）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/mcp/EditModal.tsx
- **涉及角色**: 有 API 密钥创建权限的团队成员
- **前置条件**: 用户具有 `hasApikeyCreatePer` 权限；已存在 MCP 服务器

### 删除 MCP 服务器

- **业务描述**: 删除指定的 MCP 服务器，删除前弹出确认提示。
- **触发入口**: MCP 服务器列表每行的「删除」按钮
- **路由路径**: /dashboard/mcpServer（Popover 确认窗内操作）
- **页面组件位置**: projects/app/src/pages/dashboard/mcpServer/index.tsx
- **涉及角色**: 有 API 密钥创建权限的团队成员
- **前置条件**: 用户具有 `hasApikeyCreatePer` 权限；已存在 MCP 服务器

### 查看使用方式

- **业务描述**: 查看 MCP 服务器的连接端点 URL 和 JSON 配置，可在 Streamable HTTP 和 SSE 两种协议间切换，支持一键复制端点地址和配置。
- **触发入口**: MCP 服务器列表每行的「开始使用」按钮
- **路由路径**: /dashboard/mcpServer（弹窗内展示）
- **页面组件位置**: projects/app/src/pageComponents/dashboard/mcp/usageWay.tsx
- **涉及角色**: 所有团队成员
- **前置条件**: 已存在 MCP 服务器

## 场景关系

- **创建 MCP 服务器** → **查看使用方式**: 创建成功后通常需要获取端点信息以配置外部客户端
- **创建 MCP 服务器** → **编辑 MCP 服务器**: 创建后可继续修改应用配置和工具描述
- **查看 MCP 服务器列表** → 所有其他操作: 列表页是所有操作的入口
