# 账户 — 模块概览

## 1. 模块业务概述

账户模块是 FastGPT 的用户个人中心与团队管理入口。它集中提供了用户管理个人账号信息、查看用量统计、管理团队、处理账单发票、配置第三方集成、管理 API 密钥以及设置通知偏好等功能。

本模块面向**所有已登录用户**，根据用户角色和系统配置展示不同的功能选项卡。团队管理员（owner/admin）拥有团队管理、账单查看和用量统计的全部权限；普通成员（member）仅可查看个人相关信息。

模块内部通过统一的侧边栏导航（PC 端）或顶部选项卡（移动端）组织 11 个子能力页面，所有子页面共享 `AccountContainer` 布局组件。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| owner | 全部 11 个页面 | 全部操作（含团队管理、删除成员、管理密钥、查看账单、配置推广、域名管理） | 最高权限，拥有所有可见页面的完整操作权限 |
| admin | 用量统计、账号信息、团队管理、账单、第三方集成、模型管理、API 密钥、通知、个性化设置 | 管理团队成员、查看账单、配置模型、管理密钥 | 不能查看推广记录和执行敏感操作（如解散团队） |
| member | 账号信息、第三方集成、模型管理、API 密钥、通知、个性化设置 | 查看个人信息、配置第三方集成、管理个人密钥、设置偏好 | 不能查看团队管理、账单、用量统计、推广和自定义域名 |

**权限模型**: 基于 `TeamPermission`（含 `hasManagePer`、`hasWritePer`、`isOwner`、`hasApikeyCreatePer` 等字段）的权限位模型，由后端在用户登录时注入到 `userInfo.team.permission` 中。部分选项卡的可见性还受 `feConfigs` 功能开关控制（如 `isPlus`、`show_pay`、`show_promotion`、`customDomain.enable` 等）。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 共享组件库（@fastgpt/web） | MyIcon 图标、Avatar 头像、MyModal 弹窗、MyBox 容器、LightRowTabs 选项卡 | 组件 |
| 用户状态管理（useUserStore） | 用户信息、团队信息、权限位、初始化登录 | Store |
| 系统配置（useSystemStore） | feConfigs 功能开关、系统级配置 | Store |
| 账单/钱包模块 | 订阅套餐展示、价格格式化、充值弹窗 | 组件/Hook |
| i18n 国际化 | 多语言翻译（account、account_thirdParty、account_usage 等命名空间） | Hook |

## 4. 对外共享

| 共享能力 | 消费模块 | 能力类型 |
|---------|---------|---------|
| AccountContainer 账户布局 | 所有 11 个账户子页面 | 组件 |
| ApiKeyTable API 密钥表格 | 应用发布页面（Publish/API） | 组件 |
| 团队选择器（TeamSelector） | 账号信息页面内部使用 | 组件 |
