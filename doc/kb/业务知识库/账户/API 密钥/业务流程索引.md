# API 密钥 — 业务流程索引

## 查看和管理 API 密钥

- **业务描述**：用户进入 API 密钥页面，查看已有密钥列表，可执行创建、编辑、删除密钥等管理操作，也可快速复制 API 基础地址和查看调用示例。
- **触发入口**：账户页→左侧/顶部导航点击「API 密钥」Tab。
- **路由路径**：`/account/apikey`
- **页面组件位置**：projects/app/src/pages/account/apikey.tsx
- **涉及角色**：团队管理员、团队成员（需要 hasApikeyCreatePer 权限）
- **前置条件**：已登录 FastGPT；团队权限中 hasApikeyCreatePer 为 true；账户 Tab 导航中可见「API 密钥」入口。

## 场景关系

各场景独立执行，无严格先后关系。创建密钥后可进行编辑和删除操作。
