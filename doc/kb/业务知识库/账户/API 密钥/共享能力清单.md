## 共享组件与 API 的消费详情

### ApiKeyTable 被复用时行为差异

| 属性 | 账户场景（apikey.tsx） | 应用发布场景（Publish/API） |
|------|----------------------|--------------------------|
| appId | 不传（undefined） | 传入具体 appId |
| API 密钥范围 | 账户下全部应用的密钥 | 仅指定应用的密钥 |
| 调用示例可用性 | 不可用（无 appId） | 可用（有 appId 时可获取应用详情） |
| tips 内容 | "API 密钥管理提示"（i18n: account_apikey:key_tips） | 默认为空或应用相关提示 |
