# API 密钥 — 模块概览

## 1. 模块业务概述

API 密钥模块是 FastGPT 账户体系下的核心安全功能，为用户提供应用对外 API 访问的鉴权凭证管理。用户在此模块中创建、编辑、删除 API 密钥（API Key），并通过密钥对外暴露 FastGPT 应用的对话能力。该模块同时提供 API 基础地址复制和调用示例展示功能，帮助开发者快速集成。

API 密钥被广泛应用于应用发布中的「API 访问」渠道，一个良好管理的 API 密钥体系是应用对外开放的安全基础。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队管理员（Owner/Admin） | API 密钥管理页面 | 创建密钥、编辑密钥、删除密钥、复制 API 基础地址、查看调用示例 | 完整管理权限，可配置高级功能（用量上限、过期时间） |
| 团队普通成员 | API 密钥管理页面（如有 hasApikeyCreatePer 权限） | 创建密钥、编辑密钥、删除密钥、复制 API 基础地址、查看调用示例 | 操作权限与管理员一致，但可能无法配置 Plus 版高级功能 |

**权限模型**：基于团队权限位 `hasApikeyCreatePer` 控制整个 API 密钥模块的可见性。Tab 入口在账户侧边栏/顶部导航中根据 `userInfo.team.permission.hasApikeyCreatePer` 条件渲染。Plus 版本的功能（用量上限、过期时间）由 `feConfigs.isPlus` 控制。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 应用（Publish/API） | API 密钥表格组件被应用发布 API 渠道复用 | 组件 |
| 账户容器 | AccountContainer 提供统一的账户页面布局和 Tab 导航 | 组件 |
| 系统配置（useSystemStore） | feConfigs 系统特性开关和自定义 API 域名配置 | Store |

| 公共 Hooks（useRequest/useCopyData/useConfirm） | 异步请求封装、剪贴板复制、确认弹窗 | Hook |
