# 自定义域名 — 模块概览

## 1. 模块业务概述

自定义域名模块为团队所有者提供将 AI 应用绑定到自有域名的能力。用户无需使用 FastGPT 默认提供的对话链接，而是可以通过自己的域名（如 `chat.example.com`）对外发布 AI 应用，提升品牌一致性和专业形象。

本模块是账户管理下的一个子功能，仅对 Plus（商业版）用户开放。团队所有者在本模块中可完成域名的添加、DNS 解析验证、激活和删除等全生命周期管理。支持的 DNS 提供商包括阿里云、腾讯云和火山引擎。

本模块为叶子节点（无子能力），所有域名管理操作集中在单一列表页面完成。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队所有者 | 自定义域名列表 | 添加域名、编辑域名、删除域名、激活域名 | 仅团队所有者可访问，需 Plus 订阅且套餐包含自定义域名配额 |
| 普通成员 | 无 | 无 | 自定义域名入口在账户侧边栏中不显示 |

**权限模型**: 基于套餐订阅的权限控制。入口可见性由 `feConfigs.customDomain.enable` 控制；操作权限由 `teamPlanStatus.standard.customDomain > 0` 控制。未订阅用户看到升级引导页面，引导跳转到价格页面。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 账户（AccountContainer） | 账户侧边栏布局与导航容器 | 组件 |
| 用户状态（useUserStore） | 团队订阅计划状态查询 | Store |
| 系统配置（useSystemStore） | Plus 功能开关与 DNS 提供商域名配置 | Store |
| 价格 | 未订阅用户跳转到价格页面升级套餐 | 页面 |

## 4. 对外共享

本模块未对外暴露共享能力。自定义域名的 API 函数（`listCustomDomain`、`deleteCustomDomain`、`createCustomDomain` 等）仅在账户模块内使用，不被其他模块引用。
