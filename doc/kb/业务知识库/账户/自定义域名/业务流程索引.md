# 自定义域名 — 业务流程索引

## 业务场景列表

### 查看自定义域名列表

- **业务描述**: 团队所有者进入自定义域名管理页面，查看已绑定的域名列表及其状态、提供商、CNAME 记录等信息。
- **触发入口**: 账户管理侧边栏 → "自定义域名" 标签
- **路由路径**: `/account/customDomain`
- **页面组件位置**: `projects/app/src/pages/account/customDomain/index.tsx`
- **涉及角色**: 团队所有者
- **前置条件**: Plus 订阅已激活，套餐包含自定义域名配额

### 添加自定义域名

- **业务描述**: 团队所有者选择一个 DNS 提供商，输入要绑定的自定义域名，系统生成 CNAME 记录。用户在自己的 DNS 提供商处配置 CNAME 解析后，系统自动检测 DNS 解析状态，解析成功后确认创建。
- **触发入口**: 自定义域名列表页 → "添加" 按钮
- **路由路径**: `/account/customDomain`（弹窗模式）
- **页面组件位置**: `projects/app/src/pages/account/customDomain/index.tsx` → 动态加载 `createModal.tsx`
- **涉及角色**: 团队所有者
- **前置条件**: 当前域名数量未达到套餐配额上限

### 编辑/激活自定义域名

- **业务描述**: 对于状态为"未激活"的域名，团队所有者可重新进入编辑流程，再次检测 DNS 解析状态后执行激活操作。
- **触发入口**: 自定义域名列表中未激活域名的 "编辑" 按钮
- **路由路径**: `/account/customDomain`（弹窗模式，复用创建弹窗）
- **页面组件位置**: `projects/app/src/pages/account/customDomain/index.tsx` → 动态加载 `createModal.tsx`（refresh 模式）
- **涉及角色**: 团队所有者
- **前置条件**: 域名状态为 inactive（未激活）

### 删除自定义域名

- **业务描述**: 团队所有者删除不再需要的自定义域名绑定。
- **触发入口**: 自定义域名列表中每条域名的 "删除" 按钮 → 确认弹窗
- **路由路径**: `/account/customDomain`
- **页面组件位置**: `projects/app/src/pages/account/customDomain/index.tsx`
- **涉及角色**: 团队所有者
- **前置条件**: 无

### 升级套餐（未订阅时）

- **业务描述**: 未订阅 Plus 或套餐不含自定义域名配额的团队所有者，在自定义域名页面看到升级引导提示，点击跳转到价格页面进行套餐升级。
- **触发入口**: 自定义域名列表页的空状态提示 → "升级套餐" 按钮
- **路由路径**: `/account/customDomain` → 跳转 `/price`
- **页面组件位置**: `projects/app/src/pages/account/customDomain/index.tsx`
- **涉及角色**: 团队所有者
- **前置条件**: 当前套餐不含自定义域名功能

## 场景关系

- **查看列表** → **添加域名**: 用户在列表中点击"添加"按钮发起添加流程，添加成功后列表自动刷新。
- **添加域名** → **编辑/激活**: 添加域名时若 DNS 未解析成功，域名状态为 inactive，用户可后续通过编辑功能重新检测 DNS 并激活。
- **查看列表** → **删除域名**: 用户在列表中直接删除任意域名。
- **查看列表** → **升级套餐**: 未订阅用户进入页面即看到升级引导。
