# detail — 业务流程索引

## 业务场景列表

### 用量记录列表

- **业务描述**: 以分页表格形式查看团队/个人的用量消耗流水记录，支持多维度筛选和导出。
- **触发入口**: 账户 → 用量统计 → 切换至「用量明细」Tab
- **路由路径**: `/account/usage?usageTab=detail`
- **页面组件位置**: `projects/app/src/pageComponents/account/usage/UsageTable.tsx`
- **涉及角色**: 团队管理员、普通成员
- **前置条件**: 用户已登录并加入团队；需有对应团队的数据访问权限

### 用量详情

- **业务描述**: 查看单条用量记录的计费模块明细，包括各模块的模型、Token 消耗、文本长度、时长等计费维度的具体数值。
- **触发入口**: 用量记录列表 → 点击某条记录的「详情」按钮
- **路由路径**: `/account/usage?usageTab=detail`（弹窗内，不改变路由）
- **页面组件位置**: `projects/app/src/pageComponents/account/usage/UsageDetail.tsx`
- **涉及角色**: 团队管理员、普通成员
- **前置条件**: 用量记录列表已加载且有数据

### 导出用量记录

- **业务描述**: 将当前筛选条件下的全部用量记录导出为 CSV 文件，方便外部数据分析。
- **触发入口**: 用量记录列表 → 点击「导出」按钮
- **路由路径**: `/account/usage?usageTab=detail`（触发下载，不改变路由）
- **页面组件位置**: `projects/app/src/pageComponents/account/usage/UsageTable.tsx`
- **涉及角色**: 团队管理员、普通成员
- **前置条件**: 当前筛选条件下存在用量数据（导出前会提示总数确认）

## 场景关系

- **用量记录列表** → **用量详情**：列表为详情提供入口，详情依赖列表中的单条数据
- **用量记录列表** → **导出用量记录**：导出操作的筛选条件与列表当前筛选状态同步

各场景独立触发，列表查看、详情查看、导出无严格先后依赖关系（导出前需列表已加载，但导出确认弹窗有取消选项）。
