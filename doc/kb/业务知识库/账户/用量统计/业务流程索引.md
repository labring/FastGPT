# 用量统计 — 业务流程索引

## 查看用量看板

- **业务描述**: 以折线图形式展示选定时间范围内的积分消耗趋势和 Token（输入/输出/总计）消耗趋势，顶部汇总显示总积分和总 Token 数
- **触发入口**: 账户中心 → 用量统计（默认进入看板 Tab）
- **路由路径**: `/account/usage?usageTab=dashboard`
- **页面组件位置**: `projects/app/src/pages/account/usage/index.tsx` + `projects/app/src/pageComponents/account/usage/Dashboard.tsx`
- **涉及角色**: 普通成员、管理员
- **前置条件**: 已登录；管理员可按成员/群组/部门筛选查看不同维度的数据

## 查看用量明细

- **业务描述**: 以表格形式展示每次 AI 调用的详细记录，包含时间、成员、来源、项目名称、积分消耗、Token 消耗。点击「详情」可查看该次调用的计费模块明细
- **触发入口**: 账户中心 → 用量统计 → 切换到「用量明细」Tab
- **路由路径**: `/account/usage?usageTab=detail`
- **页面组件位置**: `projects/app/src/pages/account/usage/index.tsx` + `projects/app/src/pageComponents/account/usage/UsageTable.tsx`
- **涉及角色**: 普通成员、管理员
- **前置条件**: 已登录

## 导出用量记录

- **业务描述**: 将当前筛选条件下的用量数据导出为 CSV 文件下载
- **触发入口**: 用量明细 Tab → 点击「导出」按钮 → 确认导出
- **路由路径**: `/account/usage?usageTab=detail`
- **页面组件位置**: `projects/app/src/pageComponents/account/usage/UsageTable.tsx`
- **涉及角色**: 普通成员、管理员
- **前置条件**: 已进入用量明细 Tab；当前筛选条件下有数据

## 筛选用量数据

- **业务描述**: 通过日期范围、用量来源、成员/群组/部门（管理员专属）等维度筛选用量数据
- **触发入口**: 用量统计页面顶部的筛选器区域
- **路由路径**: `/account/usage`
- **页面组件位置**: `projects/app/src/pages/account/usage/index.tsx`
- **涉及角色**: 普通成员（仅日期范围和来源筛选）、管理员（额外有成员/群组/部门筛选）
- **前置条件**: 管理员权限方可使用成员/群组/部门筛选

## 查看剩余积分

- **业务描述**: 快捷查看账户剩余积分，可直接跳转充值
- **触发入口**: 用量统计页面右上角「查看剩余积分」按钮
- **路由路径**: `/account/usage`
- **页面组件位置**: 两个 Tab 内均提供此入口
- **涉及角色**: 普通成员、管理员
- **前置条件**: 无

## 场景关系

- **筛选用量数据** → **查看用量看板** / **查看用量明细**: 筛选条件在两个 Tab 间共享，切换 Tab 时保留筛选状态
- **查看用量明细** → **查看用量详情**: 在明细列表中点击某条记录的「详情」按钮，弹出详情弹窗
- **查看用量明细** → **导出用量记录**: 筛选数据后导出
