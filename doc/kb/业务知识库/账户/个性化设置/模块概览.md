# 个性化设置 — 模块概览

## 1. 模块业务概述

个性化设置模块为登录用户提供个人偏好配置功能，允许用户自定义语言和时区两项基本设置。该模块位于"账户"侧边栏导航中，以"语言与时区"为入口标签，是所有登录用户均可访问的基础设置页。

模块的核心价值在于：用户可以根据自身习惯选择界面显示语言，并设定所在时区以确保系统中时间相关信息的正确展示。设置变更后立即生效，无需刷新页面。

本模块无子能力，为单一叶子页面。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 所有登录用户 | 个性化设置页 | 修改语言、修改时区 | 无角色差异，所有登录用户权限一致 |

**权限模型**: 页面访问由登录状态控制，未登录用户将被重定向至登录页。页面本身无额外角色限制。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 用户 Store (useUserStore) | 读取当前用户信息、调用 updateUserInfo 保存设置 | Store |
| I18nLngSelector 组件 | 语言选择下拉框（含语言切换逻辑） | 组件 |
| TimezoneSelect 组件 | 时区选择下拉框（含全球时区列表） | 组件 |
| AccountContainer 组件 | 账户页外层容器（含侧边栏导航和 Tab 切换） | 组件 |
| 用户 API (putUserInfo) | 调用 PUT /support/user/account/update 持久化设置 | API |

## 4. 对外共享

本模块为叶子页面，不对外提供可复用的组件、API 或 Hook。
