# 渠道日志 — 业务流程索引

## 业务场景列表

### 渠道日志查询

- **业务描述**: root 管理员查看通过 AI 代理渠道转发的模型请求历史记录，支持按渠道、模型、请求状态和日期范围筛选。
- **触发入口**: 账户管理 → 模型管理 → 点击「渠道日志」Tab
- **路由路径**: /account/model（父路由，Tab 切换至 channel_log）
- **页面组件位置**: projects/app/src/pageComponents/account/model/Log/index.tsx
- **涉及角色**: root
- **前置条件**: 当前用户为 root 且系统配置 `feConfigs.show_aiproxy` 为 true

### 渠道日志详情查看

- **业务描述**: 在日志列表中点击某条记录的「详情」按钮，查看该次请求的完整信息，包括请求 ID、IP、EndPoint、状态码、耗时、Token 消耗、TTFB 时间以及请求/响应体。
- **触发入口**: 渠道日志列表 → 点击某条记录的「详情」按钮
- **路由路径**: /account/model（父路由，弹窗覆盖）
- **页面组件位置**: projects/app/src/pageComponents/account/model/Log/index.tsx（LogDetail 组件）
- **涉及角色**: root
- **前置条件**: 已在渠道日志列表中

## 场景关系

- **渠道日志查询** → **渠道日志详情查看**: 日志查询是详情查看的前置操作，用户需先在列表中定位目标记录，再点击详情按钮查看。
