# 活跃模型 — 共享能力清单

## 消费的外部能力

本模块依赖以下外部模块/包提供的能力：

| 消费能力 | 来源模块/包 | 能力类型 | 使用方式 |
|---------|-----------|---------|---------|
| 模型列表数据（含各类型模型、供应商） | useSystemStore（系统全局 Store） | Store | 通过 `getModelProviders`、`getModelProvider`、`llmModelList` 等字段获取 |
| 用户信息和团队权限 | useUserStore（用户全局 Store） | Store | 通过 `userInfo.team.permission.hasManagePer` 等字段判断角色 |
| FillRowTabs 标签切换组件 | @fastgpt/web 通用组件库 | 组件 | 用于基座模型/自定义模型 Tab 切换 |
| MyModal 弹窗容器 | @fastgpt/web 通用组件库 | 组件 | 训练弹窗、默认模型弹窗的基础容器（dynamic import） |
| MySelect 下拉选择器 | @fastgpt/web 通用组件库 | 组件 | 供应商筛选、模型类型筛选下拉 |
| SearchInput 搜索输入框 | @fastgpt/web 通用组件库 | 组件 | 模型关键词搜索 |
| useTableMultipleSelect 多选 Hook | @fastgpt/web 通用 Hooks | Hook | 模型列表批量选择功能 |
| useRequest 请求 Hook | @fastgpt/web 通用 Hooks | Hook | 训练详情、知识库树等异步数据获取 |
| LazyCollaboratorProvider 协作者权限配置 | 权限管理模块（components/support/permission） | 组件 | 批量权限配置弹窗的协作者选择器 |
| AIModelSelector AI 模型选择器 | AI 模型选择器模块（components/core/ai） | 组件 | 默认模型弹窗中各类型的模型下拉选择 |
| 训练 API（embedding/reRank） | web/core/app/api/train | API | 创建训练任务、获取训练任务列表 |
| 知识库 API（getDatasetsWithChildren） | web/core/app/api/dataset | API | 获取团队知识库树 |
| 系统 API（updateModelCollaborators、putUpdateDefaultModels） | web/common/system/api | API | 更新模型协作者、更新默认模型配置 |
| 全局数据刷新（clientInitData） | web/common/system/staticData | 工具函数 | 训练/权限/默认模型变更后刷新全局缓存 |

## 对外共享能力

本模块向外部提供以下能力：

| 共享能力 | 消费方 | 能力类型 | 说明 |
|---------|--------|---------|------|
| ModelTable 完整模型表格 | Points（积分页面）、ValidModelTable（有效模型表） | 页面组件 | 以 `permissionConfig` prop 控制是否显示权限管理列 |
| ModelPriceModal 模型价格弹窗 | 外部调用方（通过 children render prop） | 弹窗组件 | 以 render prop 模式提供，调用方通过 `children({ onOpen })` 控制触发 |
| ModelReferenceModal 模型引用提示 | AddModelBox（添加模型弹窗）、ModelConfigTable（模型配置表） | 弹窗组件 | 当模型权限更新被现有引用阻止时展示引用详情 |
| useModelTableState 状态管理 Hook | ModelTable 组件内部使用 | Hook | 封装 Tab 切换、筛选、排序、多选、弹窗控制等全部状态逻辑 |
| 模型过滤/排序/选项生成工具函数 | ModelTable 系列组件 | 工具函数 | `getFilteredModelList`、`getSortedModelList`、`getProviderOptions`、`getFilteredProviderList`、`getModelTypeOptions` |
