# 活跃模型 — 业务流程索引

## 业务场景列表

### 查看基座模型列表

- **业务描述**: 在「活跃模型」页面的基座模型 Tab 中，查看系统中所有官方提供的基座模型，按模型类型和供应商筛选，按训练任务数量排序。
- **触发入口**: 账户 → 模型管理 → 活跃模型 Tab（默认激活）
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/index.tsx`（ModelTable，activeTab='base'）
- **涉及角色**: root、团队管理员、团队成员
- **前置条件**: 用户已登录团队账户

### 查看自定义模型列表

- **业务描述**: 切换到自定义模型 Tab，查看团队自行训练的微调模型列表，支持按训练时间排序和训练详情查看。
- **触发入口**: 活跃模型页面 → 点击「自定义模型」Tab
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/index.tsx`（ModelTable，activeTab='custom'）
- **涉及角色**: root、团队管理员、团队成员
- **前置条件**: 用户已登录团队账户

### 筛选和排序模型

- **业务描述**: 通过供应商筛选、模型类型筛选、关键词搜索三个维度快速定位目标模型；可对训练任务数量或训练时间进行升降序排序。
- **触发入口**: 模型列表顶部的筛选栏
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/ModelFilterBar.tsx`、`projects/app/src/components/core/ai/ModelTable/ModelListTable.tsx`
- **涉及角色**: root、团队管理员、团队成员
- **前置条件**: 已在基座模型或自定义模型 Tab 中

### 查看训练详情

- **业务描述**: 点击自定义模型的「训练详情」按钮，右侧弹出抽屉展示该模型的训练任务列表和历史记录。
- **触发入口**: 自定义模型列表行 → 操作列 → 训练详情按钮
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/TrainDetailDrawer.tsx`
- **涉及角色**: root、团队管理员、团队成员
- **前置条件**: 当前处于自定义模型 Tab；该模型为可训练类型（embedding/reRank）

### 创建模型训练任务

- **业务描述**: 选择一个基座模型，指定训练数据集和新模型名称，提交训练任务生成微调模型。
- **触发入口**: 基座模型或自定义模型列表行 → 操作列 → 训练按钮；训练详情抽屉 → 创建训练按钮
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/BaseModelTrainModal.tsx`
- **涉及角色**: root、团队管理员
- **前置条件**: 当前用户有管理权限；所选基座模型支持训练（supportTrain）；有可用的知识库数据集

### 批量配置模型权限

- **业务描述**: 勾选多个有管理权限的模型后，底部浮出批量操作栏，可统一修改所选模型的团队协作权限。
- **触发入口**: 勾选模型复选框 → 底部浮出操作栏 → 权限配置按钮
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/BatchPermissionActionBar.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 权限配置模式开启（permissionConfig=true）；选中了至少一个有管理权限的模型

### 设置默认模型

- **业务描述**: root 用户点击「默认模型」按钮，在弹出的对话框中为各类型（LLM/TTS/Embedding/STT/ReRank）指定系统默认使用的模型。
- **触发入口**: 活跃模型页面右上角 → 默认模型按钮（仅 root 可见）
- **路由路径**: `/account/model`
- **页面组件位置**: `projects/app/src/components/core/ai/ModelTable/DefaultModelModal.tsx`
- **涉及角色**: root
- **前置条件**: 当前登录用户为 root（username === 'root'）

## 场景关系

- **查看基座模型列表** → **创建模型训练任务**: 在基座模型列表中选择一个模型进行训练。
- **查看自定义模型列表** → **查看训练详情**: 在自定义模型列表中点击训练详情，查看该模型的训练记录。
- **查看训练详情** → **创建模型训练任务**: 在训练详情抽屉中可继续发起新的训练任务。
- **查看基座/自定义模型列表** → **筛选和排序模型**: 筛选和排序贯穿于两个 Tab 的列表浏览过程。
- **筛选和排序模型** → **批量配置模型权限**: 筛选出目标模型后，批量选择进行权限配置。
- 各场景独立执行，无严格先后关系。
