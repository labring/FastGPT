# 审计日志 — 业务流程索引

## 查看审计日志

- **业务描述**: 团队管理员查看团队内所有成员的操作行为记录，支持按成员和事件类型筛选，用于安全审计和问题追溯。
- **触发入口**: 团队管理页面 → 点击"审计日志"标签页
- **路由路径**: `/account/team?teamTab=audit`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/Audit/index.tsx`
- **涉及角色**: 团队管理员（hasManagePer）
- **前置条件**: 管理员已登录并加入团队；当前团队套餐支持审计日志存储功能（auditLogStoreDuration 有值）

## 筛选审计日志

- **业务描述**: 在审计日志列表上通过下拉多选筛选器按成员或事件类型筛选日志记录，筛选条件变化时自动刷新列表。
- **触发入口**: 审计日志页面 → 点击成员筛选器或事件类型筛选器
- **路由路径**: `/account/team?teamTab=audit`（不改变路由，本地状态筛选）
- **页面组件位置**: `projects/app/src/pageComponents/account/team/Audit/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 已进入审计日志页面；成员列表已加载；事件类型列表可用

## 场景关系

- **查看审计日志** → **筛选审计日志**: 筛选是日志查看过程中的交互操作，不是独立场景，依附于查看场景。
- **筛选审计日志** → **查看审计日志**: 筛选结果即时更新到列表中。

> 各场景独立执行，无严格先后关系。筛选操作是日志查看场景的增强交互。
