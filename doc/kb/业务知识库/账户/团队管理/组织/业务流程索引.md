# 组织 — 业务流程索引

## 业务场景列表

### 查看组织架构

- **业务描述**: 用户进入组织管理 Tab 后，首先看到当前层级下的子组织和直属成员列表，通过左侧边栏可以查看当前选中组织的详细信息。
- **触发入口**: 团队管理页 → 点击「组织」Tab
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 用户已登录且属于某个团队

### 层级导航与浏览

- **业务描述**: 用户点击组织列表中的某个组织进入下级，通过面包屑路径返回上级；也可在树状侧边栏中选择目标组织查看其详情和操作。
- **触发入口**: 组织列表 → 点击组织名称
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 当前组织层级下存在子组织

### 搜索组织

- **业务描述**: 用户通过搜索框输入关键词，按组织名称筛选当前团队下的组织。
- **触发入口**: 组织管理页 → 顶部搜索框
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 无

### 创建子组织

- **业务描述**: 团队管理员在当前选中组织下创建一个新的子级组织，填写组织名称、头像和描述信息。
- **触发入口**: 组织管理页 → 右侧边栏 → 点击「创建子组织」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx` → `OrgInfoModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 当前已选中一个组织（非同步成员模式）

### 编辑组织信息

- **业务描述**: 团队管理员修改组织的名称、头像或描述信息。
- **触发入口**: 组织列表中组织行 → 更多菜单 →「编辑信息」；或右侧边栏 → 编辑图标
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/OrgInfoModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 目标组织非根组织（path 不为空）

### 移动组织

- **业务描述**: 团队管理员将某个组织移动到另一个父组织下，改变其在层级树中的位置。
- **触发入口**: 组织列表中组织行 → 更多菜单 →「移动」；或右侧边栏 →「移动组织」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/OrgMoveModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 目标组织非根组织；存在其他可选父组织

### 删除组织

- **业务描述**: 团队管理员删除一个组织（需二次确认）。
- **触发入口**: 组织列表中组织行 → 更多菜单 →「删除」；或右侧边栏 →「删除组织」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 目标组织非根组织（path 不为空）

### 管理组织成员

- **业务描述**: 团队管理员在弹窗中从全团队成员列表中选择成员加入当前组织，或从组织中移除已选成员。
- **触发入口**: 右侧边栏 →「管理成员」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/OrgMemberManageModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 当前已选中一个组织（非同步成员模式）

### 从组织中移除成员

- **业务描述**: 团队管理员将某个成员从当前组织中移除（不删除该成员，仅取消组织归属）。
- **触发入口**: 成员列表 → 成员行 → 更多菜单 →「从组织移除」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 非同步成员模式

### 从团队移除成员

- **业务描述**: 团队管理员直接将某个成员从团队中彻底移除（删除团队归属）。
- **触发入口**: 成员列表 → 成员行 → 更多菜单 →「从团队移除」
- **路由路径**: `/account/team?teamTab=org`
- **页面组件位置**: `projects/app/src/pageComponents/account/team/OrgManage/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 无（两种成员模式均可）

## 场景关系

- **查看组织架构** → **层级导航与浏览**: 用户在查看组织列表后，通过点击组织名称深入下级查看
- **层级导航与浏览** → **创建子组织/编辑组织/移动组织/删除组织/管理成员**: 管理员选中目标组织后，通过侧边栏操作触发管理功能
- **搜索组织** 与其他场景独立，搜索后直接定位到目标组织
- 创建、编辑、移动、删除、管理成员等操作互相独立，可单独执行
