# 账户 — 业务流程索引

## 业务场景列表

### 查看用量统计

- **业务描述**: 查看团队的 AI 用量消耗情况，支持按时间范围、成员/部门/群组维度和使用来源进行筛选，提供仪表盘和明细两种视图。
- **触发入口**: PC 端左侧导航栏 → 用量统计；移动端顶部选项卡 → 用量统计
- **路由路径**: `/account/usage`
- **页面组件位置**: `pages/account/usage/index.tsx`
- **涉及角色**: owner, admin
- **前置条件**: 用户具有团队管理权限（hasManagePer）；系统为 Plus 版本（feConfigs.isPlus）

### 管理账号信息

- **业务描述**: 查看和编辑个人信息（头像、名称），查看当前订阅套餐详情，管理优惠券和兑换码，修改密码和联系方式。
- **触发入口**: PC 端左侧导航栏 → 账号信息；移动端顶部选项卡 → 账号信息
- **路由路径**: `/account/info`
- **页面组件位置**: `pages/account/info/index.tsx`
- **涉及角色**: owner, admin, member
- **前置条件**: 用户已登录

### 管理团队

- **业务描述**: 查看团队成员列表，邀请新成员加入，管理成员角色和权限，处理成员退出。
- **触发入口**: PC 端左侧导航栏 → 团队管理；移动端顶部选项卡 → 团队管理
- **路由路径**: `/account/team`
- **页面组件位置**: `pages/account/team/index.tsx`
- **涉及角色**: owner, admin
- **前置条件**: 系统为 Plus 版本（feConfigs.isPlus）

### 管理账单与发票

- **业务描述**: 查看历史订单列表，查看账单详情，管理发票抬头和开票信息。
- **触发入口**: PC 端左侧导航栏 → 账单；移动端顶部选项卡 → 账单
- **路由路径**: `/account/bill`
- **页面组件位置**: `pages/account/bill/index.tsx`
- **涉及角色**: owner, admin
- **前置条件**: 系统开启支付功能（feConfigs.show_pay）；用户具有管理权限（hasManagePer）

### 配置第三方集成

- **业务描述**: 配置 Laf 云开发账户、OpenAI 账户和工作流外部变量，查看各集成的用量配额。
- **触发入口**: PC 端左侧导航栏 → 第三方集成；移动端顶部选项卡 → 第三方集成
- **路由路径**: `/account/thirdParty`
- **页面组件位置**: `pages/account/thirdParty/index.tsx`
- **涉及角色**: owner, admin, member
- **前置条件**: 用户已登录；非 owner 角色编辑时提示无权限

### 管理自定义域名

- **业务描述**: 为应用添加自定义域名，配置 DNS 解析和 CNAME 记录，激活域名。
- **触发入口**: PC 端左侧导航栏 → 自定义域名；移动端顶部选项卡 → 自定义域名
- **路由路径**: `/account/customDomain`
- **页面组件位置**: `pages/account/customDomain/index.tsx`
- **涉及角色**: owner, admin
- **前置条件**: 系统为 Plus 版本且启用了自定义域名功能（feConfigs.customDomain.enable）

### 管理模型

- **业务描述**: 查看和配置 AI 模型供应商，管理模型列表，自定义训练模型，设置模型权限。
- **触发入口**: PC 端左侧导航栏 → 模型管理；移动端顶部选项卡 → 模型管理
- **路由路径**: `/account/model`
- **页面组件位置**: `pages/account/model/index.tsx`
- **涉及角色**: owner, admin, member
- **前置条件**: 用户已登录

### 查看推广记录

- **业务描述**: 查看推广邀请统计（邀请人数、收益金额），复制邀请链接，查看推广收益明细。
- **触发入口**: PC 端左侧导航栏 → 推广；移动端顶部选项卡 → 推广
- **路由路径**: `/account/promotion`
- **页面组件位置**: `pages/account/promotion.tsx`
- **涉及角色**: owner
- **前置条件**: 系统开启推广功能（feConfigs.show_promotion）；用户为团队 owner（permission.isOwner）

### 管理 API 密钥

- **业务描述**: 创建、查看和删除 API 访问密钥，用于外部应用调用 FastGPT 接口。
- **触发入口**: PC 端左侧导航栏 → API 密钥；移动端顶部选项卡 → API 密钥
- **路由路径**: `/account/apikey`
- **页面组件位置**: `pages/account/apikey.tsx`
- **涉及角色**: owner, admin
- **前置条件**: 用户具有 API 密钥创建权限（hasApikeyCreatePer）

### 管理通知偏好

- **业务描述**: 配置通知接收方式和偏好设置。
- **触发入口**: PC 端左侧导航栏 → 通知；移动端顶部选项卡 → 通知
- **路由路径**: `/account/inform`
- **页面组件位置**: `pages/account/inform.tsx`
- **涉及角色**: owner, admin, member

### 个性化设置

- **业务描述**: 切换系统语言等个人偏好设置。
- **触发入口**: PC 端左侧导航栏 → 语言设置；移动端顶部选项卡 → 语言设置
- **路由路径**: `/account/setting`
- **页面组件位置**: `pages/account/setting.tsx`
- **涉及角色**: owner, admin, member
- **前置条件**: 用户已登录

### 退出登录

- **业务描述**: 安全退出当前账号，清除登录状态并跳转到登录页。
- **触发入口**: PC 端左侧导航栏底部 → 退出登录；移动端顶部选项卡 → 退出登录
- **路由路径**: 无独立路由（触发后跳转 `/login`）
- **页面组件位置**: 逻辑位于 `AccountContainer.tsx`
- **涉及角色**: owner, admin, member
- **前置条件**: 用户已登录；需二次确认

## 场景关系

- 各场景**独立执行**，无严格先后关系。用户通过侧边栏/选项卡自由切换，无需按特定顺序操作。
- **退出登录** 为终止性操作，执行后清空所有会话状态，需重新登录。
