# 账单记录 — 模块概览

## 1. 模块业务概述

账单记录是账户中心「账单与发票」页面的核心子模块，为用户提供已生成账单的完整记录查看与管理功能。用户在此可查看所有历史账单（含充值账单、订阅套餐账单、额外数据集存储账单、额外积分账单），按类型筛选，对未支付账单执行支付刷新、取消或查看详情操作。

本模块是账户→账单下的叶子能力节点，与「发票记录」「发票抬头」并列构成账单页面的三大 Tab 区域。

## 2. 角色差异

本模块不涉及角色差异化展示。所有已登录用户均可查看自身团队的账单记录并执行支付/取消/查看详情操作。权限控制由上层 `AccountContainer` 统一处理登录态校验。

## 4. 对外共享

本模块为叶子能力节点，不对外提供共享组件或 API。账单相关 API 函数定义在 `@/web/support/wallet/bill/api` 共享 API 层，可能被其他模块调用。
