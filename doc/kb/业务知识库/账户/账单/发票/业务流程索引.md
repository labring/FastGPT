# 发票 — 业务流程索引

## 查看发票记录

- **业务描述**: 用户进入发票记录页面，查看团队已提交的所有开票申请记录，包括发票金额、开票时间和处理状态。
- **触发入口**: 账户中心 → 账单页面 → 点击"发票记录"Tab
- **路由路径**: `/account/bill?invoiceTab=invoice`
- **页面组件位置**: `projects/app/src/pageComponents/account/bill/InvoiceTable.tsx`
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 已登录并加入团队

## 查看发票详情

- **业务描述**: 在发票记录列表中点击某条记录的"详情"按钮，查看该发票的完整信息（开票金额、公司信息、联系方式等）。
- **触发入口**: 发票记录列表 → 点击某条记录的"详情"按钮
- **路由路径**: `/account/bill?invoiceTab=invoice`（弹窗展示，不改变路由）
- **页面组件位置**: `projects/app/src/pageComponents/account/bill/InvoiceTable.tsx`（InvoiceDetailModal）
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 存在至少一条发票记录

## 下载发票文件

- **业务描述**: 对于状态为"已完成"的发票记录，在详情弹窗中点击下载链接，将电子发票 PDF 文件下载到本地。
- **触发入口**: 发票详情弹窗 → 点击"点击下载"链接
- **路由路径**: `/account/bill?invoiceTab=invoice`（弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/account/bill/InvoiceTable.tsx`（InvoiceDetailModal）
- **涉及角色**: 团队管理员、团队成员
- **前置条件**: 发票状态为"已完成"（status=2），即已有发票文件

## 场景关系

- **查看发票记录** → **查看发票详情**: 在列表中选择一条记录打开详情弹窗
- **查看发票详情** → **下载发票文件**: 对于已完成的发票，从详情中下载 PDF 文件
- 查看记录、查看详情、下载文件为递进关系；各条发票记录之间独立，无相互依赖
