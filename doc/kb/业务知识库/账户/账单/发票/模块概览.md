# 发票 — 模块概览

## 1. 模块业务概述

发票管理模块是账户中心账单系统的子能力，为团队用户提供发票记录查看和发票详情下载功能。当团队完成订单支付后，可在此模块查看已提交的开票申请记录，跟踪发票开具状态（已提交/已完成），并下载已完成的电子发票文件。

本模块是"账单→发票"路径下的一个叶子能力，与父级"账单"模块通过 Tab 切换关联（账单记录Tab / 发票记录Tab / 发票抬头Tab），共同构成团队费用管理闭环。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队管理员 | 发票记录列表 | 查看发票记录、查看发票详情、下载已完成的发票文件 | 可查看团队所有成员的发票记录 |
| 团队成员 | 发票记录列表 | 查看发票记录、查看发票详情、下载已完成的发票文件 | 与管理员权限一致，均可查看团队级发票数据 |

**权限模型**: 基于团队层级的访问控制。发票数据属于团队级别资源，所有团队成员均可查看团队的发票记录和下载发票文件。具体权限校验由后端 API 层处理，前端通过 teamId 进行数据隔离。

## 4. 对外共享

本模块为叶子能力节点，InvoiceTable 组件仅被父级账单页面引用，无其他外部模块消费。InvoiceDetailModal 为内部组件，不对外暴露。
