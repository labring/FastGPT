# 发票抬头 — 业务流程索引

## 1. 业务场景列表

### 管理发票抬头

- **业务描述**: 团队管理员查看和编辑团队的默认发票抬头信息，包括组织名称、统一社会信用代码、公司地址、联系电话、银行账户以及是否需要增值税专票等。
- **触发入口**: 账户中心 → 账单页 → 点击「默认抬头」Tab
- **路由路径**: `/account/bill?invoiceTab=invoiceHeader`
- **页面组件位置**: `projects/app/src/pageComponents/account/bill/InvoiceHeaderForm.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 用户已登录且有团队管理权限；团队账单功能已开通。

## 2. 场景关系

各场景独立执行，无严格先后关系。
