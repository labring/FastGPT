# 账单 — 模块概览

## 1. 模块业务概述

账单模块是 FastGPT 账户体系中的财务管理入口，为团队管理员提供账单记录查看、发票申请与发票抬头管理的完整财务闭环。用户在此可以查看历史支付订单、对未支付订单进行支付或取消操作、选中已完成支付的账单申请开具发票，以及维护企业的发票抬头信息（如公司名称、税号、银行账户等）。

本模块包含三个子 Tab 页面：**账单记录**（查看/支付/取消账单）、**发票**（查看已开具发票并下载 PDF 文件）和**发票抬头**（维护企业发票抬头信息）。三个功能围绕"支付 → 开票"这一核心财务链路展开。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队管理员（hasManagePer） | 全部（账单记录、发票、发票抬头） | 查看账单、支付/取消未支付订单、申请发票、管理发票抬头 | 仅具有管理权限的团队成员可见此模块 |
| 普通成员 | 无 | 无 | 账单模块仅在 `feConfigs.show_pay` 开启且用户具有 `hasManagePer` 权限时才在侧边栏可见 |

**权限模型**：基于团队权限字段 `userInfo.team.permission.hasManagePer` 和系统配置 `feConfigs.show_pay` 的前端条件渲染。账单 Tab 仅在两个条件同时满足时出现在账户侧边栏。
