# 账户 — 第三方集成

## 模块业务概述

第三方集成模块为团队管理员提供统一的外部服务配置入口。管理员可在此连接 Laf 云函数平台、OpenAI/OneAPI 兼容的 API 账号，以及平台定义的第三方工作流变量（如外部服务鉴权 Token）。已配置的第三方服务可在团队级别共享，供团队所有成员在应用工作流中使用，无需每个成员单独配置。

本模块属于账户管理的子模块之一，面向团队管理员提供配置能力，团队成员仅可查看已配置状态。

## 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队拥有者（owner） | 第三方集成管理页 | 查看所有集成；点击卡片配置 Laf 账号、OpenAI 账号、工作流变量 | 唯一可执行配置操作的角色 |
| 团队成员（非 owner） | 第三方集成管理页 | 查看所有集成的配置状态；点击卡片时提示"请联系管理员配置" | 无配置权限，仅可查看是否有配置 |

**权限模型**: 基于用户 `team.role` 的字符串比较（`TeamMemberRoleEnum.owner`），在页面组件的 `onClick` 事件中进行判断。非 owner 角色点击配置卡片时弹出提示 toast。
