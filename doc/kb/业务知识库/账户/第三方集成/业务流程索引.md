# 账户 — 第三方集成 业务流程索引

## 查看第三方集成列表

- **业务描述**: 进入第三方集成管理页面，以卡片网格形式查看所有已启用的第三方服务集成状态（已配置/未配置）和用量信息。
- **触发入口**: 账户管理页面侧边栏 → "第三方集成"菜单项（桌面端），或账户页面顶部 Tab 栏 → "第三方集成"（移动端）
- **路由路径**: `/account/thirdParty`
- **页面组件位置**: `projects/app/src/pages/account/thirdParty/index.tsx`
- **涉及角色**: owner, member
- **前置条件**: 已登录状态

## 配置 Laf 账号

- **业务描述**: 管理员配置 Laf 云函数平台账号，输入 PAT（个人访问令牌）验证后选择目标应用，保存后团队可在工作流中调用 Laf 云函数。
- **触发入口**: 第三方集成管理页 → 点击 "Laf 账号" 卡片
- **路由路径**: `/account/thirdParty`（模态弹窗）
- **页面组件位置**: `projects/app/src/components/support/laf/LafAccountModal.tsx`
- **涉及角色**: owner
- **前置条件**: 系统配置中已启用 Laf 环境（`feConfigs.lafEnv` 存在）

## 配置 OpenAI 账号

- **业务描述**: 管理员配置 OpenAI 或 OneAPI 兼容的 API 密钥和请求地址，团队在应用中使用 AI 对话、问题分类和内容提取功能时将优先使用此密钥，不从平台计费。
- **触发入口**: 第三方集成管理页 → 点击 "OpenAI/OneAPI 账号" 卡片
- **路由路径**: `/account/thirdParty`（模态弹窗）
- **页面组件位置**: `projects/app/src/pageComponents/account/thirdParty/OpenAIAccountModal.tsx`
- **涉及角色**: owner
- **前置条件**: 系统配置中已显示 OpenAI 账号配置入口（`feConfigs.show_openai_account` 为 true）

## 配置工作流变量

- **业务描述**: 管理员为平台定义的第三方工作流变量提供商配置鉴权参数（如 API Token），保存后团队在工作流中可使用对应变量。
- **触发入口**: 第三方集成管理页 → 点击对应工作流变量提供商的卡片
- **路由路径**: `/account/thirdParty`（模态弹窗）
- **页面组件位置**: `projects/app/src/pageComponents/account/thirdParty/WorkflowVariableModal.tsx`
- **涉及角色**: owner
- **前置条件**: 系统配置中定义了外部工作流变量提供商（`feConfigs.externalProviderWorkflowVariables` 非空）；该提供商入口已开启

## 场景关系

- **查看第三方集成列表** → 是进入模块的默认场景，其他三个配置场景均从该页面触发
- **配置 Laf 账号**、**配置 OpenAI 账号**、**配置工作流变量** 三个场景独立执行，无先后依赖关系
