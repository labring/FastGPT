# 账号信息 — 模块概览

## 1. 模块业务概述

账号信息模块是 FastGPT 账户体系中的个人信息管理中心，为所有已登录用户提供个人资料查看与编辑、团队套餐使用情况查看、以及客服支持入口等功能。该模块是「账户」分组下的核心节点之一，承载用户对自身账户状态和团队资源消耗的感知。

模块面向三类角色提供服务：普通成员可查看和编辑个人信息；团队管理员可额外观阅余额兑换入口；团队拥有者还可使用优惠券兑换功能。页面采用 PC 端左右双栏布局（左侧个人信息+客服入口，右侧套餐用量），移动端则纵向堆叠展示。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 普通成员 | /account/info | 查看/修改个人信息、修改密码（Plus版）、修改联系方式（Plus版）、上传头像、修改团队昵称、查看套餐用量、查看帮助文档、联系客服 | 基础个人信息管理 |
| 团队管理员 | /account/info | 上述所有操作 + 余额兑换AI积分 | 额外显示余额兑换按钮（`permission.hasManagePer` 且有标准套餐） |
| 团队拥有者 | /account/info | 上述所有操作 + 兑换优惠券、查看折扣券 | 额外显示优惠券和折扣券入口（`permission.isOwner` 且配置开启） |

**权限模型**: 基于 `useUserStore.userInfo.permission` 中的 `hasManagePer` 和 `isOwner` 字段进行条件渲染；功能开关还受 `feConfigs.isPlus`、`feConfigs.show_coupon`、`feConfigs.show_discount_coupon` 等系统配置控制。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 用户状态管理（useUserStore） | 获取/更新用户信息、获取团队套餐状态 | Store |
| 系统配置（useSystemStore） | 读取 feConfigs 功能开关、子套餐配置 | Store |
| 共享UI组件库（@fastgpt/web） | Avatar、Icon、MyTooltip、QuestionTip、MyDivider | 组件 |
| 账户容器（AccountContainer） | PC侧边栏导航 + 移动端Tab切换 | 组件 |
| 团队管理 | TeamSelector 团队选择器组件 | 组件 |
| 钱包/套餐（wallet） | 套餐内容清单组件、价格格式化工具、余额兑换 | 组件/API |

## 4. 对外共享

本模块为叶子节点，不对外暴露共享能力。模块内组件（如 ConversionModal、UpdatePswModal）仅在本页面内部使用。
