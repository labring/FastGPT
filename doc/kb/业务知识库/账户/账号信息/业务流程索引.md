# 账号信息 — 业务流程索引

## 查看与管理个人信息

- **业务描述**: 用户进入账号信息页面后，可查看用户名、密码状态、联系方式、头像、团队昵称、团队余额等个人账户信息，并进行修改操作。
- **触发入口**: 账户侧边栏（PC）→ 点击「个人信息」菜单项；移动端顶部 Tab → 选择「个人信息」
- **路由路径**: `/account/info`
- **页面组件位置**: `projects/app/src/pages/account/info/index.tsx`
- **涉及角色**: 所有已登录用户
- **前置条件**: 用户已登录

## 查看套餐与资源用量

- **业务描述**: 有标准套餐的团队用户可查看当前套餐类型、到期时间、套餐内容明细，以及AI积分用量、知识库索引用量、成员数/应用数/知识库数等资源使用情况。
- **触发入口**: 账号信息页面右侧区域（PC）或个人信息下方（移动端）
- **路由路径**: `/account/info`
- **页面组件位置**: `projects/app/src/pages/account/info/index.tsx`（PlanUsage 子组件）
- **涉及角色**: 所有已登录用户（需团队有标准套餐）
- **前置条件**: 团队已开通标准套餐

## 修改密码

- **业务描述**: Plus 版用户可修改登录密码，需输入旧密码、新密码并确认新密码，新密码需满足复杂度要求。
- **触发入口**: 个人信息区域 → 点击密码行「修改」按钮
- **路由路径**: `/account/info`（弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/account/info/UpdatePswModal.tsx`
- **涉及角色**: 所有已登录用户（Plus版）
- **前置条件**: 系统配置 `feConfigs.isPlus` 为 true

## 修改联系方式

- **业务描述**: Plus 版用户可修改绑定的联系方式（手机号/邮箱），用于账号安全和消息通知。
- **触发入口**: 个人信息区域 → 点击联系方式行「修改」按钮
- **路由路径**: `/account/info`（弹窗内操作）
- **页面组件位置**: `projects/app/src/components/support/user/inform/UpdateContactModal.tsx`
- **涉及角色**: 所有已登录用户（Plus版）
- **前置条件**: 系统配置 `feConfigs.isPlus` 为 true

## 余额兑换AI积分

- **业务描述**: 团队管理员可将账户余额按比例兑换为AI积分（Tokens），用于模型调用消耗。
- **触发入口**: 个人信息区域 → 余额行右侧「兑换」按钮
- **路由路径**: `/account/info`（弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/account/info/ConversionModal.tsx`
- **涉及角色**: 团队管理员（`permission.hasManagePer`）
- **前置条件**: 团队有余额且已开通标准套餐

## 兑换优惠券

- **业务描述**: 团队拥有者可输入优惠券兑换码，兑换额外的资源包或权益。
- **触发入口**: 套餐用量区域 →「兑换优惠券」按钮
- **路由路径**: `/account/info`（弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/account/info/RedeemCouponModal.tsx`
- **涉及角色**: 团队拥有者（`permission.isOwner`）
- **前置条件**: 系统配置 `feConfigs.show_coupon` 为 true

## 查看折扣券

- **业务描述**: 团队拥有者可查看当前可用的折扣券列表，了解可使用优惠。
- **触发入口**: 套餐用量区域 →「折扣券」按钮
- **路由路径**: `/account/info`（弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/account/info/DiscountCouponsModal.tsx`
- **涉及角色**: 团队拥有者（`permission.isOwner`）
- **前置条件**: 系统配置 `feConfigs.show_discount_coupon` 为 true

## 查看帮助文档

- **业务描述**: 用户可点击进入帮助文档页面，了解产品使用指南。
- **触发入口**: 页面底部 →「帮助文档」按钮
- **路由路径**: 外部链接（`/docs/introduction`）
- **页面组件位置**: `projects/app/src/pages/account/info/index.tsx`（Other 子组件）
- **涉及角色**: 所有已登录用户
- **前置条件**: 系统配置 `feConfigs.docUrl` 存在

## 联系客服

- **业务描述**: 用户可通过社区群联系客服，获取人工支持。
- **触发入口**: 页面底部 →「联系我们」按钮
- **路由路径**: `/account/info`（弹窗）
- **页面组件位置**: `projects/app/src/pages/account/info/index.tsx`（Other 子组件 → CommunityModal）
- **涉及角色**: 所有已登录用户
- **前置条件**: 系统配置 `feConfigs.concatMd` 为 true

## 问题反馈

- **业务描述**: 有标准套餐支持工单的用户可提交问题反馈工单。
- **触发入口**: 页面底部 →「问题反馈」按钮
- **路由路径**: 外部链接（工单系统重定向）
- **页面组件位置**: `projects/app/src/pages/account/info/index.tsx`（Other 子组件）
- **涉及角色**: 所有已登录用户（需有工单访问权限）
- **前置条件**: 系统配置 `feConfigs.show_workorder` 为 true，且团队套餐支持工单

## 场景关系

- **查看与管理个人信息** → **修改密码** / **修改联系方式** / **余额兑换**: 个人信息区域的子操作
- **查看套餐与资源用量** → **兑换优惠券** / **查看折扣券**: 套餐区域的子操作
- 各场景均为独立操作入口，页面一次性加载全部信息，场景间无严格先后依赖关系
