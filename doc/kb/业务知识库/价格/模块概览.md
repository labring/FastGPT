# 价格 — 模块概览

## 1. 模块业务概述

价格模块是 FastGPT 的订阅购买中心，为用户提供完整的套餐选购、增值服务购买和 AI 模型计费查询功能。模块面向所有团队用户，支持标准订阅套餐的购买/续费/升级、额外 AI 积分的购买、额外知识库容量的购买，以及查看不同 AI 模型的价格信息。

页面采用单页滚动布局，自上而下依次展示标准订阅方案、额外增值服务、AI 模型计费表和常见问题解答，用户无需跳转即可完成从选套餐到支付的完整流程。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队管理员 | 全部价格页面 | 购买/续费/升级标准套餐，购买额外积分和知识库容量，查看模型价格 | 有完整购买权限 |
| 团队成员 | 全部价格页面 | 查看套餐方案和模型价格 | 购买操作由管理员权限控制 |

**权限模型**: 基于团队订阅状态的前端条件渲染，通过 `useUserStore` 获取用户信息和团队计划状态。企业微信团队（`isWecomTeam`）有专属的购买限制和价格逻辑。免费套餐的企业微信团队无法购买额外增值服务。

| 依赖模块 | 使用的具体能力 | 能力类型 |

| 钱包/账单 | 创建支付账单 API（`postCreatePayBill`）、支付二维码弹窗组件（`QRCodePayModal`） | API / 组件 |
| 账户/团队 | 获取团队计划状态、团队计划列表 API | API |
