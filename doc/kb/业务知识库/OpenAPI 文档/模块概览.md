# OpenAPI 文档 — 模块概览

## 1. 模块业务概述

OpenAPI 文档模块为 FastGPT 平台的 API 使用者（开发者、集成方）提供了一个交互式 API 参考文档界面。该页面使用 Scalar API Reference 组件，以可视化方式展示 FastGPT 开放 API 的完整规范，包括请求方法、参数、响应格式和认证方式，支持开发者在浏览器中直接浏览和测试 API。

本模块属于平台的基础设施型页面，面向所有可访问平台的用户开放，无需登录即可查阅 API 文档。

## 2. 角色差异

本模块为公开文档页面，所有角色（含未登录用户）具有相同的访问权限：均可查看完整的 OpenAPI 规范文档。

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 所有用户（含未登录） | OpenAPI 文档页 | 浏览 API 规范、展开/折叠接口详情、查看请求/响应示例 | 无角色差异，页面完全公开 |

**权限模型**: 无权限校验。页面未调用 `getServerSideProps` 中的认证逻辑，直接渲染公开内容。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| @scalar/api-reference-react（外部依赖） | Scalar API Reference React 组件，提供交互式 API 文档渲染 | 组件 |
| @fastgpt/global/openapi | OpenAPI 规范文档定义 (`openAPIDocument`)，包含所有 API 路径和 Schema | API |
| /api/openapi.json | 后端 API 端点，返回完整的 OpenAPI 3.1.0 JSON 规范 | API |

## 4. 对外共享

本模块为终端展示页面，不向其他模块提供可消费的能力。无组件、API、Hook 或 Store 被其他模块引用。
