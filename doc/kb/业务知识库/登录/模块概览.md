# 登录 — 模块概览

## 1. 模块业务概述

登录模块是 FastGPT 平台的用户身份认证入口，为所有未登录用户提供多种方式的安全登录能力。模块支持密码登录、OAuth 第三方登录（SSO/Google/GitHub/Microsoft）、微信扫码登录、企业微信自动登录，以及账号注册和密码找回等完整的账号生命周期管理功能。

模块包含三个子能力页面：**登录首页**提供登录方式选择与表单交互的主界面；**快速登录**支持通过一次性令牌实现免密快速登录；**第三方登录**处理 OAuth 回调后的认证流程，完成第三方账号与平台账号的绑定与登录。

## 2. 角色差异

登录模块面向**未认证用户**，所有访问者处于未登录状态，无角色区分。系统根据 `feConfigs` 配置动态展示可用的登录方式：

- 社区版（`register_method` 有值且 `isPlus` 为 false）仅展示 root 账号密码登录
- 商业版可根据配置展示密码登录、微信扫码、OAuth 第三方登录等
- 配置了 SSO 自动登录时，进入页面自动跳转 SSO 认证
- 企业微信终端访问时自动触发企业微信登录

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 系统 Store（useSystemStore） | 读取 `feConfigs` 配置（OAuth、登录方式）、读写 `loginStore` 状态 | Store |
| 用户 Store（useUserStore） | 登录成功后调用 `setUserInfo` 写入用户信息 | Store |
| 对话 Store（useChatStore） | 登录页初始化时调用 `setLastChatAppId` 清空对话上下文 | Store |
| 用户 API（@/web/support/user/api） | 调用登录、注册、密码找回、微信登录等全部认证 API | API |
| 营销工具（@/web/support/marketing/utils） | 读取/清除营销追踪参数（bd_vid、fastgpt_sem、msclkid 等） | 工具函数 |

## 4. 对外共享

| 共享能力 | 消费模块 | 能力类型 |

| OAuth 认证流程 | 第三方登录回调页面 | 流程 |
