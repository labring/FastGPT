# 快速登录 — 模块概览

## 1. 模块业务概述

快速登录是 FastGPT 登录体系中的免密认证入口，面向已持有有效临时凭证（code + token）的未登录用户。用户通过携带 `code` 和 `token` 查询参数访问 `/login/fastlogin` 页面，系统自动完成身份认证并跳转至目标页面，全程无需用户手动输入密码或点击任何按钮。

该模块为"登录"能力树的叶子节点，与[登录首页](../登录首页/模块概览.md)、[第三方登录](../第三方登录/模块概览.md)共同构成完整的登录入口体系。快速登录通常作为邮件链接、通知跳转等外部触达场景的认证桥梁。

本模块无子能力。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 未登录用户 | 快速登录页 | 通过 code+token 自动认证 | 持有有效凭证才可完成登录，凭证失效则跳回登录首页 |

**权限模型**: 本模块无角色权限校验。页面为公开路由，任何携带有效 `code` 和 `token` 参数的访客均可访问。认证成功后的权限由 `setUserInfo` 写入的用户信息决定。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 用户认证服务 | `postFastLogin` — 提交快速登录凭证完成认证 | API |
| 用户状态管理 | `useUserStore.setUserInfo` — 写入登录后的用户信息 | Store |
| 认证工具 | `clearToken` — 登录前清除旧凭证、登出旧会话 | 工具函数 |
| URL 安全校验 | `validateRedirectUrl` — 校验回调 URL 合法性，防止开放重定向攻击 | 工具函数 |
| 通用组件 | `Loading` — 认证过程中的加载状态展示 | 组件 |

## 4. 对外共享

快速登录模块为纯入口页面，不对外提供可复用的组件、API 或 Store。
