# 快速登录 — 业务流程索引

## 快速登录认证

- **业务描述**: 用户通过外部链接携带的 code 和 token 参数访问快速登录页面，系统自动校验凭证有效性，校验通过后完成登录并跳转至目标页面。
- **触发入口**: 外部链接（邮件、通知等）→ 携带 code、token、callbackUrl 参数访问 `/login/fastlogin`
- **路由路径**: `/login/fastlogin`
- **页面组件位置**: `projects/app/src/pages/login/fastlogin.tsx`
- **涉及角色**: 未登录用户
- **前置条件**: 持有有效的 `code` 和 `token` 临时凭证

## 场景关系

本模块仅有一个业务场景，无场景间依赖关系。
