# 登录 — 业务流程索引

## 业务场景列表

### 密码登录

- **业务描述**: 用户通过用户名和密码完成身份认证，登录后进入工作台。
- **触发入口**: 访问 `/login` 页面，默认展示密码登录表单
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/pages/login/index.tsx`
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 系统未配置强制 SSO 自动登录；用户已有账号

### 注册账号

- **业务描述**: 新用户通过邮箱/手机号注册平台账号，注册成功后自动登录。
- **触发入口**: 密码登录表单下方的"注册"链接
- **路由路径**: `/login`（切换为 register 表单类型）
- **页面组件位置**: `projects/app/src/pages/login/index.tsx` → RegisterForm
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 系统配置了 `register_method`；用户尚未注册

### 忘记密码

- **业务描述**: 用户通过验证码重置密码，重置成功后自动登录。
- **触发入口**: 密码登录表单下方的"忘记密码"链接
- **路由路径**: `/login`（切换为 forgetPassword 表单类型）
- **页面组件位置**: `projects/app/src/pages/login/index.tsx` → ForgetPasswordForm
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 系统配置了 `find_password_method`

### 微信扫码登录

- **业务描述**: 用户使用微信扫描二维码完成登录，无需输入密码。
- **触发入口**: OAuth 列表中点击"微信"按钮，或系统配置默认展示微信登录
- **路由路径**: `/login`（切换为 wechat 表单类型）
- **页面组件位置**: `projects/app/src/pages/login/index.tsx` → WechatForm
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 系统配置了 `feConfigs.oauth.wechat`

### OAuth 第三方登录

- **业务描述**: 用户通过 Google/GitHub/Microsoft/SSO 等第三方平台授权登录。
- **触发入口**: 登录页面 OAuth 列表中点击对应第三方按钮
- **路由路径**: `/login` → 跳转第三方授权页 → 回调 `/login/provider`
- **页面组件位置**: `projects/app/src/pages/login/provider.tsx`
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 系统配置了对应的 OAuth provider（Google/GitHub/Microsoft/SSO/企业微信）

### 快速登录

- **业务描述**: 用户通过一次性 code 和 token 参数完成免密登录，适用于邮件链接等场景。
- **触发入口**: 携带 code 和 token 参数的 URL 直接访问
- **路由路径**: `/login/fastlogin`
- **页面组件位置**: `projects/app/src/pages/login/fastlogin.tsx`
- **涉及角色**: 所有用户（未认证状态）
- **前置条件**: 有效的 code 和 token 参数

## 场景关系

- **密码登录** ↔ **注册账号**: 密码登录页可切换到注册页；注册成功后自动登录
- **密码登录** ↔ **忘记密码**: 密码登录页可切换到忘记密码页；重置成功后自动登录
- **密码登录** → **微信扫码登录**: 可切换登录方式
- **密码登录** → **OAuth 第三方登录**: 可切换到第三方登录
- **各登录方式** → 登录成功后均跳转到 **工作台**（`/dashboard/agent`）或 `lastRoute` 指定路径
