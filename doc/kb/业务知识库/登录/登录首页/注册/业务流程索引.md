# 注册 — 业务流程索引

## 注册账号

- **业务描述**：访客在登录页切换到注册 Tab，填写用户名（邮箱/手机号）、获取验证码、设置密码后提交注册。注册成功自动登录并跳转至应用首页。
- **触发入口**：登录首页 → 点击"注册"链接 → 切换到注册 Tab
- **路由路径**：`/login`（注册 Tab 内，无独立路由）
- **页面组件位置**：`projects/app/src/pageComponents/login/RegisterForm.tsx`
- **涉及角色**：访客
- **前置条件**：系统配置中 `feConfigs.register_method` 非空（系统管理员启用了注册功能）

## 场景关系

各场景独立执行，无严格先后关系。注册表单为单一场景——用户进入页面后需依次完成填写用户名、获取验证码、设置密码、确认密码、提交注册。
