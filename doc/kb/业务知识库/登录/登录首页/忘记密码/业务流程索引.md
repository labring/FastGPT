# 忘记密码 — 业务流程索引

## 找回密码

- **业务描述**: 用户遗忘登录密码时，通过已注册的邮箱/手机号获取验证码，验证身份后设置新密码，完成密码重置并自动登录。
- **触发入口**: 登录页面（`/login`）密码登录表单底部 → 点击"忘记密码"链接。
- **路由路径**: `/login`（登录页面内嵌，通过 `setPageType('forgetPassword')` 切换为忘记密码表单）。
- **页面组件位置**: `projects/app/src/pageComponents/login/ForgetPasswordForm.tsx`
- **涉及角色**: 所有用户（公开）
- **前置条件**: 
  - 系统配置中启用了密码找回方式（`feConfigs.find_password_method` 不为空，支持 email/phone）
  - 用户拥有已注册的账号

## 场景关系

- **密码登录** → **忘记密码**：用户在密码登录流程中，因遗忘密码而跳转到忘记密码流程。找回密码成功后自动登录，进入系统。
