# 登录首页 — 业务流程索引

## 业务场景列表

### 页面初始化

- **业务描述**: 用户访问登录页面时，系统自动加载页面布局、检测微信 OAuth 配置决定默认登录方式（微信或密码登录），并执行 IP 归属地检测和 Cookie 版本检查
- **触发入口**: 直接访问 `/login` 路由
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/pages/login/index.tsx`
- **涉及角色**: 未登录用户
- **前置条件**: 用户未登录状态

### OAuth 登录通道

- **业务描述**: 用户通过 FormLayout 中展示的第三方 OAuth 按钮（SSO/Google/GitHub/Microsoft/企业微信）跳转到对应平台完成授权登录
- **触发入口**: 登录表单下方的 OAuth 按钮区
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/pageComponents/login/LoginForm/FormLayout.tsx`
- **涉及角色**: 未登录用户
- **前置条件**: 系统配置中启用了对应的 OAuth 提供商

### 中国 IP 重定向

- **业务描述**: 系统通过百度 IP 定位 API 检测用户是否在中国大陆，若是则弹出重定向提示引导用户访问国内站点
- **触发入口**: 页面加载时自动触发
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/pages/login/index.tsx` 中的 RedirectDrawer
- **涉及角色**: 未登录用户（中国大陆 IP）
- **前置条件**: 环境变量配置了 `CHINESE_IP_REDIRECT_URL`，且用户未关闭提示

### Cookie 同意

- **业务描述**: 用户首次访问时弹出 Cookie 同意抽屉，引导用户阅读隐私政策并授权 Cookie 使用
- **触发入口**: 页面加载时自动触发（localStorage 无对应版本标记时）
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/pages/login/index.tsx` 中的 CookiesDrawer
- **涉及角色**: 未登录用户
- **前置条件**: 用户本地未记录当前 Cookie 版本的同意标记

### 社区联系

- **业务描述**: 用户点击"无法登录"链接时，弹出社区联系方式弹窗（支持 Markdown 自定义内容或企业微信团队专用提示）
- **触发入口**: 登录表单区域下方的"无法登录"链接
- **路由路径**: `/login`
- **页面组件位置**: `projects/app/src/components/CommunityModal/index.tsx`
- **涉及角色**: 未登录用户
- **前置条件**: 系统配置了 `concatMd` 联系方式内容

## 场景关系

- **页面初始化** → **OAuth 登录通道**: 页面加载完成后，OAuth 按钮根据系统配置动态渲染
- **页面初始化** → **中国 IP 重定向**: IP 检测与页面加载并行执行
- **页面初始化** → **Cookie 同意**: Cookie 版本检查与页面加载并行执行
- **页面初始化** → **Tab 子能力（密码登录等）**: 默认展示密码登录 Tab
- 各 Tab 子能力（密码登录/注册/忘记密码/微信登录）之间通过切换互斥展示
