# 密码登录 — 业务流程索引

## 业务场景列表

### 密码登录

- **业务描述**: 用户输入已注册的用户名和密码，系统验证身份后签发登录令牌并跳转至应用工作台。支持通过 URL 参数携带凭据实现一键自动登录。
- **触发入口**: 访问登录页 `/login`，默认展示密码登录 Tab；或其他登录方式（微信扫码登录等）切换到密码登录
- **路由路径**: `/login`（pageType=passwordLogin）
- **页面组件位置**: `projects/app/src/pageComponents/login/LoginForm/LoginForm.tsx`
- **涉及角色**: 未认证用户（无需角色鉴权）
- **前置条件**: 
  - 用户未登录（未持有有效 token）
  - 系统配置允许密码登录（默认始终可用）
  - 如为自动登录，URL 需携带 `u`（用户名）和 `p`（密码）参数

## 场景关系

各登录子能力（密码登录、微信登录、注册、忘记密码）之间通过页面内 Tab 切换关联，互不依赖：

- **密码登录** → **忘记密码**: 用户忘记密码时通过底部链接切换
- **密码登录** → **注册**: 新用户通过底部链接切换到注册 Tab
