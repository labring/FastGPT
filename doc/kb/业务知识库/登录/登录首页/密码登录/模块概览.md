# 密码登录 — 模块概览

## 1. 模块业务概述

密码登录是 FastGPT 平台的核心身份认证方式，为所有用户提供基于用户名+密码的登录入口。用户在登录页面输入已注册的账号凭证后，系统完成身份验证并签发登录令牌，随后跳转至应用工作台。

本模块是登录首页的密码登录子能力（Tab），与微信登录、注册、忘记密码等其他登录相关子能力共享同一页面入口。支持根据系统配置自动适配登录方式：社区版（Community Version）仅允许 root 账号登录；商业版支持用户名、邮箱、手机号多种登录方式。同时支持通过 URL 参数携带凭据实现一键自动登录。

## 2. 角色差异

密码登录页面面向**未认证用户**，无需角色鉴权。系统通过前端配置 `feConfigs` 控制可用的登录方式：

| 配置项 | 影响 |
|--------|------|
| `feConfigs.register_method` | 控制注册入口和社区版判断 |
| `feConfigs.isPlus` | 区分社区版/商业版 |
| `feConfigs.login_method` | 控制用户名/邮箱/手机号登录方式的占位提示 |
| `feConfigs.find_password_method` | 控制"忘记密码"入口显示 |
| `feConfigs.register_method` | 控制"注册"入口显示 |
