# 微信登录 — 业务流程索引

## 微信扫码登录

- **业务描述**: 用户在登录页通过微信客户端扫描二维码完成身份认证，无需输入账号密码即可登录平台。系统自动轮询检测扫码和确认状态。
- **触发入口**: 系统配置启用微信 OAuth 后，访问登录页默认进入微信登录页；或从其他登录方式（密码登录）切换至微信登录
- **路由路径**: `/login`（pageType 为 wechat）
- **页面组件位置**: `projects/app/src/pageComponents/login/LoginForm/WechatForm.tsx`
- **涉及角色**: 未登录用户（所有访问者）
- **前置条件**: 系统配置中微信 OAuth 已启用（`feConfigs.oauth.wechat` 为 true）；用户未被识别为百度推广来源（无 bd_vid）

## 场景关系

本模块仅包含微信扫码登录一个场景，无其他场景关联。
