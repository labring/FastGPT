# 第三方登录 — 业务流程索引

## 业务场景列表

### OAuth 回调登录

- **业务描述**：用户从登录页选择第三方登录（GitHub / Google / 微信 / Microsoft / 企业微信 / SSO）并在第三方平台完成授权后，浏览器被重定向到本页面，系统自动完成令牌交换、用户信息获取和会话建立。
- **触发入口**：第三方 OAuth 平台授权成功后的自动重定向（非用户手动触发）
- **路由路径**：`/login/provider`
- **页面组件位置**：`projects/app/src/pages/login/provider.tsx`
- **涉及角色**：所有用户
- **前置条件**：
  - 已从登录页点击了第三方登录按钮
  - 被重定向到第三方授权页面并完成授权
  - URL 中携带了授权 code 和 state 参数
  - `loginStore` 中存储了 provider 类型和预期 state 值

## 场景关系

各 OAuth 提供商的回调处理共用同一页面入口，按 `loginStore.provider` 区分类型，但核心流程一致。本模块为单场景模块，无场景间依赖关系。
