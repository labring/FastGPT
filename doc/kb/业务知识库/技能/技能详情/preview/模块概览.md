# 预览调试 — 模块概览

## 1. 模块业务概述

预览调试模块是技能详情页中的"预览"Tab，为技能开发者提供一个**即时调试和测试技能的交互式聊天环境**。用户可以在不离开技能编辑页面的情况下，选择 AI 模型并与当前技能进行对话测试，实时验证技能的行为是否符合预期。

该模块面向**技能开发者（Owner）和协作者（Collaborator）**，是技能开发迭代中的核心验证工具。通过预览调试，开发者可以：选择不同的 AI 模型来测试技能效果、发送消息并观察流式响应、查看历史对话记录、清空对话重新开始测试。

本模块是技能详情的 Tab 子模块，与"配置"Tab 并列，配置 Tab 中修改技能内容后可在预览 Tab 即时测试。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| Owner（技能所有者） | 预览 Tab | 选择模型、发送消息、查看历史记录、清空对话 | 完整预览调试权限 |
| Collaborator（协作者） | 预览 Tab | 选择模型、发送消息、查看历史记录、清空对话 | 与 Owner 在预览调试上权限一致 |

**权限模型**：预览调试功能的访问受技能详情页的整体权限控制（基于 SkillPermission），Tab 层面不做额外权限区分。技能详情可访问即可使用预览调试。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 技能配置（AgentSkillEditor） | 同步技能沙箱（syncSkillSandbox）— 发送消息前先同步最新配置到沙箱 | API |
| 核心聊天（ChatBox） | 复用 ChatBox 聊天容器组件及 ChatContainer 类型定义 | 组件 |
| 聊天上下文（ChatItemContext/ChatRecordContext） | 聊天消息渲染配置、历史记录分页加载 | Context/Hook |
| 系统设置（useSystemStore） | 获取可用 LLM 模型列表供用户选择 | Store |
| AI 模型选择器（AIModelSelector） | 模型选择下拉组件 | 组件 |

## 4. 对外共享

本模块为技能详情页的内部 Tab，不对外提供共享能力。预览调试的聊天测试能力专属于技能详情场景。

## 5. 关键页面与入口

| 页面名称 | 路由路径 | 页面组件位置 | 角色 | 备注 |
