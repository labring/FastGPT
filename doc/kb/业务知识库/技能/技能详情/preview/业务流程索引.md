# 预览调试 — 业务流程索引

## 技能调试对话

- **业务描述**: 在技能详情页的预览 Tab 中，选择 AI 模型后与技能进行对话测试，实时观察技能响应。首次发送消息时会触发沙箱预热，后续消息直接进行对话。
- **触发入口**: 技能详情页 → 点击顶部"预览"Tab → 进入预览调试界面
- **路由路径**: `/skill/detail?skillId={id}`（预览 Tab 内）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/preview/SkillPreview.tsx`
- **涉及角色**: Owner, Collaborator
- **前置条件**: 用户有技能详情页的访问权限（SkillPermission）；技能 ID 有效

## 切换调试模型

- **业务描述**: 在预览界面顶部的模型选择器中，切换不同的 AI 模型来测试技能在不同模型下的表现。
- **触发入口**: 预览 Tab 顶部 → 点击模型选择器下拉框
- **路由路径**: `/skill/detail?skillId={id}`（预览 Tab 内）
- **页面组件位置**: AIModelSelector 组件（`@/components/Select/AIModelSelector`）
- **涉及角色**: Owner, Collaborator
- **前置条件**: 系统已加载 LLM 模型列表

## 清空对话重新开始

- **业务描述**: 点击清空按钮，生成新的聊天会话 ID，清空当前对话记录，重新开始调试。
- **触发入口**: 预览 Tab 顶部 → 点击清空图标按钮
- **路由路径**: `/skill/detail?skillId={id}`（预览 Tab 内）
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/preview/SkillPreview.tsx`
- **涉及角色**: Owner, Collaborator
- **前置条件**: 无特殊前置条件

## 查看历史对话记录

- **业务描述**: 在聊天区域中向上滚动查看之前的对话记录，支持分页加载历史消息。
- **触发入口**: 预览 Tab → 聊天区域向上滚动
- **路由路径**: `/skill/detail?skillId={id}`（预览 Tab 内）
- **页面组件位置**: ChatRecordContextProvider（`@/web/core/chat/context/chatRecordContext`）分页加载
- **涉及角色**: Owner, Collaborator
- **前置条件**: 已有对话历史

## 场景关系

- **技能调试对话** → **查看历史对话记录**: 发送消息后产生历史记录，向上滚动可查看
- **切换调试模型** → **技能调试对话**: 模型切换后，后续发送的消息使用新模型
- **清空对话重新开始** → **技能调试对话**: 清空后进入全新的对话会话
- 各场景之间无严格先后依赖关系
