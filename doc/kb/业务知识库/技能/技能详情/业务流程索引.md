# 技能详情 — 业务流程索引

## 业务场景列表

### 查看与编辑技能

- **业务描述**: 进入技能详情页后，默认展示 config Tab，用户可通过文件树浏览技能文件，在 Monaco 编辑器中编辑代码。
- **触发入口**: 技能列表 → 点击技能名称
- **路由路径**: /skill/detail?skillId={skillId}
- **页面组件位置**: projects/app/src/pages/skill/detail.tsx
- **涉及角色**: 拥有者、协作者（有写权限）
- **前置条件**: 已登录，有技能的访问权限

### 调试预览技能

- **业务描述**: 切换到 preview Tab，选择 AI 模型后通过对话窗口测试技能效果，支持清空对话重新开始。
- **触发入口**: 技能详情页 → 点击 preview Tab
- **路由路径**: /skill/detail?skillId={skillId}
- **页面组件位置**: projects/app/src/pages/skill/detail.tsx
- **涉及角色**: 拥有者、协作者
- **前置条件**: 已登录，有技能的访问权限

### 管理技能信息

- **业务描述**: 通过技能名旁边的菜单按钮，可编辑技能基本信息（名称、描述、头像）、配置权限、导出配置包或删除技能。
- **触发入口**: 技能详情页 Header → 技能名右侧菜单按钮
- **路由路径**: /skill/detail?skillId={skillId}
- **页面组件位置**: projects/app/src/pageComponents/dashboard/skill/detail/Header.tsx
- **涉及角色**: 拥有者（编辑/删除/导出）、协作者（按权限位）
- **前置条件**: 已登录，拥有者或有对应权限

## 场景关系

- **查看与编辑技能** 与 **调试预览技能** 为同一页面的两个 Tab，可自由切换，无先后依赖。
- **管理技能信息** 可在任意 Tab 下触发，独立于编辑和预览流程。
- 各场景独立执行，无严格先后关系。
