# 技能详情 — Config Tab 业务流程索引

## 文件浏览与搜索

- **业务描述**: 用户进入技能详情后，在 Config Tab 中查看技能包的文件目录树，展开/折叠目录，搜索文件名定位目标文件。
- **触发入口**: 技能列表 → 点击技能名称 → 进入详情页（默认展示 Config Tab）
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/index.tsx`
- **涉及角色**: skill_owner, skill_writer, 无写权限用户
- **前置条件**: 已登录且有技能查看权限；技能包文件列表加载成功

## 文件编辑

- **业务描述**: 在文件树中点击文件名打开文件，在 Monaco Editor 中编辑源代码，编辑器自动将修改暂存并在 800ms 无操作后自动保存到后端。
- **触发入口**: 文件树 → 点击目标文件名
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/components/EditorContent.tsx`
- **涉及角色**: skill_owner, skill_writer
- **前置条件**: 已打开一个文本文件；用户具有写权限（canWrite = true）

## 文件创建

- **业务描述**: 在文件树的指定目录下创建新文件或新目录。输入名称后，后端创建空文件/目录，文件树自动刷新并打开新文件进入编辑状态。
- **触发入口**: 文件树 → 创建文件/创建文件夹按钮
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/hooks/useFileOperations.tsx`
- **涉及角色**: skill_owner, skill_writer
- **前置条件**: 用户具有写权限；父目录存在

## 文件重命名

- **业务描述**: 在文件树中选中文件或目录，输入新名称后重命名。重命名前自动刷新该路径下的待保存内容。
- **触发入口**: 文件树 → 右键菜单/操作按钮 → 重命名
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/hooks/useFileOperations.tsx`
- **涉及角色**: skill_owner, skill_writer
- **前置条件**: 用户具有写权限；目标文件/目录存在

## 文件删除

- **业务描述**: 删除文件树中的文件或目录。弹出确认对话框，用户确认后执行删除。删除目录时递归删除所有子内容。
- **触发入口**: 文件树 → 右键菜单/操作按钮 → 删除
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/hooks/useFileOperations.tsx`
- **涉及角色**: skill_owner, skill_writer
- **前置条件**: 用户具有写权限；目标文件/目录存在

## 文件上传

- **业务描述**: 将本地文件上传到技能包的指定目录。支持批量上传二进制文件（图片、音频、视频等）。
- **触发入口**: 文件树 → 上传按钮
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/hooks/useFileOperations.tsx`
- **涉及角色**: skill_owner, skill_writer
- **前置条件**: 用户具有写权限；目标目录存在

## 源码/预览切换

- **业务描述**: 对于 Markdown 和 SVG 文件，可在源码编辑视图与预览视图之间切换。Markdown 文件预览渲染为富文本，SVG 文件预览渲染为矢量图。
- **触发入口**: 打开 Markdown/SVG 文件 → 编辑器顶部切换按钮
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/components/EditorContent.tsx`
- **涉及角色**: skill_owner, skill_writer, 无写权限用户
- **前置条件**: 已打开 Markdown 或 SVG 文件

## 版本过期检测与刷新

- **业务描述**: 系统每 30 秒轮询检查技能包版本是否发生变化（其他副本/会话的编辑）。检测到过期时顶部显示黄色提示条，用户点击刷新按钮重新加载最新文件。
- **触发入口**: 页面自动轮询 或 窗口获得焦点时
- **路由路径**: `/skill/detail?skillId={skillId}`
- **页面组件位置**: `projects/app/src/pageComponents/dashboard/skill/detail/config/AgentSkillEditor/index.tsx`
- **涉及角色**: skill_owner, skill_writer, 无写权限用户
- **前置条件**: 初始版本号已获取

## 场景关系

- **文件浏览与搜索** → **文件编辑**：浏览定位文件后点击打开进入编辑
- **文件浏览与搜索** → **文件创建/重命名/删除/上传**：在目录树中定位目标位置后执行操作
- **文件编辑** → **源码/预览切换**：编辑 Markdown/SVG 文件时可切换查看方式
- 各场景之间独立，无严格先后顺序。
