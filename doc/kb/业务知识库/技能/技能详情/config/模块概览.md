# 技能详情 — Config Tab 模块概览

## 1. 模块业务概述

Config Tab 是技能详情页的核心编辑区域，为用户提供一个基于 Monaco Editor 的**技能代码包在线编辑器**。用户在此 Tab 中可以对技能包内的所有源文件进行浏览、编辑、创建、删除、重命名、上传等操作，实现对技能代码的完整管理。

该模块是「技能管理」工作台的基础能力——用户在技能列表中选择一个技能进入详情后，默认展示此编辑器，可直接修改技能包的源代码。模块服务于具备技能开发和维护权限的用户（skill_owner、skill_writer），是整个技能生命周期管理中最关键的操作界面。

## 2. 角色差异

本模块通过父级 SkillDetailContext 中的 `permission.hasWritePer` 控制编辑权限：

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| skill_owner | Config Tab 完整可见 | 浏览文件树、打开文件、编辑源码、创建/删除/重命名文件及目录、上传文件 | 拥有全部编辑权限 |
| skill_writer | Config Tab 完整可见 | 浏览文件树、打开文件、编辑源码、创建/删除/重命名文件及目录、上传文件 | 与 owner 权限一致 |
| 无写权限用户 | Config Tab 可见 | 浏览文件树、打开文件（只读模式查看） | 编辑器设为只读，无法修改源码；文件创建、删除、重命名、上传按钮不可用 |

**权限模型**：基于 `SkillPermission` 类，通过 `permission.hasWritePer` 布尔值判断。编辑器中 Monaco Editor 的 `readOnly` 选项绑定 `!canWrite`，文件操作按钮通过 `canWrite` prop 控制可用性。
