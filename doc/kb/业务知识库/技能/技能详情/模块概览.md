# 技能详情 — 模块概览

## 1. 模块业务概述

技能详情模块是 Agent Skill 的管理入口，用户在技能列表中点击某个技能后进入此页面。模块提供两个核心能力：**技能文件编辑**（config Tab）和 **技能对话预览**（preview Tab）。

该模块主要面向技能拥有者和协作者，允许他们在线编辑技能的代码文件（SKILL.md 及配套资源），并通过内置的调试对话窗口实时验证技能效果。技能详情页面以居中 Tab 切换的方式组织两种工作模式，左侧提供返回按钮和技能信息菜单，方便用户快速回到技能列表或执行编辑信息、权限配置、导出配置、删除等管理操作。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 拥有者 | 技能详情全部 | 编辑文件、预览调试、编辑信息、权限管理、导出/删除技能、转让所有权 | 全部权限 |
| 协作者 | 按权限位决定 | 取决于权限位（可写/只读） | 无权限管理、无转让、无删除权限 |

**权限模型**: 基于 `SkillPermission` 类的自定义权限位模型，通过 `permission.hasWritePer` 判断是否可写。`isOwner` 标识拥有者身份。权限继承（`inheritPermission`）通过 `parentId` 从父级资源获取。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 技能列表 | 技能详情数据（getSkillDetail API） | API |
| 应用工作台 | 技能关联的应用列表查询 | API |
| 权限管理 | ConfigPerModal 权限配置弹窗 | 组件 |
| 通用组件 | MyTabs、MyIcon、Avatar、MyMenu、EditResourceModal | 组件 |
| AI 模型选择 | AIModelSelector 模型选择器 | 组件 |
| 对话记录 | ChatItemContextProvider、ChatRecordContextProvider | 组件 |

| 共享能力 | 消费模块 | 能力类型 |
