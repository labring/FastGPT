# 数据集 — 业务流程索引

## 数据集列表

- **业务描述**: 浏览和管理所有知识库，支持创建新知识库、创建文件夹、搜索过滤、编辑知识库信息、配置权限和协作者。
- **触发入口**: 侧边栏「知识库」菜单项
- **路由路径**: `/dataset/list`
- **页面组件位置**: `projects/app/src/pages/dataset/list/index.tsx`
- **涉及角色**: 管理员、协作者、只读用户
- **前置条件**: 用户已登录且具有团队权限

## 数据集详情

- **业务描述**: 对单个知识库进行数据管理，包括浏览知识集合、导入多种数据源（文件、链接、API、外部文件、网页、文本等）、搜索测试、管理同义词、查看和修改知识库配置信息。
- **触发入口**: 数据集列表中点击知识库卡片
- **路由路径**: `/dataset/detail?datasetId={id}&currentTab={tab}`
- **页面组件位置**: `projects/app/src/pages/dataset/detail/index.tsx`
- **涉及角色**: 管理员、协作者、只读用户（Tab 可见性因角色而异）
- **前置条件**: 存在已创建的数据集

## 场景关系

- **数据集列表** → **数据集详情**: 用户在列表中点击知识库卡片进入该知识库的详情管理页面
- **数据集详情** → **数据集列表**: 通过面包屑导航返回列表或上级文件夹
