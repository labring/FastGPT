## 跨模块数据流

- **数据集列表 → 数据集详情**: 用户选择数据集后通过路由 `datasetId` 参数跳转，详情页通过 `DatasetPageContextProvider` 加载数据集详情数据
- **数据集 → 权限系统**: 通过 `ConfigPerModal` 组件管理细粒度权限，协作者数据通过独立 API 获取
- **数据集 → 对话模块**: 对话中使用数据集时，通过 `getQuoteData` API 获取引用数据，通过 `QuoteItem` 和 `RawSourceBox` 组件展示
- **数据集 → 文件服务**: 通过 `getUploadDatasetFilePresignedUrl` API 获取文件上传预签名 URL
