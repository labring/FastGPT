# 检索测试 — 模块概览

## 1. 模块业务概述

检索测试是数据集详情页中的一个功能 Tab，用于验证数据集的检索效果。用户输入测试文本后，系统按照配置的检索模式（混合检索、ReRank 重排序等）对数据集内容进行检索，返回匹配度最高的数据块及评分，帮助用户评估数据集的检索质量和参数配置是否合理。

本模块支持两种检索测试方式：
- **向量/全文检索测试**：适用于普通知识库数据集，支持 embedding 检索、全文检索、ReRank 重排序等多种模式的组合检索
- **数据库检索测试**：适用于数据库类型数据集，通过大模型将自然语言查询转为 SQL 并在数据库中执行，返回查询结果

## 2. 角色差异

本模块为数据集详情页的功能 Tab，无独立的角色权限控制。用户的访问权限由父级"数据集详情"页面统一控制。
