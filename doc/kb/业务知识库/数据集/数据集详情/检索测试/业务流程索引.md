# 检索测试 — 业务流程索引

## 普通数据集检索测试

- **业务描述**：在普通知识库数据集（向量/全文索引）上执行检索测试，支持 embedding 检索、全文检索、ReRank 重排序等多种搜索模式的组合，返回匹配数据块及其评分
- **触发入口**：数据集详情页 → 切换到"检索测试"Tab
- **路由路径**：`/dataset/detail?currentTab=test`
- **页面组件位置**：`projects/app/src/pageComponents/dataset/detail/Test.tsx`
- **涉及角色**：所有可访问数据集详情的用户
- **前置条件**：用户已进入某个普通类型数据集的详情页；数据集已完成导入和向量化训练

## 数据库数据集检索测试

- **业务描述**：在数据库类型数据集上执行检索测试，系统将用户输入的自然语言查询转为 SQL 语句并在数据库中执行，返回 SQL 查询结果和自然语言解析答案
- **触发入口**：数据集详情页 → 切换到"检索测试"Tab（当数据集为数据库类型时自动切换为数据库检索模式）
- **路由路径**：`/dataset/detail?currentTab=test`
- **页面组件位置**：`projects/app/src/pageComponents/dataset/detail/Test.tsx`
- **涉及角色**：所有可访问数据集详情的用户
- **前置条件**：用户已进入某个数据库类型数据集的详情页；数据集已配置数据库连接

## 查看检索测试历史

- **业务描述**：查看当前数据集下所有的检索测试历史记录，点击历史记录可回看该次测试的检索结果和参数
- **触发入口**：检索测试 Tab 左侧面板 → 历史记录列表
- **路由路径**：`/dataset/detail?currentTab=test`
- **涉及角色**：所有可访问数据集详情的用户
- **前置条件**：当前数据集已有检索测试历史记录

## 场景关系

各场景独立执行，无严格先后关系。普通数据集检索测试和数据库数据集检索测试互斥，由数据集类型决定使用哪种模式。
