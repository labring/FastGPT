# 集合管理 — 业务流程索引

## 集合列表浏览

- **业务描述**: 进入数据集详情页的集合管理 Tab，查看数据集内所有集合的列表，支持分页、搜索、按状态/标签筛选、按名称/时间/数据量排序。
- **触发入口**: 数据集详情页 → 点击「集合管理」Tab（默认 Tab）
- **路由路径**: `/dataset/detail?currentTab=collectionCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedCollectionCard/index.tsx`
- **涉及角色**: owner、collaborator、viewer
- **前置条件**: 用户有该数据集的读权限；数据集已加载

## 单个集合操作

- **业务描述**: 对表格中的单个集合执行操作，包括查看/下载源文件、重命名、移动、删除、标签管理、权限配置、同步（仅链接类型）。
- **触发入口**: 集合列表 → 点击行尾操作菜单（三个点图标）
- **路由路径**: `/dataset/detail?currentTab=collectionCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedCollectionCard/index.tsx`
- **涉及角色**: owner、collaborator（hasWritePer 或 hasManagePer）
- **前置条件**: 用户有该集合的写权限

## 批量操作

- **业务描述**: 选中多个集合后，对选中的集合执行批量操作：批量删除、批量下载、批量同步（链接类型）、批量设置标签。
- **触发入口**: 勾选集合前的 Checkbox → 底部浮动操作栏
- **路由路径**: `/dataset/detail?currentTab=collectionCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedCollectionCard/index.tsx`
- **涉及角色**: owner、collaborator（hasWritePer 或 hasManagePer）
- **前置条件**: 至少选中一个集合；被批量删除/下载/同步的集合均为可见

## 数据库表管理

- **业务描述**: 当数据集类型为数据库（database）时，使用 DatabaseListTable 组件渲染数据库表格式列表，支持编辑数据结构、查看训练状态、移除表等操作。
- **触发入口**: 数据集详情页 → 集合管理 Tab（数据库数据集）
- **路由路径**: `/dataset/detail?currentTab=collectionCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedCollectionCard/index.tsx` + `DatabaseListTable`
- **涉及角色**: owner、collaborator
- **前置条件**: 数据集类型为 database；数据集有 databaseConfig

## 场景关系

- **集合列表浏览** → **单个集合操作**: 浏览是操作的前置场景，用户先浏览列表再对具体集合执行操作
- **集合列表浏览** → **批量操作**: 浏览是批量操作的前置场景，用户先勾选集合再执行批量操作
- **数据库表管理**: 与普通集合列表互斥，取决于数据集类型
