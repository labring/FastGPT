# 数据集配置 — 业务流程索引

## 业务场景列表

### 查看数据集配置

- **业务描述**: 进入数据集详情页的配置Tab，查看数据集的基本信息、模型配置、同步策略和服务连接信息。
- **触发入口**: 数据集列表 → 点击数据集名称进入详情 → 点击"配置"Tab
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员、协作者
- **前置条件**: 用户已登录且具有该数据集的访问权限

### 编辑数据集基本信息

- **业务描述**: 管理员修改数据集的名称、头像和简介。
- **触发入口**: 数据集配置页 → 点击名称旁的编辑图标
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`（调用 EditResourceModal）
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限

### 切换向量模型并重建索引

- **业务描述**: 管理员更换数据集使用的向量模型，系统重建全部向量嵌入索引。
- **触发入口**: 数据集配置页 → 向量模型下拉选择器
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限；非结构化文档类型；数据集不在训练中

### 切换Agent模型

- **业务描述**: 管理员更换数据集使用的Agent对话模型。
- **触发入口**: 数据集配置页 → Agent模型下拉选择器
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限；该数据集类型支持Agent模型配置

### 切换VLM多模态模型

- **业务描述**: 管理员配置或更换数据集使用的视觉语言模型。
- **触发入口**: 数据集配置页 → VLM模型下拉选择器
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限；该数据集类型支持VLM模型配置

### 配置自动同步

- **业务描述**: 管理员开启或关闭数据集的自动同步功能（Plus版功能）。
- **触发入口**: 数据集配置页 → 同步计划开关
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限；为Plus版功能；非数据库类型数据集

### 配置外部读取地址

- **业务描述**: 为外部文件类型数据集设置外部文件读取地址模板。
- **触发入口**: 数据集配置页 → 外部读取地址输入框
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 数据集类型为 externalFile

### 编辑API数据集配置

- **业务描述**: 管理API数据集、语雀或飞书类型数据集的服务连接配置。
- **触发入口**: 数据集配置页 → API地址/语雀配置/飞书配置旁的编辑图标
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`（调用 EditAPIDatasetInfoModal）
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 数据集类型为 apiDataset / yuque / feishu

### 管理协作者

- **业务描述**: 管理员添加、修改或移除数据集的协作者及其权限。
- **触发入口**: 数据集配置页 → 协作者区域的设置图标
- **路由路径**: `/dataset/detail?currentTab=info`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Info/index.tsx`（调用 MemberManager → CollaboratorContextProvider → MemberModal）
- **涉及角色**: 管理员（hasManagePer）
- **前置条件**: 用户具有管理权限

## 场景关系

- **查看数据集配置** 是其他所有场景的入口页面，其他场景均在此页面内操作
- **切换向量模型**操作会触发重建嵌入索引，训练期间该操作被禁用
- **切换Agent模型**和**切换VLM模型**各自独立，互不影响
- **编辑API数据集配置**的弹窗内容根据数据集类型（API/语雀/飞书）显示不同字段
- 各配置场景独立执行，无严格先后关系
