# 结构化数据卡片 — 模块概览

## 模块业务概述

结构化数据卡片是数据集详情页的子视图，用于预览**文件数据库类型知识库**中某个集合（Collection）的结构化数据内容。当用户在数据集详情页点击某个文件型数据集合时，系统会切换到本视图，以**只读表格**形式展示该集合的列结构和前 20 行数据，帮助用户快速验证导入数据的结构和内容是否符合预期。

本模块面向**所有可访问数据集的用户**，仅提供数据查看功能，无编辑、删除或导出操作。

## 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 有数据集访问权限的用户 | 结构化数据预览页 | 查看表格数据、返回上级 | 所有有权限用户可见内容一致 |

**权限模型**: 依赖数据集详情页的路由权限控制。用户需先拥有目标数据集的访问权限才能进入数据集详情页，进而查看本视图。无独立权限控制。

## 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 数据集 API 层 | `getStructureCollectionPreview` — 获取结构化集合预览数据 | API |
| 数据集 API 层 | `getDatasetCollectionById` — 获取集合基本信息 | API |
| Web 共享组件 | `MyBox` — 带加载状态的容器组件 | 组件 |
| Web 共享组件 | `MyIcon`、`MyTooltip`、`EmptyTip` — 通用 UI 组件 | 组件 |
| NavBar | `TabEnum` — 数据集详情页 Tab 枚举定义 | 常量 |

## 对外共享

本模块为数据集详情页的内部子视图，不对其他模块提供共享能力。
