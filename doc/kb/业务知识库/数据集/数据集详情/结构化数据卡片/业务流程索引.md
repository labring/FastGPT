# 结构化数据卡片 — 业务流程索引

## 业务场景列表

### 查看结构化数据预览

- **业务描述**: 用户在数据集详情页点击一个文件数据库类型的集合后，进入数据预览视图，以表格形式查看该集合的列结构和前 20 行数据。
- **触发入口**: 数据集详情页 → 集合列表中点击某个集合名称
- **路由路径**: `/dataset/detail?currentTab=fileDataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/FileDataCard.tsx`
- **涉及角色**: 有数据集访问权限的用户
- **前置条件**: 已进入数据集详情页（`/dataset/detail`），且 URL 中携带 `collectionId` 参数
