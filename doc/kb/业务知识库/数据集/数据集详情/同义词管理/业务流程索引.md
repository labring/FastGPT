# 同义词管理 — 业务流程索引

## 查看同义词文件

- **业务描述**: 进入同义词管理 Tab 后，系统自动加载已上传的同义词文件信息（如果有），展示文件名、大小、上传者、上传时间等元信息。
- **触发入口**: 数据集详情页 → 点击"同义词"Tab
- **路由路径**: `/dataset/detail?currentTab=synonym`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Synonym/index.tsx`
- **涉及角色**: 管理员、成员
- **前置条件**: 已登录且有该数据集的读取权限

## 上传同义词文件

- **业务描述**: 用户拖拽或选择一个 xlsx/xls/csv 格式的同义词文件上传到服务器，系统处理后使知识库检索支持同义词映射。支持不超过 50MB 的单文件上传。
- **触发入口**: 同义词管理页面 → 点击或拖拽文件到上传区域
- **路由路径**: `/dataset/detail?currentTab=synonym`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Synonym/index.tsx`
- **涉及角色**: 管理员、成员
- **前置条件**: 已登录且有该数据集的写入权限；当前无正在上传的文件

## 下载同义词模板

- **业务描述**: 用户下载一个预格式化的 XLSX 模板文件，用于按规范填写同义词映射关系。模板由前端基于 ExcelJS 库实时生成，包含表头和示例数据。
- **触发入口**: 同义词管理页面空状态 → 点击"下载模板"链接
- **路由路径**: `/dataset/detail?currentTab=synonym`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Synonym/index.tsx`
- **涉及角色**: 管理员、成员
- **前置条件**: 页面处于空状态（无已上传文件且无上传中的文件）

## 下载同义词文件

- **业务描述**: 用户下载已上传的同义词文件到本地，用于查看或修改内容。
- **触发入口**: 同义词管理页面 → 已上传文件信息行 → 点击下载图标
- **路由路径**: `/dataset/detail?currentTab=synonym`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Synonym/index.tsx`
- **涉及角色**: 管理员、成员
- **前置条件**: 存在已上传的同义词文件

## 删除同义词文件

- **业务描述**: 用户删除已上传的同义词文件。删除前弹出二次确认弹窗，确认后执行删除操作。
- **触发入口**: 同义词管理页面 → 已上传文件信息行 → 点击删除图标 → 确认弹窗中选择"删除"
- **路由路径**: `/dataset/detail?currentTab=synonym`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/Synonym/index.tsx`
- **涉及角色**: 管理员、成员
- **前置条件**: 存在已上传的同义词文件；有该数据集的写入权限

## 场景关系

- **查看同义词文件** → **上传同义词文件**：查看页面由空状态切换至文件上传区域
- **上传同义词文件** → **查看同义词文件**：上传成功后自动刷新显示新文件
- **删除同义词文件** → **查看同义词文件**：删除成功后回退到空状态
- **下载同义词模板**：独立操作，不依赖其他场景
- **下载同义词文件**：依赖于"查看同义词文件"场景（需有已上传文件）
