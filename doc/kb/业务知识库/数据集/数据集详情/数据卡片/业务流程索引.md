# 数据卡片 — 业务流程索引

> 本文档描述数据卡片模块覆盖的所有业务场景。场景交互细节见 [10-业务流程详解](./业务流程详解.md)。

## 浏览数据列表

- **业务描述**: 在数据集详情页中查看当前集合的数据分页列表，支持搜索过滤、折叠展开卡片，以及查看选中数据项的详细内容索引。
- **触发入口**: 数据集详情页 → 点击集合列表中的某个集合 → 自动跳转至 dataCard Tab
- **路由路径**: `/dataset/detail?currentTab=dataCard&datasetId={id}&collectionId={id}`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/index.tsx`
- **涉及角色**: 管理员/编辑者、只读用户
- **前置条件**: 数据集详情页已加载；当前集合存在且非空

## 新增数据

- **业务描述**: 在当前集合中新增一条数据内容，支持文本输入和 FAQ 问答对两种模式。
- **触发入口**: 数据卡片页 → 点击搜索栏右侧"新增数据"按钮
- **路由路径**: `/dataset/detail?currentTab=dataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/EditContentModal.tsx`
- **涉及角色**: 管理员/编辑者
- **前置条件**: 当前用户具有写权限；集合训练类型非图片解析模式

## 编辑数据

- **业务描述**: 编辑已有数据的 Q（问题/内容）和 A（回答，FAQ 模式下）字段。
- **触发入口**: 数据卡片页 → 鼠标悬停数据卡片 → 点击编辑按钮
- **路由路径**: `/dataset/detail?currentTab=dataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/EditContentModal.tsx`
- **涉及角色**: 管理员/编辑者
- **前置条件**: 当前用户具有写权限；目标数据项不在训练中状态

## 删除数据

- **业务描述**: 删除单条数据，删除后自动刷新列表。
- **触发入口**: 数据卡片页 → 鼠标悬停数据卡片 → 点击删除按钮 → 确认弹窗
- **路由路径**: `/dataset/detail?currentTab=dataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/index.tsx`
- **涉及角色**: 管理员/编辑者
- **前置条件**: 当前用户具有写权限

## 管理索引内容

- **业务描述**: 查看、新增、编辑、删除某条数据关联的内容索引（知识点）。支持自定义索引的增删改。
- **触发入口**: 数据卡片页 → 选中一条数据 → 右侧面板显示索引列表 → 操作索引
- **路由路径**: `/dataset/detail?currentTab=dataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/ContentIndexCard.tsx`
- **涉及角色**: 管理员/编辑者（增删改）、只读用户（仅查看）
- **前置条件**: 已选中一条数据项；写权限用户可操作索引

## 查看异常数据

- **业务描述**: 当集合存在错误数据时，点击异常计数标签查看异常数据详情。
- **触发入口**: 数据卡片页 → 点击搜索栏中的红色异常计数标签
- **路由路径**: `/dataset/detail?currentTab=dataCard`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedCollectionCard/ExceptionInfoModal.tsx`
- **涉及角色**: 管理员/编辑者、只读用户
- **前置条件**: 当前集合的 `errorCount` > 0

## 通过 activeId 定位数据

- **业务描述**: 通过 URL 参数 `activeId` 指定某条数据，页面自动计算其所在分页并滚动定位到该数据卡片。
- **触发入口**: 从其他页面携带 `activeId` 参数进入数据卡片页
- **路由路径**: `/dataset/detail?currentTab=dataCard&activeId={dataId}`
- **页面组件位置**: `projects/app/src/pageComponents/dataset/detail/RefinedDataCard/index.tsx`
- **涉及角色**: 管理员/编辑者、只读用户
- **前置条件**: URL 中包含 `activeId` 参数；目标数据存在且未删除

## 场景关系

- **浏览数据列表** → **新增数据**：浏览列表后可点击新增按钮添加数据
- **浏览数据列表** → **编辑数据**：选中数据卡片后可编辑其内容
- **浏览数据列表** → **删除数据**：选中数据卡片后可删除该数据
- **浏览数据列表** → **管理索引内容**：选中数据卡片后可在右侧面板管理索引
- **浏览数据列表** → **查看异常数据**：存在异常数据时可查看异常详情
- **通过 activeId 定位数据** → **浏览数据列表**：定位后自动展开目标卡片，继续浏览

各场景之间无严格先后依赖关系，用户可自由切换。
