# 数据卡片 — 模块概览

## 模块业务概述

数据卡片是数据集详情页中用于**查看、编辑和管理单个知识库集合内的数据内容**的核心模块。当用户在数据集详情页切换到"数据卡片"Tab 时展示，提供数据列表的搜索、浏览、折叠展开以及选中数据项的索引内容管理功能。

本模块服务于两类角色：具有写权限的管理员/编辑者可对数据进行增删改和索引内容编辑；只读用户仅可浏览数据和索引内容。

## 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 管理员/编辑者 | 数据卡片页 | 搜索数据、浏览数据列表、折叠展开数据卡片、查看索引内容、新增数据、编辑数据、删除数据、新增索引、编辑索引、删除索引 | 具有完整的数据管理权限 |
| 只读用户 | 数据卡片页 | 搜索数据、浏览数据列表、折叠展开数据卡片、查看索引内容 | 无编辑权限，不显示新增/编辑/删除按钮 |

**权限模型**：基于 `datasetDetail.permission.hasWritePer` 的写权限控制。`canWrite` 为 `true` 时显示新增数据按钮、编辑/删除操作按钮、新增/编辑/删除索引按钮。

## 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |

| 数据集 API 层 | getDatasetDataList、getDatasetDataItemById、putDatasetDataById、getDatasetDataIndex、delOneDatasetDataById、getDatasetCollectionById、postInsertData2Dataset（via EditContentModal） | API |
| Web 通用组件库（@fastgpt/web） | MyBox、EmptyTip、MyIcon、MyDivider、MyImage、FormLabel、MyTooltip、MyTag、PopoverConfirm | 组件 |

| 集合卡片模块（RefinedCollectionCard） | ExceptionInfoModal 异常信息弹窗 | 组件 |

## 对外共享

本模块为数据集详情页的内部子模块，不被其他外部模块直接引用。入口组件 `RefinedDataCard` 通过 Next.js `dynamic()` 懒加载，仅在数据集详情页的 `dataCard` Tab 下渲染。
