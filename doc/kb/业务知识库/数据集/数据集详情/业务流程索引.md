# 数据集详情 — 业务流程索引

## 业务场景列表

### 进入数据集详情

- **业务描述**: 从数据集列表点击某个数据集进入详情页，默认展示集合管理 Tab
- **触发入口**: 数据集列表 → 点击数据集名称或操作按钮
- **路由路径**: `/dataset/detail?datasetId={id}`
- **页面组件位置**: `projects/app/src/pages/dataset/detail/index.tsx`
- **涉及角色**: 所有用户
- **前置条件**: 用户已登录，且对该数据集有读取权限；若数据集不存在或加载失败，自动跳回数据集列表并提示错误

### 数据集详情页面公共功能

- **业务描述**: 页面加载时完成数据集信息获取、权限校验、训练队列监控等公共初始化
- **触发入口**: 页面路由进入时自动触发
- **路由路径**: `/dataset/detail?datasetId={id}`
- **涉及角色**: 所有用户
- **前置条件**: 有效的 `datasetId` 查询参数

## 场景关系

各 Tab 场景并列独立，无严格先后依赖关系。用户可通过 Tab 栏自由切换：
- **集合管理** → 点击集合可进入 **数据卡片** 或 **结构化数据卡片**
- **导入数据** 完成后自动回到 **集合管理**
- 各 Tab 之间无数据流转依赖，每次切换只重新获取该 Tab 所需数据
