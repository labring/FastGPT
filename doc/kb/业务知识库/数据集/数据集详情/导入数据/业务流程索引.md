# 导入数据 — 业务流程索引

## 业务场景列表

### 本地文件导入

- **业务描述**: 用户上传本机文件（文档、图片等），将其解析为知识库集合。支持同时上传多个文件，图片和文档文件走不同处理路径。
- **触发入口**: 数据集详情页 → 导航栏 → 点击"导入数据"Tab → 弹出导入源选择器 → 选择「本地文件」
- **路由路径**: `/dataset/detail?currentTab=import&source=fileLocal`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/FileLocal.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 用户具有数据集的 `hasManagePer` 权限；数据集已创建

### 网页链接导入

- **业务描述**: 用户粘贴一个或多个网页链接（URL），系统抓取网页内容并解析为知识库集合。支持对网页指定 CSS 选择器以精准抓取内容区域。
- **触发入口**: 数据集详情页 → 导入 Tab → 导入源选择器 → 选择「网页链接」
- **路由路径**: `/dataset/detail?currentTab=import&source=fileLink`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/FileLink.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 链接必须为有效的 HTTP/HTTPS URL

### 自定义文本导入

- **业务描述**: 用户手动输入一段文本内容和名称，将其作为知识库集合。适用于临时性、短文本的知识录入场景。
- **触发入口**: 数据集详情页 → 导入 Tab → 导入源选择器 → 选择「自定义文本」
- **路由路径**: `/dataset/detail?currentTab=import&source=fileCustom`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/FileCustomText.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 需输入集合名称和文本内容

### 外部文件导入

- **业务描述**: 用户提供外部文件的 URL 地址和文件 ID，系统从外部源拉取文件并解析为知识库集合。
- **触发入口**: 数据集详情页 → 导入 Tab → 导入源选择器 → 选择「外部文件」（需在 URL 参数中指定）
- **路由路径**: `/dataset/detail?currentTab=import&source=externalFile`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/ExternalFile.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 需提供有效的外部文件 URL 和对应的文件 ID

### API 数据集导入

- **业务描述**: 用户从已对接的外部 API 数据源浏览文件目录，选择文件或整个文件夹作为知识库集合。适用于企业已对接的第三方数据平台。
- **触发入口**: 数据集详情页 → 导入 Tab → API 数据集类型知识库的对应入口
- **路由路径**: `/dataset/detail?currentTab=import&source=apiDataset`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/APIDataset.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 数据集类型为 API 数据集（`ApiDatasetTypeMap` 中存在）；外部 API 服务已配置

### 重训练

- **业务描述**: 用户选择已有的知识库集合，重新调整数据处理参数（分块策略、索引配置等）后进行重新训练。适用于优化已有集合的数据质量。
- **触发入口**: 数据集详情页 → 集合卡片 → 操作菜单 → 选择「重训练」
- **路由路径**: `/dataset/detail?currentTab=import&source=reTraining&collectionId={id}`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/ReTraining.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 存在已有的集合；传入有效的 `collectionId`

### 图片数据集导入

- **业务描述**: 用户上传一批图片，创建图片类型的知识库集合。图片不经文本分块处理，直接以原图形式入库。
- **触发入口**: 数据集详情页 → 导入 Tab → 图片知识库类型入口
- **路由路径**: `/dataset/detail?currentTab=import&source=imageDataset`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/ImageDataset.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 支持 `.jpg`、`.jpeg`、`.png` 格式的图片文件

### 数据库直连导入

- **业务描述**: 用户配置数据库连接信息，选择数据库中的表和字段，将结构化的数据库数据映射为知识库集合。支持 MySQL、PostgreSQL、MSSQL、Oracle。
- **触发入口**: 数据集详情页 → 导入 Tab → 数据库类型知识库的对应入口
- **路由路径**: `/dataset/detail?currentTab=import&source=database`
- **页面组件位置**: `pageComponents/dataset/detail/Import/index.tsx` → `diffSource/ConnectDatabaseConfig.tsx`
- **涉及角色**: 管理员、有管理权限的成员
- **前置条件**: 数据库类型知识库已创建；数据库网络可达

## 场景关系

- 各导入方式独立执行，无严格先后关系。同一数据集中可混合使用多种导入方式。
- **重训练**与其它方式存在隐式依赖——需要先通过其它方式导入数据后，才存在可重训练的集合。
- **数据预览**和**参数配置**是文件导入、链接导入、文本导入、外部文件导入、API 导入、重训练共用的中间步骤，共享 `commonProgress/` 组件。
