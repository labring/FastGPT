# 首页 — 模块概览

## 1. 模块业务概述

首页是对话页面的默认面板（`ChatSidebarPaneEnum.HOME`），用户在此选择 AI 模型和工具后直接开始对话，或一键切换至预设的快速应用。它是 FastGPT 对话功能最直接的入口——无需先创建应用即可自由组合模型与工具进行 AI 交互。

本模块是对话首页的 Tab 子能力之一，当用户在侧边栏切换到"首页"面板时展示。页面提供两种对话模式：

- **快速应用模式**：用户侧边栏中预设的快速应用，一键切换即可使用该应用已配置的知识库、工作流和提示词进行对话。
- **自定义模式**：用户从下拉菜单中自由选择 AI 模型，并可多选工具（插件），以自定义配置发起对话。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 登录用户（非 Plus） | 团队应用面板 | 不可见首页 | 非商业版用户无法访问首页面板，自动跳转至团队应用 |
| Plus版用户 | 首页面板 | 选择模型、选择工具、切换快速应用、发起对话 | 可使用全部首页功能 |
| 管理员（Plus版） | 首页面板 + 设置面板 | 同上 + 管理首页配置 | 可通过设置面板管理快速应用列表、首页标语等 |

**权限模型**：首页面板的可见性由 `feConfigs.isPlus` 开关控制。非 Plus 用户进入 `/chat` 时自动跳转至 `TEAM_APPS` 面板。页面内部无额外的角色级别权限校验——只要能看到首页面板，即可使用全部对话功能。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 对话核心 | ChatBox 聊天容器（对话渲染、流式响应、输入发送） | 组件 |
| 对话核心 | ChatContext / ChatItemContext / ChatPageContext / ChatRecordContext（对话状态、聊天记录） | Context |
| 对话核心 | useChatStore（对话 appId/chatId 管理） | Store |
| 用户系统 | useUserStore（用户信息、头像） | Store |
| 系统配置 | useSystemStore（feConfigs、llmModelList、defaultModels） | Store |
| 对话侧边栏 | ChatHistorySidebar / ChatSliderMobileDrawer（对话历史列表） | 组件 |
| 应用管理 | onChangeGlobalAppId（切换应用） | Hook |
| 应用 API | getToolPreviewNode（获取工具节点预览） | API |
| 对话 API | getInitChatInfo（初始化对话信息） | API |

本模块为叶子节点，主要消费外部能力，不对外提供大量共享能力。
