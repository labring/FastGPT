# 精选应用 — 业务流程索引

## 浏览精选应用列表

- **业务描述**: 用户在对话首页切换到精选应用面板，按标签筛选或按名称搜索应用，浏览管理员配置的精选应用集合。
- **触发入口**: 对话首页侧边栏 → 点击"精选应用"导航项（或在移动端点击菜单图标打开侧边栏后操作）
- **路由路径**: `/chat?pane=fa`（通过 `ChatSidebarPaneEnum.FAVORITE_APPS` 切换面板）
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatFavouriteApp/index.tsx`
- **涉及角色**: 普通用户、管理员
- **前置条件**: 用户已登录，且为 Plus 版本（`feConfigs.isPlus` 为 true）

## 跳转配置精选应用

- **业务描述**: 管理员在精选应用列表为空时，点击"去配置"按钮跳转到精选应用管理页面，进行应用的添加、排序和分类管理。
- **触发入口**: 精选应用面板空状态 → 点击"去配置"按钮
- **路由路径**: `/chat?pane=s&tab=f`（通过 `handlePaneChange(SETTING, undefined, FAVOURITE_APPS)` 切换）
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/index.tsx`（设置面板，非本模块）
- **涉及角色**: 管理员
- **前置条件**: 用户具有管理权限（`userInfo.permission.hasManagePer`），且当前精选应用列表为空

## 场景关系

- **浏览精选应用列表** → **点击应用卡片进入对话**: 用户点击任意应用卡片后，面板切换到最近使用应用面板（`RECENTLY_USED_APPS`），加载对应应用对话
- **浏览精选应用列表** → **跳转配置精选应用**: 仅管理员在列表为空时可见此操作
