# 最近使用 — 业务流程索引

## 业务场景列表

### 打开最近使用的应用进行对话

- **业务描述**: 用户在侧边栏"最近使用"列表中点击某个应用，进入该应用的聊天窗口，可以查看历史对话记录并继续对话或开始新对话。
- **触发入口**: 对话页左侧侧边栏 → "最近使用"区域 → 点击应用名称
- **路由路径**: /chat?pane=ra&appId={appId}
- **页面组件位置**: projects/app/src/pages/chat/index.tsx（入口）→ projects/app/src/pageComponents/chat/ChatWindow/AppChatWindow.tsx
- **涉及角色**: 所有已登录用户
- **前置条件**: 
  - 用户已登录
  - 该应用存在于用户的最近使用列表中
  - 用户有权限访问该应用

### 对话完成后刷新最近使用列表

- **业务描述**: 用户与 AI 应用完成一轮对话后，系统自动刷新"最近使用"列表，确保该应用出现在列表顶部。
- **触发入口**: 自动触发（对话完成后无需用户额外操作）
- **路由路径**: /chat（当前页面内自动刷新）
- **页面组件位置**: projects/app/src/web/core/chat/context/chatPageContext.tsx（数据层）
- **涉及角色**: 所有已登录用户
- **前置条件**: 已完成一轮有效对话

### 查看对话历史记录

- **业务描述**: 用户在当前应用聊天窗口中查看之前的对话记录，可切换到不同历史会话继续对话。
- **触发入口**: 聊天窗口顶部 ChatHeader 中的历史记录切换控件
- **路由路径**: /chat?pane=ra&appId={appId}
- **页面组件位置**: projects/app/src/pageComponents/chat/ChatWindow/AppChatWindow.tsx → ChatHistorySidebar / ChatSliderMobileDrawer
- **涉及角色**: 所有已登录用户
- **前置条件**: 用户已进入某个应用聊天窗口，该应用有历史对话记录

### 侧边栏查看最近使用应用列表

- **业务描述**: 用户在对话页侧边栏浏览最近对话过的 AI 应用列表（最多20个），列表每30秒自动轮询刷新。
- **触发入口**: 对话页 → 侧边栏"最近使用"区域（非折叠状态下可见）
- **路由路径**: /chat
- **页面组件位置**: projects/app/src/pageComponents/chat/slider/index.tsx
- **涉及角色**: 所有已登录用户
- **前置条件**: 用户已登录，侧边栏处于展开状态

## 场景关系

- **侧边栏查看最近使用应用列表** → **打开最近使用的应用进行对话**: 侧边栏列表是对话的入口，用户先从列表中选择应用再进入对话
- **打开最近使用的应用进行对话** → **查看对话历史记录**: 进入应用后可查看历史记录
- **对话完成后刷新最近使用列表**: 对话完成后自动触发，不依赖用户主动操作
