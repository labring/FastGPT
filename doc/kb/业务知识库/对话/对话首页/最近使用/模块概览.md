# 最近使用 — 模块概览

## 1. 模块业务概述

"最近使用"是对话首页的子模块，为用户提供快速回到最近对话过的 AI 应用的入口。当用户在侧边栏"最近使用"列表中点击某个应用时，对话主区域将切换到该应用的聊天窗口，用户可以继续之前的对话或开始新对话。

本模块是对话首页三大核心面板之一（首页推荐、收藏应用、最近使用），面向所有已登录用户，无需额外权限。模块的核心价值是降低用户重复查找应用的认知成本——通过自动记录用户最近交互的应用并展示在侧边栏，用户可以一键切换回常用的 AI 助手。

本模块为叶子节点，无子能力。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 已登录用户 | 最近使用面板（聊天窗口） | 查看最近使用的应用列表、切换应用进行对话、发起新对话、查看历史对话记录 | 基础权限 |
| 未登录用户 | 无 | 无 | 未登录用户无法访问对话页面，会被重定向到登录页 |

**权限模型**: 基于用户登录状态的前端路由守卫。API 层通过 `authUserPer({ authToken: true, authApiKey: true })` 验证用户身份，根据 `tmbId`（团队成员ID）查询个人最近使用的应用记录。未登录用户在 `ChatContent` 组件中被拦截并展示登录弹窗。

## 3. 跨模块依赖摘要

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 对话首页 | ChatPageContext（面板切换、最近使用数据）、ChatContext（对话初始化、标题更新）、ChatItemContext（对话配置）、ChatRecordContext（对话记录） | Context/Store |
| 对话首页 | ChatSlider 侧边栏（最近使用列表展示与切换入口） | 组件 |
| 对话首页 | ChatHeader（应用名称、历史记录导航） | 组件 |
| 共享基础设施 | ChatBox 聊天容器组件（消息展示、输入、流式响应） | 组件 |
| 共享基础设施 | useRequest Hook（异步请求管理、轮询） | Hook |

## 4. 对外共享

本模块不对外提供共享能力。模块内的 `AppChatWindow` 组件仅在对话首页内部使用（由 `projects/app/src/pages/chat/index.tsx` 根据 pane 状态条件渲染）。
