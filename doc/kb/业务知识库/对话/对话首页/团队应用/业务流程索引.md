# 团队应用 — 业务流程索引

## 业务场景列表

### 浏览团队应用

- **业务描述**: 用户进入对话页面的团队应用面板，查看当前团队下所有可用的应用列表，以卡片形式展示应用名称、头像、简介、创建者和更新时间。
- **触发入口**: 对话页面左侧面板 → 切换到"团队应用"视图（`pane=ta`）
- **路由路径**: 对话页面（`/chat` 或 `/chat?appId=...` 下的 `pane=ta` 查询参数）
- **页面组件位置**: `pageComponents/chat/ChatTeamApp/index.tsx`
- **涉及角色**: 所有角色
- **前置条件**: 用户已登录并属于某个团队

### 按类型筛选应用

- **业务描述**: PC 端用户通过顶部 Tab 栏切换应用类型（全部/智能客服/工作流机器人/工作流工具），系统仅展示匹配类型的应用卡片。
- **触发入口**: 团队应用面板 → 点击顶部类型 Tab
- **路由路径**: 对话页面（`pane=ta`）
- **页面组件位置**: `pageComponents/chat/ChatTeamApp/index.tsx`
- **涉及角色**: 所有角色
- **前置条件**: PC 端访问

### 搜索应用

- **业务描述**: 用户在搜索框中输入关键词，系统按应用名称进行实时过滤（500ms 防抖），展示匹配的应用卡片。
- **触发入口**: 团队应用面板 → 搜索输入框
- **路由路径**: 对话页面（`pane=ta`）
- **页面组件位置**: `pageComponents/chat/ChatTeamApp/index.tsx`
- **涉及角色**: 所有角色
- **前置条件**: 无

### 进入文件夹浏览

- **业务描述**: 用户点击文件夹类型的应用卡片，进入该文件夹查看其子应用。顶部的面包屑导航展示当前路径，用户可点击面包屑返回上级文件夹。
- **触发入口**: 团队应用面板 → 点击文件夹卡片
- **路由路径**: 对话页面（`pane=ta&parentId=...`）
- **页面组件位置**: `pageComponents/chat/ChatTeamApp/index.tsx`
- **涉及角色**: 所有角色
- **前置条件**: 当前团队下存在文件夹类型的应用

### 开始对话

- **业务描述**: 用户在应用卡片列表中点击某个非文件夹应用，系统将导航至该应用的对话窗口，用户可直接开始与 AI 对话。
- **触发入口**: 团队应用面板 → 点击应用卡片
- **路由路径**: 对话页面（更新 `appId` 查询参数，面板切换到 `RECENTLY_USED_APPS`）
- **页面组件位置**: `pageComponents/chat/ChatTeamApp/List.tsx`
- **涉及角色**: 所有角色
- **前置条件**: 应用存在且用户有访问权限

## 场景关系

- **浏览团队应用** → **按类型筛选应用**: 筛选是浏览的子操作，在浏览基础上缩小范围
- **浏览团队应用** → **搜索应用**: 搜索是浏览的子操作，在浏览基础上按关键词过滤
- **浏览团队应用** → **进入文件夹浏览**: 文件夹是层级导航，进入子级目录
- **浏览团队应用** → **开始对话**: 最终目标场景
