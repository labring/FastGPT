# 收藏应用 — 模块概览

## 1. 模块业务概述

收藏应用模块是对话首页「设置」面板下的一个子功能，面向**团队管理员**，用于管理「精选应用」列表——即展示在团队对话首页快捷入口的应用集合。管理员可通过本模块配置哪些应用作为精选推荐、如何分类展示、以及排序优先级，最终影响所有团队成员的对话首页体验。

本模块是 FastGPT 商业版（Plus）专属功能，必须在团队管理员角色且已启用商业授权（`isPlus`）的前提下才能访问。模块本身为叶子节点，无子能力。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 团队管理员 | 收藏应用管理页 | 查看列表、搜索筛选、拖拽排序、添加应用、删除应用、管理分类标签 | 完整管理权限 |
| 普通团队成员 | 无 | 无法访问本模块 | 普通成员在对话首页仅可使用已配置的精选应用，无法进入管理页面 |

**权限模型**: 基于 `userInfo.team.permission.hasManagePer` 和 `feConfigs.isPlus` 的双重校验。在 ChatSetting 父组件 `useMount` 中检查，不满足条件时自动跳转到团队应用面板。收藏应用 Tab（`ChatSettingTabOptionEnum.FAVOURITE_APPS`）仅在管理权限 + 商业版环境下可见。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|
| 对话设置（父模块） | 路由导航、SettingHeader 插槽、Tab 切换、权限前置校验 | 组件/路由 |
| 对话页上下文 | ChatPageContext — chatSettings（含 favouriteTags）、refreshChatSetting、handlePaneChange | Context |
| 应用管理 | getMyApps（应用树浏览）、getAppFolderPath（目录路径） | API |
| 系统配置 | useSystemStore — feConfigs.isPlus（商业版开关）、isPc（响应式布局） | Store |
| 用户认证 | useUserStore — userInfo.team.permission.hasManagePer（管理权限） | Store |

## 4. 对外共享

| 共享能力 | 消费模块 | 能力类型 |
|---------|---------|---------|

| 收藏应用条目数据 | ChatFavouriteApp — 卡片列表渲染 | API 数据（共享 getFavouriteApps） |
