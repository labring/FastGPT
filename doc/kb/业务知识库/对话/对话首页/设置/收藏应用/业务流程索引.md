# 收藏应用 — 业务流程索引

## 1. 业务场景列表

### 查看收藏应用列表

- **业务描述**: 管理员进入收藏应用管理页面，查看当前团队已配置的精选应用列表，支持按名称搜索和按分类标签筛选。
- **触发入口**: 对话首页 → 设置面板 → 顶部 Tab 栏点击「精选应用」（仅在管理权限 + 商业版下可见）
- **路由路径**: `/chat?pane=s&tab=f`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/FavouriteAppSetting/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 已登录，当前团队拥有商业版授权，用户具有团队管理权限（`hasManagePer`）

### 添加收藏应用

- **业务描述**: 管理员从应用树（支持目录浏览和搜索）中勾选一个或多个应用，将其加入精选应用列表。
- **触发入口**: 收藏应用管理页 → 顶部工具栏「添加应用」按钮
- **路由路径**: `/chat?pane=s&tab=f`（模态弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/FavouriteAppSetting/AddFavouriteAppModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 已进入收藏应用管理页，团队下已有可添加的应用

### 管理分类标签

- **业务描述**: 管理员创建、重命名、删除分类标签，并为每个标签分配精选应用。标签用于在对话首页按分类筛选展示精选应用。
- **触发入口**: 收藏应用管理页 → 顶部工具栏「分类管理」按钮
- **路由路径**: `/chat?pane=s&tab=f`（模态弹窗内操作）
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/FavouriteAppSetting/TagManageModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 已进入收藏应用管理页

### 拖拽排序

- **业务描述**: 管理员通过拖拽操作调整精选应用在列表中的显示顺序，拖拽结束后自动保存新顺序。
- **触发入口**: 收藏应用管理页 → 列表中任意行的拖拽手柄图标
- **路由路径**: `/chat?pane=s&tab=f`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/FavouriteAppSetting/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 列表中有至少一个应用

### 删除收藏应用

- **业务描述**: 管理员从精选列表中移除某个应用，操作前弹出确认提示。
- **触发入口**: 收藏应用管理页 → 列表中目标行右侧删除按钮
- **路由路径**: `/chat?pane=s&tab=f`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/FavouriteAppSetting/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 列表中有至少一个应用

## 2. 场景关系

- **查看收藏应用列表** ← **添加收藏应用**: 添加完成后自动刷新列表
- **查看收藏应用列表** ← **管理分类标签**: 标签变更后刷新列表数据
- **查看收藏应用列表** ← **拖拽排序**: 拖拽结束时即更新列表顺序并保存
- **查看收藏应用列表** ← **删除收藏应用**: 删除后本地立即移除，同时刷新列表

各场景以列表页为中心，其他操作均为列表页上触发的模态弹窗交互，无严格的先后依赖关系。
