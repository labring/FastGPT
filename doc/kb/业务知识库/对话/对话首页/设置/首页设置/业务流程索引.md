# 首页设置 — 业务流程索引

## 业务场景列表

### 场景：配置首页展示开关

- **业务描述**: 管理员控制对话首页的启用与关闭。关闭首页后，用户进入对话时将跳过首页直接进入对话界面。
- **触发入口**: 对话首页 → 设置按钮 → HOME Tab →「启用首页」开关
- **路由路径**: `/chat?pane=s&tab=h`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/HomepageSetting/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 用户已登录且具有团队管理权限（`team.permission.hasManagePer`）；已加载 chatSettings 数据

### 场景：管理快捷应用列表

- **业务描述**: 管理员从团队应用中挑选最多 4 个应用作为对话首页的快捷入口，供终端用户快速访问。支持通过拖拽调整快捷应用的排列顺序。
- **触发入口**: 对话首页 → 设置按钮 → HOME Tab →「快捷应用」区域的设置图标按钮
- **路由路径**: `/chat?pane=s&tab=h`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/HomepageSetting/index.tsx` + `AddQuickAppModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 用户已登录且具有团队管理权限；已有团队应用可供选择

### 场景：管理可用工具列表

- **业务描述**: 管理员选择对话首页上对用户可见的插件工具。选中的工具将显示在对话首页供终端用户使用，支持添加和移除操作。
- **触发入口**: 对话首页 → 设置按钮 → HOME Tab →「可用工具」区域的添加按钮
- **路由路径**: `/chat?pane=s&tab=h`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/HomepageSetting/index.tsx` + `ToolSelectModal.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 用户已登录且具有团队管理权限；系统中有可用的工具模板

### 场景：修改欢迎标语

- **业务描述**: 管理员自定义对话首页的欢迎标语（Slogan），该标语将显示在对话首页的显著位置。
- **触发入口**: 对话首页 → 设置按钮 → HOME Tab →「标语」输入框
- **路由路径**: `/chat?pane=s&tab=h`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/HomepageSetting/index.tsx`
- **涉及角色**: 团队管理员
- **前置条件**: 用户已登录且具有团队管理权限

## 场景关系

- **配置首页展示开关** → **管理快捷应用列表**: 快捷应用仅在首页启用时有意义，关闭首页后快捷应用入口不会展示给终端用户
- **配置首页展示开关** → **管理可用工具列表**: 可用工具仅在首页启用时对终端用户可见
- 其他场景之间独立执行，无严格先后关系
