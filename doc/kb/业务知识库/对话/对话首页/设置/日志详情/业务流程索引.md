# 日志详情 — 业务流程索引

## 业务场景列表

### 查看日志列表

- **业务描述**: 进入日志详情页面后，自动加载指定时间范围内的对话日志列表，以分页表格形式展示
- **触发入口**: 对话首页 → 设置面板 → 点击「日志」Tab
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/LogDetails.tsx`
- **涉及角色**: 管理员
- **前置条件**: 用户具有团队管理权限（`hasManagePer`）且为商业版（`isPlus`）

### 多维度筛选日志

- **业务描述**: 通过日期范围选择器、对话来源多选、用户筛选、关键词搜索等条件组合筛选日志数据
- **触发入口**: 日志详情页面上方的筛选栏
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/chat/ChatSetting/LogDetails.tsx`（LogsFilterBar）
- **涉及角色**: 管理员
- **前置条件**: 已进入日志详情页面

### 配置日志列显示

- **业务描述**: 通过拖拽调整日志表格中列的显示/隐藏和排列顺序，支持将列配置同步到团队
- **触发入口**: 日志详情页面筛选栏右侧的列配置按钮
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/app/detail/Logs/LogKeysConfigPopover.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入日志详情页面，有团队日志 Key 配置

### 查看单条对话详情

- **业务描述**: 点击日志记录，在弹窗中查看该对话的完整内容（用户消息与 AI 回复），支持查看和提交纠错记录
- **触发入口**: 日志列表中点击某条记录
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/app/detail/Logs/LogTable.tsx`（DetailLogsModal）
- **涉及角色**: 管理员
- **前置条件**: 日志列表已加载，有可点击的记录

### 导出日志数据

- **业务描述**: 将当前筛选条件下的日志数据导出为 CSV 文件下载
- **触发入口**: 日志列表右上角的导出按钮
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/app/detail/Logs/LogTable.tsx`
- **涉及角色**: 管理员
- **前置条件**: 日志列表已加载

### 批量删除对话

- **业务描述**: 勾选多条日志记录后批量删除对应的对话历史
- **触发入口**: 日志列表中的复选框 → 批量删除按钮
- **路由路径**: `/chat?pane=s&tab=l`
- **页面组件位置**: `projects/app/src/pageComponents/app/detail/Logs/LogTable.tsx`
- **涉及角色**: 管理员
- **前置条件**: 日志列表已加载，有可勾选的记录

## 场景关系

- **查看日志列表** → **多维度筛选日志**：筛选条件变更后自动触发日志列表刷新
- **查看日志列表** → **查看单条对话详情**：列表是详情弹窗的入口
- **查看日志列表** → **批量删除对话**：需先在列表中勾选记录
- **配置日志列显示** → **查看日志列表**：列配置变更直接影响列表的列展示
- 导出日志数据、查看单条对话详情、批量删除对话 为独立操作，无先后依赖关系
