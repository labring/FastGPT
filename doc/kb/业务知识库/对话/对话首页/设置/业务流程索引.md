# 设置 — 业务流程索引

## 业务场景列表

### 进入设置页面

- **业务描述**: 管理员通过侧边栏进入对话首页的设置管理页面，系统自动校验商业版许可和管理权限
- **触发入口**: 对话首页侧边栏 → 设置图标
- **路由路径**: `/chat?pane=s`
- **页面组件位置**: `pageComponents/chat/ChatSetting/index.tsx`
- **涉及角色**: 管理员
- **前置条件**: 商业版许可已激活（`feConfigs.isPlus` 为 `true`），用户拥有团队管理权限（`hasManagePer` 为 `true`）

### Tab 切换

- **业务描述**: 管理员在首页配置、首页数据、首页日志、精选应用四个 Tab 之间切换
- **触发入口**: 设置页面顶部 Tab 栏
- **路由路径**: `/chat?pane=s&tab={h|d|l|f}`
- **页面组件位置**: `pageComponents/chat/ChatSetting/SettingTabs.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入设置页面（`chatSettings` 已加载）

## 场景关系

- **进入设置页面** → **Tab 切换**: 进入设置后默认展示「首页配置」Tab，管理员可自由切换至其他 Tab
- 各 Tab 内的操作场景独立执行，无严格先后关系
