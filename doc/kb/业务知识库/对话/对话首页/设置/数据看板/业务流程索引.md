# 数据看板 — 业务流程索引

## 业务场景列表

### 查看数据看板

- **业务描述**: 管理员进入首页数据看板页面，查看首页 AI 应用的综合运营数据，包含用户增长、对话活跃度和用户反馈三大维度的图表
- **触发入口**: 设置页面 Tab 栏 →「首页数据」
- **路由路径**: `/chat?pane=s&tab=d`
- **页面组件位置**: `pageComponents/chat/ChatSetting/DataDashboard.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入设置页面，chatSettings 已加载完成（含 appId）

### 筛选数据范围

- **业务描述**: 管理员通过筛选栏调整数据看板的展示范围，包括选择日期范围、勾选渠道来源
- **触发入口**: 数据看板顶部筛选栏
- **路由路径**: 同上（无路由变化）
- **页面组件位置**: `pageComponents/chat/ChatSetting/DataDashboard.tsx`（LogsFilterBar 子组件）
- **涉及角色**: 管理员
- **前置条件**: 数据看板已加载

### 切换统计维度

- **业务描述**: 管理员切换各数据面板的时间粒度（日/周/月/季度），以不同时间维度审视数据趋势
- **触发入口**: 各 Accordion 面板右侧的时间粒度下拉选择器
- **路由路径**: 同上（无路由变化）
- **页面组件位置**: `pageComponents/app/detail/Logs/LogChart.tsx`
- **涉及角色**: 管理员
- **前置条件**: 图表数据已加载

### 展开/折叠数据面板

- **业务描述**: 管理员展开或折叠用户数据面板或对话数据面板，聚焦关注的数据维度
- **触发入口**: 各面板标题栏（Accordion 按钮）
- **路由路径**: 同上（无路由变化）
- **页面组件位置**: `pageComponents/app/detail/Logs/LogChart.tsx`
- **涉及角色**: 管理员
- **前置条件**: 数据看板已加载

## 场景关系

```
进入数据看板
  ├── 查看数据看板（默认加载全部图表）
  ├── 筛选数据范围（调整日期和渠道 → 图表数据重新请求）
  └── 切换统计维度（选择时间粒度 → 图表数据重新请求）
        └── 展开/折叠数据面板（切换面板显示状态）
```

所有操作场景均在数据看板页面内完成，筛选和切换均触发图表数据重新请求。Accordion 默认全部展开（`defaultIndex={[0,1,2]}`）。
