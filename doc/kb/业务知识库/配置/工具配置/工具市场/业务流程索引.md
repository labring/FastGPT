# 工具市场 — 业务流程索引

## 业务场景列表

### 浏览与搜索工具

- **业务描述**: 管理员进入工具市场，浏览可用工具列表。可通过顶部搜索框按工具名称搜索，通过标签筛选器按工具分类过滤。
- **触发入口**: 左侧导航 → 配置 → 工具管理 → 点击「添加资源」→ 选择「打开市场」
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 系统管理员已登录

### 安装工具

- **业务描述**: 管理员在市场中找到需要的工具，点击工具卡片上的安装按钮，从 Marketplace 服务下载并安装工具到本地系统工具库。
- **触发入口**: 工具市场 → 工具卡片 hover 时显示的「安装」按钮
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 工具未安装；Marketplace 服务可访问

### 卸载工具

- **业务描述**: 管理员卸载已安装的工具，从本地系统工具库中移除。
- **触发入口**: 工具市场 → 已安装工具卡片 hover 时显示的「卸载」按钮
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 工具已安装

### 更新工具

- **业务描述**: 管理员将已安装的工具更新至 Marketplace 上的最新版本。
- **触发入口**: 工具市场 → 有更新的工具卡片 hover 时显示的「更新」按钮
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 工具已安装且 Marketplace 上有更新版本

### 批量更新工具

- **业务描述**: 管理员一次性更新多个有可用更新的已安装工具。
- **触发入口**: 工具市场 → 右上角「可更新 (N)」按钮 → 打开批量更新抽屉
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 存在至少一个可更新的已安装工具

### 查看工具详情

- **业务描述**: 管理员点击工具卡片查看工具的详细信息，包括标签、描述、作者、参数说明、使用指南等。
- **触发入口**: 工具市场 → 点击工具卡片
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 无

### 筛选未安装工具

- **业务描述**: 管理员通过下拉菜单切换显示模式，仅显示未安装的工具，便于发现新工具。
- **触发入口**: 工具市场 → 标签筛选栏右侧下拉菜单 → 选择「未安装」
- **路由路径**: `/config/tool/marketplace`
- **页面组件位置**: `projects/app/src/pages/config/tool/marketplace.tsx`
- **涉及角色**: admin
- **前置条件**: 无

## 场景关系

- **浏览与搜索工具** → **安装工具 / 查看工具详情**: 浏览是安装和查看详情的前置动作
- **浏览与搜索工具** → **批量更新工具**: 浏览时发现有可用更新时触发批量更新
- **查看工具详情** → **安装工具 / 卸载工具 / 更新工具**: 详情抽屉中可直接执行安装、卸载或更新操作
- 其他场景之间无严格先后依赖，可独立执行
