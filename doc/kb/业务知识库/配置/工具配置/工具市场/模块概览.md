# 工具市场 — 模块概览

## 1. 模块业务概述

工具市场是 FastGPT 系统工具的在线应用商店，为系统管理员提供浏览、搜索、安装和管理第三方工具的入口。管理员可以在市场中按标签分类或关键词搜索工具，一键安装到本地系统工具库中，也可以卸载已安装的工具或批量更新至最新版本。

本模块是「工具配置」的子能力，作为工具管理的补充渠道——除了手动创建和导入工具外，管理员可通过市场获取社区或官方发布的工具资源。市场数据来自远端 Marketplace 服务，页面通过代理 API 与 Marketplace 后端通信。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| admin（系统管理员） | 工具市场主页 | 浏览工具列表、按标签/关键词筛选、安装/卸载工具、更新单个工具、批量更新、查看工具详情（参数/使用指南） | 唯一可访问此模块的角色，拥有全部操作权限 |

**权限模型**: 页面路由 `/config/tool/marketplace` 位于配置管理区，需系统管理员身份才可访问。工具删除操作的后端接口通过 `authSystemAdmin` 进行权限校验。

## 3. 跨模块依赖

| 依赖模块 | 使用的具体能力 | 能力类型 |
|---------|---------------|---------|

## 4. 对外共享

本模块为叶子节点，不对外共享组件或 API。外部模块通过路由跳转进入本页面（如工具管理主页的「打开市场」菜单项）。
