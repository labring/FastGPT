# 工具配置 — 模块概览

## 1. 模块业务概述

工具配置是 FastGPT 管理员对系统插件/工具进行全生命周期管理的功能模块。管理员可以在此查看、配置、导入和管理系统工具，并通过工具市场发现和安装新的插件，从而扩展平台能力。

本模块服务于**管理员**角色，是整个平台「插件体系」的管控入口。包含两个子能力：

- **工具配置首页**：系统工具的集中管理台，支持工具上线/下线、默认安装、标签分类、系统密钥配置、拖拽排序和导入导出
- **工具市场**：连接外部 Marketplace，支持浏览、搜索、安装、更新和卸载社区/官方提供的工具插件

两个子能力协同工作：市场负责发现和获取，首页负责配置和管控，形成「发现→安装→配置→管控」的完整链条。

## 2. 角色差异

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| admin | 工具配置首页、工具市场 | 配置工具状态/默认安装/标签/密钥、导入插件、安装/卸载/更新市场插件、拖拽排序 | 全部管理权限 |
| 普通用户 | 无 | 无 | 本模块仅对管理员开放，普通用户不可见 |

**权限模型**：页面级权限控制，通过路由守卫校验管理员角色，非管理员路由不可达。
