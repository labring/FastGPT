# 工具配置首页 — 业务流程索引

## 业务场景列表

### 查看工具列表

- **业务描述**: 管理员进入工具配置页，查看当前系统中所有已安装的系统工具（含官方插件和自定义工作流工具），了解每个工具的名称、标签、简介、状态、默认安装策略、Token 费用和系统密钥配置情况。
- **触发入口**: 左侧导航菜单 → 配置 → 工具配置（或账户容器侧边栏导航）
- **路由路径**: `/config/tool`
- **页面组件位置**: `projects/app/src/pages/config/tool/index.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已登录且具有管理员权限

### 管理工具标签

- **业务描述**: 管理员打开标签管理弹窗，对系统中的工具标签进行增删改查和排序操作，用于对工具进行分类管理。
- **触发入口**: 工具配置首页 → "标签管理"按钮
- **路由路径**: `/config/tool`（弹窗形式，无路由变化）
- **页面组件位置**: `projects/app/src/pageComponents/config/TagManageModal.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入工具配置首页

### 添加工具资源

- **业务描述**: 管理员通过"添加资源"下拉菜单，选择进入工具市场发现新工具、导入插件包、或创建应用类型的工作流工具。
- **触发入口**: 工具配置首页 → "添加资源"按钮（hover 展开菜单）
- **路由路径**: `/config/tool`（弹窗形式或跳转到 `/config/tool/marketplace`）
- **页面组件位置**: `projects/app/src/pages/config/tool/index.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入工具配置首页

### 配置系统工具

- **业务描述**: 管理员点击工具行，打开系统工具配置弹窗，修改工具的状态、默认安装策略、标签、系统密钥、用户标签推广/隐藏等配置。对于文件夹型工具，可查看和配置子工具的系统密钥价格。
- **触发入口**: 工具列表 → 点击任意工具行
- **路由路径**: `/config/tool`（弹窗形式，无路由变化）
- **页面组件位置**: `projects/app/src/pageComponents/config/tool/SystemToolConfigModal.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已加载工具列表，目标工具为系统内置类型

### 配置工作流工具

- **业务描述**: 管理员点击工作流/应用类型工具行或新建应用工具，打开工作流工具配置弹窗，编辑工具的名称、头像、简介、关联应用、标签、作者、状态、价格和用户指南等信息。
- **触发入口**: 工具列表 → 点击工作流类型工具行；或"添加资源"→"选择应用"
- **路由路径**: `/config/tool`（弹窗形式，无路由变化）
- **页面组件位置**: `projects/app/src/pageComponents/config/tool/WorkflowToolConfigModal.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已加载工具列表

### 调整工具排序

- **业务描述**: 管理员通过拖拽工具行调整工具在用户侧的展示顺序，拖拽完成后自动保存新顺序。
- **触发入口**: 工具列表 → 拖拽任意工具行的拖拽手柄
- **路由路径**: `/config/tool`
- **页面组件位置**: `projects/app/src/pages/config/tool/index.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已有至少 2 个工具

### 快捷切换工具属性

- **业务描述**: 管理员在工具行上直接切换默认安装状态或 Token 费用开关，无需打开配置弹窗即可快速修改。
- **触发入口**: 工具行中的"默认安装"复选框或"Token 费用"开关
- **路由路径**: `/config/tool`
- **页面组件位置**: `projects/app/src/pageComponents/config/tool/ToolRow.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已加载工具列表；Token 费用开关仅对有关联应用的工具可用

### 导入插件包

- **业务描述**: 管理员打开导入插件弹窗，上传或通过 URL 安装外部插件包。
- **触发入口**: 工具配置首页 → "添加资源"→"导入资源"
- **路由路径**: `/config/tool`（弹窗形式，无路由变化）
- **页面组件位置**: `projects/app/src/pageComponents/config/ImportPluginModal.tsx`
- **涉及角色**: 管理员
- **前置条件**: 已进入工具配置首页

## 场景关系

- **查看工具列表** → **配置系统工具 / 配置工作流工具**: 必须先进入列表页，才能选择具体工具进行配置
- **查看工具列表** ↔ **调整工具排序**: 在列表页上直接执行，操作后列表即时更新
- **添加工具资源** → **查看工具列表**: 添加完成后刷新列表
- **快捷切换工具属性** ↔ **查看工具列表**: 在列表行上直接切换，无需进入配置弹窗
- **管理工具标签** ↔ **配置系统工具 / 配置工作流工具**: 标签在标签管理弹窗中统一管理，在工具配置弹窗中引用
- 各弹窗场景（管理标签、导入插件、系统工具配置、工作流工具配置）互斥，同时只能打开一个
