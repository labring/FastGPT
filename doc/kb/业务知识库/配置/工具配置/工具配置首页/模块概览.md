# 工具配置首页 — 模块概览

## 模块业务概述

工具配置首页是 FastGPT 管理后台中系统工具的统一管理入口。管理员在此页面查看、配置、排序所有已安装的系统工具（插件），包括官方工具和自定义工作流工具。模块支持工具的状态管理（正常/即将下线/已下线）、默认安装策略、系统密钥配置、Token 费用开关以及标签分类管理。通过拖拽排序功能，管理员可以调整工具在用户侧的展示顺序。

本模块是"配置 → 工具配置"路径下的叶子节点，无子能力。与之并列的还有[工具市场](../工具市场/模块概览.md)，用户可从市场发现并安装新工具。

## 角色差异

本模块仅对具有管理权限的用户开放。页面路由 `/config/tool` 通过 Next.js 服务端渲染时的 `serviceSideProps` 校验用户登录态，页面内部通过 `AccountContainer` 组件统一管理用户认证和导航。

| 角色 | 可见页面 | 可执行操作 | 差异说明 |
|------|---------|-----------|---------|
| 管理员 | 工具配置首页 | 查看工具列表、修改工具状态、配置系统密钥、调整默认安装、设置 Token 费用、拖拽排序、删除工具、管理标签、导入插件、创建工作流工具 | 完整管理权限 |
| 普通用户 | 无 | 无 | 路由受服务端权限校验保护，不可访问 |

**权限模型**: 基于 Next.js 服务端渲染 `serviceSideProps` 的登录态校验，结合 `AccountContainer` 中的 `useUserStore` 进行用户身份管理。
