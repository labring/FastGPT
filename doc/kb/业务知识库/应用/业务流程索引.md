# 应用 — 业务流程索引

## 业务场景汇总

| 编号 | 场景 | 触发入口 | 路由路径 | 角色 |
|------|------|---------|---------|------|
| S01 | 查看应用详情与配置 | 工作台应用列表 → 点击应用名称 | /app/detail | admin, collaborator, viewer |
| S02 | 编辑应用配置（简易应用/Chat Agent/工作流等） | 应用详情页 → 对应编辑器 Tab | /app/detail | admin, collaborator（需写权限） |
| S03 | 查看应用日志与对话记录 | 应用详情页 → 日志/看板 Tab | /app/detail | admin, collaborator, viewer |
| S04 | 发布应用版本 | 应用详情页 → 发布操作 | /app/detail | admin, collaborator（需写权限） |
| S05 | 删除应用 | 应用详情页 → 更多操作 → 删除 | /app/detail | admin（仅 Owner） |

## 场景关系

- **S01（查看详情）** → **S02（编辑配置）**: 进入详情后默认可切换到编辑 Tab（有写权限时）
- **S01（查看详情）** → **S03（查看日志）**: 无写权限用户进入后自动跳转日志 Tab
- **S02（编辑配置）** → **S04（发布版本）**: 编辑完成后可选择发布新版本
- 各场景按功能独立执行，详细交互步骤见 [应用详情 — 业务流程详解](./应用详情/业务流程详解.md)
