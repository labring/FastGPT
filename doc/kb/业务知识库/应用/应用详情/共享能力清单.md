# 应用详情 — 共享能力清单

## 对外提供的能力

| 共享能力 | 消费方 | 能力类型 | 说明 |
|---------|-------|---------|------|
| AppContext（TabEnum/route2Tab/appDetail/onSaveApp/onDelApp） | SimpleEdit, AgentEdit, Workflow, Plugin, MCPTools, HTTPTools, SmartCustomerService | Context | 提供统一的应用状态管理和操作方法 |
| RouteTab 组件 | FormComponent/Header（被 SimpleEdit/AgentEdit/SmartCustomerService 复用） | 组件 | 动态 Tab 切换器，根据权限和类型计算可见 Tab |
| FormComponent/Header | SimpleEdit, AgentEdit, SmartCustomerService | 组件 | 通用编辑器 Header（面包屑+保存+Tab+版本） |
| Logs 日志组件 | Dashboard 数据看板 | 组件 | 日志图表被 Dashboard 复用展示统计数据 |
| useChatTest Hook | SimpleEdit/ChatAgent/SmartCustomerService 的 ChatTest 面板 | Hook | 提供聊天测试 SSE 流式交互能力 |
| useSnapshots Hook | SimpleEdit, AgentEdit | Hook | 编辑器快照管理（localStorage 持久化+版本恢复） |

## 内部共享矩阵

本模块内部，以下组件在多处复用：

| 组件 | 复用位置 | 说明 |

| RouteTab | FormComponent/Header → 所有编辑器 | Tab 切换逻辑集中管理 |
| Logs/LogChart | Dashboard (数据看板) | 日志图表组件被看板复用 |
| ConversationLogs/LogFilters | Logs, ConversationLogs | 日志筛选器统一组件 |
| ConversationLogs/LogList | Logs, ConversationLogs | 日志列表统一组件 |
| Debugger | Workflow, Plugin | 工作流运行时调试器 |
