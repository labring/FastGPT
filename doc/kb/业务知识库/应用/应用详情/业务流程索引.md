# 应用详情 — 业务流程索引

## 阶段 A：Tab 信息

本模块 Tab 结构通过 `TabEnum` 枚举定义，位于 `projects/app/src/pageComponents/app/detail/context.tsx`：

- **appEdit** — 应用编辑器（不同应用类型渲染不同编辑器）
- **dashboard** — 数据看板（仅 assistant/chatAgent/workflow 类型可见）
- **logs** — 对话日志
- **publish** — 发布渠道管理

Tab 通过 `RouteTab` 组件根据权限和应用类型动态计算可见性。当前激活 Tab 由 URL 查询参数 `currentTab` 控制。

顶层条件渲染基于 `appDetail.type`（AppTypeEnum）分发 7 种编辑器，属于应用类型路由而非 Tab，不按 Tab 拆分子能力。

## 业务场景列表

### 查看与编辑简单应用

- **业务描述**：对简单模式应用进行表单配置编辑，包括提示词、变量、文件选择等参数调整
- **触发入口**：工作台应用列表 → 点击简单应用卡片
- **路由路径**：`/app/detail?appId=xxx&currentTab=appEdit`
- **页面组件位置**：`projects/app/src/pageComponents/app/detail/Edit/SimpleApp/index.tsx`
- **涉及角色**：Owner, 管理员, 编辑者
- **前置条件**：用户需有写权限（hasWritePer），URL 含有效 appId

### 查看与编辑对话 Agent

- **业务描述**：对 Agent 模式应用进行配置编辑，包括技能选择、工具调用、知识库关联等
- **触发入口**：工作台应用列表 → 点击对话 Agent 应用卡片
- **路由路径**：`/app/detail?appId=xxx&currentTab=appEdit`
- **页面组件位置**：`projects/app/src/pageComponents/app/detail/Edit/ChatAgent/index.tsx`
- **涉及角色**：Owner, 管理员, 编辑者
- **前置条件**：用户需有写权限（hasWritePer）

## 场景关系

- **进入应用详情** → **编辑应用**：页面加载时默认激活 appEdit Tab（有写权限且非 assistant/chatAgent/workflow 类型时）
- **进入应用详情** → **数据看板**：assistant/chatAgent/workflow 类型且有 hasReadChatLogPer 时默认激活 dashboard Tab
- **编辑应用** → **聊天测试**：编辑器内嵌，可同步调试
- **编辑应用** → **发布渠道**：编辑完成后可切换到 publish Tab 配置发布
- **编辑应用** → **版本历史**：保存后可查看历史版本
- **无写权限** → **对话日志**：自动跳转到 logs Tab

各场景独立执行，无严格先后依赖。编辑、看板、日志、发布四个 Tab 之间可自由切换。
