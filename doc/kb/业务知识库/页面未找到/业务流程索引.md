# 页面未找到 — 业务流程索引

## 业务场景列表

### 访问无效路径

- **业务描述**：用户访问了不存在的前端路由，系统不展示 404 错误页面，而是直接重定向至应用工作台首页，确保用户始终停留在可用页面。
- **触发入口**：在浏览器地址栏输入任意不存在的 FastGPT 路径，或点击失效的外部链接。
- **路由路径**：`/404`（Next.js 自动匹配所有未注册路由）
- **页面组件位置**：`projects/app/src/pages/404.tsx`
- **涉及角色**：无限制（所有访问者）
- **前置条件**：无（不需要登录状态或特定权限）

各场景独立执行，无严格先后关系。本模块仅包含一个业务场景。
