# 首页 — 业务流程索引

## 页面访问重定向

- **业务描述**：用户访问平台根路径时，系统自动将其重定向至应用工作台页面，确保用户进入平台核心功能区域。
- **触发入口**：用户在浏览器输入平台根地址或点击平台 Logo/首页链接
- **路由路径**：`/`
- **页面组件位置**：`projects/app/src/pages/index.tsx`
- **涉及角色**：所有用户
- **前置条件**：无

## 场景关系

各场景独立执行，无严格先后关系。首页仅包含一个场景（路由重定向），不涉及多场景流转。
