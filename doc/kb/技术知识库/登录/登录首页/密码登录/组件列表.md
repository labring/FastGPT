## 关键依赖说明

- **FormLayout**（共享布局组件）：为所有登录方式提供统一的外观（Logo、标题、OAuth 入口列表、"or"分隔线）。使用 `dynamic(() => Promise.resolve(FormLayout), { ssr: false })` 禁用 SSR。
- **PolicyTip**（协议提示组件）：在登录表单中展示服务协议和隐私政策的链接，支持居中或左对齐两种布局。当 `feConfigs.showProtocol === false` 时不渲染。
- **Chakra UI 组件**：表单使用 `FormControl`、`Input`、`Button`、`Flex`、`Box` 等 Chakra UI 组件构建，响应式适配移动端和桌面端。
