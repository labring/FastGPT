# 注册 — 组件列表

## 说明

本模块为登录页面的 Tab 子组件，仅包含一个表单组件 `RegisterForm`。该组件由登录容器 `LoginContainer` 通过 `next/dynamic` 动态加载，在页面类型为 `LoginPageTypeEnum.register` 时渲染。

RegisterForm 消费以下外部依赖：
- `useSendCode` Hook（`@/web/support/user/hooks/useSendCode`）— 验证码发送与倒计时管理
- `useSystemStore`（`@/web/common/system/useSystemStore`）— 系统配置读取
- `useToast`（`@fastgpt/web/hooks/useToast`）— Toast 通知
- `useRequest`（`@fastgpt/web/hooks/useRequest`）— 异步请求状态管理
- `postRegister`（`@/web/support/user/api`）— 注册 API
- `checkPasswordRule`（`@fastgpt/global/common/string/password`）— 密码规则校验
- 营销工具函数（`@/web/support/marketing/utils`）— 营销参数采集
