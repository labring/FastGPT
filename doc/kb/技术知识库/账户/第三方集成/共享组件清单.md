## LafAccountModal

**文件**: `projects/app/src/components/support/laf/LafAccountModal.tsx`
**用途**: Laf 云函数平台账号配置弹窗，提供 PAT 验证、应用列表拉取和应用选择保存功能
**被引用方**: 工作台 → NodeLaf（工作流 Laf 调用节点）

### Props

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `defaultData` | `LafAccountType` | 否 | `{ token: '', appid: '', pat: '' }` | 已有的 Laf 账号配置，用于回填表单 |
| `onClose` | `() => void` | 是 | — | 关闭弹窗的回调函数 |

### 使用示例

```tsx
import LafAccountModal from '@/components/support/laf/LafAccountModal';

<LafAccountModal
  defaultData={userInfo?.team?.lafAccount}
  onClose={onCloseLaf}
/>
```
