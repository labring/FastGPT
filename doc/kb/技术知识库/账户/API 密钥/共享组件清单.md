# API 密钥 — 共享组件清单

> 本文档列出本模块中可被其他模块复用的组件及其接口约定。

## ApiKeyTable

- **组件文件**：`projects/app/src/components/support/apikey/Table.tsx`（默认导出）
- **用途**：API 密钥的完整增删改查管理表格，包含列表展示、创建/编辑模态框、删除确认、调用示例展示功能
- **包装方式**：`React.memo` 包裹
- **消费模块**：应用发布 — Publish/API（projects/app/src/pageComponents/app/detail/Publish/API/index.tsx）

### Props

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| tips | string | 是 | — | 表格标题下方的提示文本 |
| appId | string | 否 | undefined | 应用 ID，传入时仅管理该应用的密钥，列表列出时过滤；不传时管理账户下全部密钥 |

### 行为说明

| 场景 | 行为 |
|------|------|
| 页面加载 | 自动调用 GET /support/openapi/list 获取密钥列表，显示 MyBox 加载遮罩 |
| 创建密钥 | 弹出 EditKeyModal，填写名称（必填），Plus 版本还可填写用量上限和过期时间 |
| 编辑密钥 | 弹出 EditKeyModal（预填数据），可修改名称和限制参数 |
| 删除密钥 | 弹出确认弹窗，确认后调用 DELETE /support/openapi/delete |
| 复制基础地址 | 点击页面顶部基础地址区域，复制到剪贴板 |
| 调用示例 | appId 存在时，获取应用详情并生成 curl 调用示例弹窗（含变量类型说明） |
| Plus 版本差异 | 通过 feConfigs.isPlus 控制：过期时间列、用量上限输入、过期时间选择器、最大用量上限字段 |

### 内部依赖

ApiKeyTable 内部包含两个私有子组件：
- **EditKeyModal**：创建/编辑 API Key 的模态框表单
- **CallExampleModal**：展示 API 调用示例（curl 命令和变量说明）

两个子组件均为 ApiKeyTable 内部定义，不独立导出，无需外部关心其 Props 接口。

### 消费方注意事项

- 当传入 `appId` 时，密钥范围限定为该应用，同时「调用示例」功能可用
- 当不传 `appId` 时（账户全局场景），「调用示例」功能不可用
- `tips` 文案通过 i18n 翻译传入，不同场景可使用不同提示文本

## 外部共享 Hooks 使用说明

> 以下 Hooks 定义在 `packages/web/hooks/` 中，属于全局共享，被 ApiKeyTable 消费。此处仅列出与本模块相关的关键接口。

### useRequest

- **定义位置**：`@fastgpt/web/hooks/useRequest`
- **在 ApiKeyTable 中的用途**：管理 API 密钥 CRUD 操作的异步状态（loading、error、data），支持手动触发和依赖刷新

### useCopyData

- **定义位置**：`@fastgpt/web/hooks/useCopyData`
- **在 ApiKeyTable 中的用途**：复制 API 基础地址和新建密钥值到剪贴板

### useConfirm

- **定义位置**：`@fastgpt/web/hooks/useConfirm`
- **在 ApiKeyTable 中的用途**：删除密钥前的二次确认弹窗（type: 'delete'，红色警告样式）
