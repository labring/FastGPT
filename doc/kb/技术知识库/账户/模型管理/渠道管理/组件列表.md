## 说明

渠道管理模块包含 3 个自有组件和 1 个内联组件（MultipleSelect）：

- **ChannelTable** 是主列表组件，承载所有数据展示和操作入口
- **EditChannelModal** 负责渠道的创建和编辑表单，内部集成供应商选择、模型多选、模型映射编辑器
- **ModelTest** 负责模型的连通性测试，支持批量并发测试和单项重测
- **MultipleSelect** 是 EditChannelModal 内部的内联多选组件，用于模型选择

所有组件均使用 React 函数式组件 + Hooks 模式，通过 Chakra UI 构建 UI，通过 `@fastgpt/web` 的通用组件库提供业务组件。
