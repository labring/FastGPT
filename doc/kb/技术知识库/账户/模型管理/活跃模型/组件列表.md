# 活跃模型 — 组件列表

## ModelTable/

主组件目录，位于 `projects/app/src/components/core/ai/ModelTable/`。

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 | Props 摘要 |
|---------|------|-----------|---------|-----------|
| `index.tsx` | 活跃模型主组件，管理双 Tab（基座/自定义）切换，组合筛选、表格、批量操作和弹窗/抽屉 | 引用自 @fastgpt/web（FillRowTabs、useTableMultipleSelect）；引用自系统 Store（useSystemStore、useUserStore） | 否（内部导出 ModelPriceModal 包装组件） | `permissionConfig: boolean` |
| `ModelFilterBar.tsx` | 模型筛选栏：供应商下拉、模型类型下拉、关键词搜索框 | 引用自 @fastgpt/web（MySelect） | 否 | `filterState: FilterState`, `providerList: ProviderOption[]` |
| `ModelListTable.tsx` | 模型列表表格（含表头排序、复选框、操作列、引用提示弹窗） | 引用自系统 API（getDatasetsWithChildren）；引用自 ModelReferenceModal | 否 | `modelList: ModelRow[]`, `tabType: ModelTabType`, `permissionConfig: boolean` |
| `BatchPermissionActionBar.tsx` | 批量权限操作栏，勾选模型后底部浮出，提供权限配置入口 | 引用自 @fastgpt/global（ReadRoleVal）；引用自 LazyCollaboratorProvider；引用自系统 API（updateModelCollaborators） | 否 | `FloatingActionBar`, `selectedItems: ModelRow[]`, `permission: TeamPermission` |
| `DefaultModelModal.tsx` | 默认模型设置弹窗，root 用户为各类型指定默认模型 | 引用自 @fastgpt/web（MyModal）；引用自 useSystemStore | 否 | `onClose: () => void`, `onSuccess?: () => void` |
| `BaseModelTrainModal.tsx` | 模型训练弹窗，选择基座模型和训练数据集发起训练 | 引用自 @fastgpt/web（MyModal）；引用自 useSystemStore；引用自训练 API（createEmbeddingTrainTask、createRerankTrainTask） | 否 | `onClose: () => void`, `defaultBaseModel?`, `onSuccess?: () => void` |
| `TrainDetailDrawer.tsx` | 训练详情抽屉，展示模型的训练任务列表和历史 | 引用自训练 API（getEmbeddingTrainTaskList、getRerankTrainTaskList） | 否 | `onClose: () => void`, `modelName: string`, `modelId: string`, `baseModelType`, `tabType: ModelTabType` |
| `ModelReferenceModal.tsx` | 模型引用提示弹窗，当权限更新被阻止时列出引用该模型的资源 | 无外部跨模块依赖 | 是 → 被 AddModelBox、ModelConfigTable 引用 | `isOpen: boolean`, `references: ModelReference[]`, `onClose: () => void` |
