# 组件列表

## 本模块组件

### Promotion

**路径**：`projects/app/src/pages/account/promotion.tsx`

**类型**：页面组件（Next.js Page）

**功能说明**：推广数据展示页面，包含推广概览统计卡片和推广记录分页表格。

**引用方式**：通过 Next.js 文件路由 `/account/promotion` 自动映射。

**内部结构**：

- **统计卡片区**：4 列网格布局（移动端 2 列 → 桌面端 4 列），展示累计邀请人数、收益金额、返现比例、邀请链接
- **记录表格区**：Chakra UI Table 组件，三列（时间 / 类型 / 金额），集成空状态提示和分页控制

**Props**：无外部 Props（页面组件）

**关键依赖**：

| 依赖 | 来源 | 用途 |
|------|------|------|
| `AccountContainer` | `@/pageComponents/account/AccountContainer` | 账户模块通用容器，提供 Tab 导航 |
| `useQuery` | `@tanstack/react-query` | 请求推广概览数据 |
| `usePagination` | `@fastgpt/web/hooks/usePagination` | 分页加载推广记录 |
| `useUserStore` | `@/web/support/user/useUserStore` | 获取用户信息（`userInfo._id`、`promotionRate`） |
| `useCopyData` | `@fastgpt/web/hooks/useCopyData` | 复制邀请链接到剪贴板 |
| `useLoading` | `@fastgpt/web/hooks/useLoading` | 加载状态管理 |
| `QuestionTip` | `@fastgpt/web/components/common/MyTooltip/QuestionTip` | 返现比例、邀请链接说明气泡 |
| `EmptyTip` | `@fastgpt/web/components/common/EmptyTip` | 空记录提示 |

**服务端渲染**：`getServerSideProps` 预加载 `account` 和 `account_promotion` 两个 i18n 命名空间。

### AccountContainer

**路径**：`projects/app/src/pageComponents/account/AccountContainer.tsx`

**类型**：布局容器组件

**功能说明**：账户模块的通用页面容器，为所有账户子页面（包括推广）提供统一的导航和布局框架。

**与推广模块的关系**：推广页面作为 `AccountContainer` 的 `children` 渲染。容器负责：
- 桌面端侧边栏导航（通过 `DashboardNavbar`）
- 移动端顶部 Tab 导航（通过 `LightRowTabs`）
- 统一的页面背景和布局结构

**Props**：

| Prop | 类型 | 说明 |
|------|------|------|
| `children` | `React.ReactNode` | 子页面内容（推广模块的 Promotion 组件） |
| `isLoading` | `boolean?` | 页面加载状态 |
| `header` | `React.ReactNode?` | 自定义页面头部 |
| `wrapperContainerProps` | `BoxProps?` | PageContainer 外层 Props |
| `containerInsertProps` | `BoxProps?` | PageContainer 内层 Props |

**Tab 导航逻辑**：根据当前路由路径名判断激活 Tab（如 `/account/promotion` → `TabEnum.promotion`）。推广 Tab 的可见性由 `feConfigs.show_promotion` 和 `userInfo.team.permission.isOwner` 联合控制。

**引用者**：被 12 个账户子页面使用（推广、账单、API 密钥、通知、个人信息、团队管理等）。
