# 账单 — 共享组件清单

## BillDetailModal

**文件**: `pageComponents/account/bill/BillDetailModal.tsx`
**用途**: 展示单条账单的完整详情信息，包括订单号、生成时间、订单类型、状态、优惠券、支付方式、金额、是否开票、订阅周期、订阅套餐、额外配置等信息。
**被引用方**: account/info（DiscountCouponsModal）

### Props

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `billId` | `string` | 是 | — | 账单 ID，用于获取账单详情 |
| `onClose` | `() => void` | 是 | — | 关闭弹窗的回调函数 |

### 展示字段

弹窗根据账单数据动态展示以下字段（全部只读）：

| 字段 | 数据来源 | 显示条件 |
|------|---------|---------|
| 订单号 | `bill.orderId` | 始终显示 |
| 生成时间 | `bill.createTime`（格式化 YYYY/MM/DD HH:mm:ss） | 始终显示 |
| 订单类型 | `billTypeMap[bill.type].label`（i18n 翻译） | `bill.type` 存在 |
| 状态 | `billStatusMap[bill.status].label` | `bill.status` 存在 |
| 优惠券 | `bill.discountCouponName` | `bill.discountCouponName` 存在 |
| 支付方式 | `billPayWayMap[bill.metadata.payWay].label` | `bill.metadata.payWay` 存在 |
| 金额 | `formatStorePrice2Read(bill.price)` 元 | `bill.price` 存在 |
| 是否开票 | 余额支付显示"无需开票"，其他显示是/否 | `bill.price` 存在 |
| 订阅周期 | `subModeMap[bill.metadata.subMode].label` | `bill.metadata.subMode` 存在 |
| 订阅套餐 | `standardSubLevelMap[bill.metadata.standSubLevel].label` | `bill.metadata.standSubLevel` 存在 |
| 订阅月数 | `bill.metadata.month` 月 | `bill.metadata.month` 存在 |
| 额外数据集大小 | `bill.metadata.datasetSize` | `bill.metadata.datasetSize` 存在 |
| 额外 AI 点数 | `bill.metadata.extraPoints` | `bill.metadata.extraPoints` 存在 |
| 自定义配置详情 | 动态解析 `bill.couponDetail.subscriptions` 中 custom 订阅的 customConfig | 订阅级别为 custom 且有 customConfig |

### 使用示例

```tsx
import BillDetailModal from '@/pageComponents/account/bill/BillDetailModal';

// 在父组件中
const [billDetailId, setBillDetailId] = useState<string>();

// 触发打开
<Button onClick={() => setBillDetailId(bill._id)}>详情</Button>

// 渲染弹窗
{billDetailId && (
  <BillDetailModal 
    billId={billDetailId} 
    onClose={() => setBillDetailId(undefined)} 
  />
)}
```
