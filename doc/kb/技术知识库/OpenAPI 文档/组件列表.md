## 说明

本模块仅包含一个页面组件 `openapi.tsx`，其核心能力由外部依赖 `@scalar/api-reference-react` 提供。页面组件本身是一个轻量级包装，负责：

- 使用 Next.js `dynamic()` 实现客户端仅加载（SSR 禁用），避免 Scalar 的 CSS side-effect 在 Node 端构建时报错
- 配置 Scalar 组件参数：隐藏深色模式切换、隐藏客户端按钮、使用默认主题、指向 `/api/openapi.json` 数据源
- 通过 `getServerSideProps` 禁用静态生成，确保每次请求都动态渲染

模块内部无自定义业务组件、无 Store、无自定义 Hooks。
