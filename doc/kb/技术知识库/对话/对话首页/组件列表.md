## Context Providers

本模块创建多层 Context Provider 嵌套结构，为各子组件提供对话上下文：

| Provider | 路径 | 用途 |
|----------|------|------|
| `ChatPageContextProvider` | `src/web/core/chat/context/chatPageContext.tsx` | 面板状态、用户信息、最近应用、对话设置 |
| `ChatContextProvider` | `src/web/core/chat/context/chatContext.tsx` | 对话历史数据管理 |
| `ChatItemContextProvider` | `src/web/core/chat/context/chatItemContext.tsx` | 对话项状态（引用、变量、插件标签） |
| `ChatRecordContextProvider` | `src/web/core/chat/context/chatRecordContext.tsx` | 对话记录数据 |
