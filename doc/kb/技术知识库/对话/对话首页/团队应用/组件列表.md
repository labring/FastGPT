## 组件详情

### ContextRender（index.tsx 默认导出）

容器组件，负责：
- 包裹 `AppListContextProvider` 提供应用列表上下文
- 内部 `MyApps` 组件实现具体 UI

**内部状态**：
- `appType`：当前选中的类型筛选（`AppTypeEnum | 'all'`）
- `searchKey`：搜索关键词（委托给 AppListContext 管理）

**响应式布局**：
- PC 端（`isPc === true`）：顶部 Chakra UI `<Tabs>` + `<SearchInput>`
- 移动端（`isPc === false`）：左侧菜单图标 + 居中搜索框 + `ChatSliderMobileDrawer`

**Tab 配置**：
- `all`：全部团队应用
- `assistant`：智能客服
- `workflow`：工作流机器人
- `workflowTool`：工作流工具

### List

展示应用卡片网格，是主要的列表渲染组件。关键行为：

- **类型过滤**：从 `AppListContext.myApps` 中按 `appType` 前端过滤（`all` 不过滤）
- **文件夹导航**：点击文件夹卡片更新 `parentId` 查询参数
- **应用导航**：点击非文件夹卡片调用 `handlePaneChange(RECENTLY_USED_APPS, app._id)`
- **时间显示**：仅 PC 端显示更新时间（移动端隐藏以节省空间）
- **空状态**：列表为空时显示"暂无应用"提示（`EmptyTip` 组件）

### AppTypeTag

应用类型标签组件，显示带颜色编码的类型标签：

| 应用类型 | 标签颜色 | 图标 |
|---------|---------|------|
| Agent / Chat Agent | 蓝色 `#0884DD` | `core/app/type/simple` |
| 工作流机器人 | 紫色 `#6F5DD7` | `core/app/type/workflow` |
| 工作流工具 | 绿色 `#007E7C` | `core/app/type/plugin` |
| HTTP 工具 | 粉色 `#E82F72` | `core/app/type/httpPlugin` |
| MCP 工具集 | 默认 | `core/app/type/mcpTools` |
| 智能客服 | 橙色 `#FF8A00` | `core/app/type/simple` |

> `folder`、`tool`、`hidden` 类型不显示标签（返回 null）。
