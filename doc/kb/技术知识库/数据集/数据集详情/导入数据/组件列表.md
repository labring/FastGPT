# 导入数据 — 组件列表

## Import/

页面级入口与上下文提供者。

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 | Props 摘要 |
|---------|------|-----------|---------|-----------|
| `Import/index.tsx` | 导入数据页面入口，根据 `importSource` 动态渲染对应的导入源组件 | 引用自数据集详情页 `pages/dataset/detail/index.tsx` | 否 | 无（通过 Context 通信） |
| `Import/Context.tsx` | 导入共享上下文，管理导入源类型、步骤状态、表单数据、文件来源列表 | 被所有 diffSource 子组件消费 | 是 → 模块内部共享 | `children: ReactNode` |

## Import/diffSource/

8 种不同导入源的页面组件。

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 | Props 摘要 |
|---------|------|-----------|---------|-----------|
| `diffSource/FileLocal.tsx` | 本地文件导入（4 步向导：选文件→参数→预览→上传） | 内部 | 否 | 无 |
| `diffSource/FileLink.tsx` | 网页链接导入（4 步向导：输入链接→参数→预览→上传） | 内部 | 否 | 无 |
| `diffSource/FileCustomText.tsx` | 自定义文本导入（4 步向导：输入文本→参数→预览→上传） | 内部 | 否 | 无 |
| `diffSource/ExternalFile.tsx` | 外部文件 URL 导入（4 步向导） | 内部 | 否 | 无 |
| `diffSource/APIDataset.tsx` | API 数据集文件导入，含多级目录浏览 | 内部 | 否 | 无 |
| `diffSource/ReTraining.tsx` | 已有集合重训练（3 步向导） | 内部 | 否 | 无 |
| `diffSource/ImageDataset.tsx` | 图片数据集导入（单页简化流程） | 内部 | 否 | 无 |
| `diffSource/ConnectDatabaseConfig.tsx` | 数据库导入路由（2 步向导） | 内部 | 否 | 无 |

## Import/commonProgress/

各导入源共用的步进组件。

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 | Props 摘要 |
|---------|------|-----------|---------|-----------|
| `commonProgress/DataProcess.tsx` | 数据处理参数配置（PDF 解析、分块策略、索引设置） | 被 7 个 diffSource 组件引用 | 是 → 模块内部共享 | 无（通过 Context 通信） |
| `commonProgress/PreviewData.tsx` | 文件分块预览（左侧文件列表 + 右侧分块内容） | 被 7 个 diffSource 组件引用 | 是 → 模块内部共享 | 无（通过 Context 通信） |
| `commonProgress/Upload.tsx` | 集合创建执行器，根据 `importSource` 路由到对应 API 逐条创建 | 被 6 个 diffSource 组件引用 | 是 → 模块内部共享 | 无（通过 Context 通信） |

## Import/components/

可复用的子组件和工具组件。

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 | Props 摘要 |
|---------|------|-----------|---------|-----------|
| `components/FileSourceSelector.tsx` | 导入源选择模态框（本地文件/网页链接/自定义文本） | 被数据集详情页 NavBar 引用 | 是 → 模块内部共享 | `onClose: () => void` |
| `components/FileSelector.tsx` | 文件拖拽上传选择器，支持文件夹递归、类型过滤 | 被 `FileLocal`、`ImageDataset` 引用 | 是 → 模块内部共享 | `fileType: string, selectFiles: Item[], onSelectFiles: Function` |
| `components/RenderFiles.tsx` | 文件上传进度表格（图标+名称+进度条+大小+删除） | 被 `FileLocal` 引用 | 否 | `files: ImportSourceItemType[], setFiles: Dispatch` |
| `components/ConnectDatabaseForm.tsx` | 数据库连接凭证表单（类型/主机/端口/库/用户/密码） | 被 `ConnectDatabaseConfig` 引用 | 否 | 无（通过 Context 通信） |
| `utils.tsx` | 工具函数（`getPreviewSourceReadType`、`databaseAddrValidator`） | 被 `PreviewData`、`ConnectDatabaseForm` 引用 | 是 → 模块内部共享 | —（导出工具函数） |
