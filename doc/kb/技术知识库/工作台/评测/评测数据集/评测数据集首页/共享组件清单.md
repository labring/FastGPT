# 评测数据集首页 — 共享组件清单

## IntelligentGeneration 智能生成弹窗

- **文件位置**: `projects/app/src/pageComponents/dashboard/evaluation/dataset/IntelligentGeneration.tsx`
- **用途**: 提供评测数据集智能生成的统一配置弹窗。用户选择知识库、配置数据量和生成模型，由 AI 自动生成评测数据集。支持两种场景模式（dataset 新建数据集、data 追加数据）
- **被以下模块引用**: 评测数据集详情页（DataListModals）、评测任务创建页（CreateModal）、对话数据集选择器（EvaluationDatasetSelector）
- **关键接口**:

### Props

| Props | 类型 | 必填 | 说明 |
|-------|------|------|------|
| isOpen | boolean | 是 | 弹窗开关 |
| onClose | () => void | 是 | 关闭回调 |
| onConfirm | (data: IntelligentGenerationForm, datasetId?: string) => void | 是 | 确认回调，返回表单数据和可选的数据集ID |
| defaultValues | Partial\<IntelligentGenerationForm\> | 否 | 默认表单值 |
| scene | 'dataset' \| 'data' | 否 | 场景模式：dataset=新建数据集（需填名称），data=追加数据（无需名称）。默认 'dataset' |
| returnDatasetId | boolean | 否 | 是否在 onConfirm 中返回数据集ID |

### IntelligentGenerationForm 表单数据

| 字段 | 类型 | 说明 |
|------|------|------|
| name | string | 数据集名称 |
| generationModel | string | 生成模型ID |
| dataAmount | number | 数据量 |
| selectedDatasets | SelectedDatasetType[] | 已选知识库列表 |
| keywords | string? | 关键词（数据场景） |
| collectionId | string? | 已有数据集ID（追加数据场景） |

### 使用示例

在数据集详情页追加数据时：
- `scene='data'`：不显示名称输入框
- `defaultValues={{ collectionId }}`：传递已有数据集ID
- 确认后 `onConfirm(data)` 刷新数据列表

在评测任务创建时：
- `scene='dataset'`：显示名称输入框
- `returnDatasetId=true`：onConfirm 同时返回 datasetId
- 创建完成后自动选中新数据集
