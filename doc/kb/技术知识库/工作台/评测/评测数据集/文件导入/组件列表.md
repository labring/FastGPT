## 说明

本模块只有单个页面组件 `FileImport`，其内部使用 Chakra UI 基础组件（`Box`, `Button`, `Flex`, `Input`, `VStack`, `IconButton`, `Switch`, `HStack`, `Progress`, `Text`, `Table`, `Thead`, `Tbody`, `Tr`, `Th`, `Td`, `TableContainer`）构建表单布局，不包含模块内共享的业务组件。

组件内部状态管理使用 React hooks（`useState`、`useCallback`、`useMemo`）和 `react-hook-form`（`useForm`），不依赖专门的状态管理 Store。全局数据（模型列表、系统配置）通过 `useSystemStore` 获取。
