# 评测维度 — 组件列表

> 评测维度为分组节点，下表列出所有子能力使用的组件。所有维度相关页面组件位于 `projects/app/src/pages/dashboard/evaluation/dimension/`，业务组件位于 `projects/app/src/pageComponents/dashboard/evaluation/dimension/`。

## 页面组件 (pages/)

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 |
|---------|------|-----------|---------|
| `projects/app/src/pages/dashboard/evaluation/dimension/index.tsx` | 评测维度列表页，作为评测首页 Tab 渲染，接收 `Tab` prop | 引用自 web → MyBox, MyIcon, MyIconButton, MyTag, Avatar, EmptyTip, useConfirm, useRequest | 否（被评测首页引用） |
| `projects/app/src/pages/dashboard/evaluation/dimension/create.tsx` | 创建自定义维度的独立页面 | 引用自 web → MyBox, MyIcon, useToast, useRequest；引用自 pageComponents → EditForm, TestRun, DashboardContainer | 否 |
| `projects/app/src/pages/dashboard/evaluation/dimension/edit.tsx` | 编辑自定义维度的独立页面，通过 query.id 加载数据 | 引用自 web → MyBox, MyIcon, Loading, useToast, useRequest；引用自 pageComponents → EditForm, TestRun, DashboardContainer | 否 |

## 业务组件 (pageComponents/)

| 组件路径 | 用途 | 跨模块依赖 | 是否共享 |
|---------|------|-----------|---------|
| `projects/app/src/pageComponents/dashboard/evaluation/dimension/EditForm.tsx` | 维度编辑/创建表单，含名称、描述、提示词三个字段，实时校验，内置模板引用入口 | 引用自 web → FormLabel, MyTextarea；引用自 pageComponents → CitationTemplate；引用自 react-hook-form → useForm | 是（被 create 和 edit 页面共用） |

| `projects/app/src/pageComponents/dashboard/evaluation/dimension/CitationTemplate.tsx` | 内置维度模板引用弹窗，左侧选择维度类型，右侧展示和编辑模板内容 | 引用自 web → MyModal；引用自 pageComponents → DIMENSION_LIST, getDimensionTemplates | 否（仅被 EditForm 使用） |
| `projects/app/src/pageComponents/dashboard/evaluation/dimension/AnswerTextarea.tsx` | 试运行弹窗中的带标签文本域组件，用于输入参考答案和实际回答 | 引用自 chakra-ui → Textarea, FormLabel | 否（仅被 TestRun 使用） |
