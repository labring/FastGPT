## 应用详情子模块 (pageComponents/app/detail/)

本模块按应用类型拆分编辑器组件，各子模块详细组件列表见子能力文档。核心结构如下：

### Edit/ 编辑器组件

| 子目录 | 说明 |
|--------|------|
| `Edit/SimpleApp/` | 简易应用编辑器（单轮对话型） |
| `Edit/ChatAgent/` | Chat Agent 编辑器（含知识库、工作流编排） |
| `Workflow/` | 工作流可视化编辑器 |
| `Plugin/` | 插件工具编辑器 |
| `Edit/MCPTools/` | MCP 工具集编辑器 |
| `Edit/HTTPTools/` | HTTP 工具集编辑器 |
| `SmartCustomerService/` | 智能客服（Assistant）编辑器 |

### 日志与看板

| 子目录 | 说明 |
|--------|------|
| `ConversationLogs/` | 对话日志查看组件（列表、详情、筛选、纠错） |
| `Logs/` | 应用日志查看组件 |
| `Dashboard/` | 应用数据看板组件 |
