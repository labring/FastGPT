# FastGPT 功能梳理

本文档目录用于汇总 FastGPT 当前项目中的主要产品功能点。梳理范围基于仓库中的页面路由、API 路由、产品文档目录、工作流节点模板、核心枚举与国际化文案，重点描述用户可见能力。

## 文档目录

- [功能清单](./00-feature-list.md)：按产品模块汇总所有主要功能点。
- [聊天与使用端](./01-chat.md)：用户使用应用进行对话、查看引用、管理历史与反馈的能力。
- [应用与工作台](./02-app-studio.md)：应用创建、应用管理、模板市场、版本与日志等能力。
- [工作流编排](./03-workflow.md)：Flow 可视化编排、节点体系、调试与运行能力。
- [知识库](./04-dataset.md)：知识库类型、导入、训练、检索、引用与权限能力。
- [工具、插件与 MCP](./05-tools-plugins-mcp.md)：系统工具、HTTP 工具、MCP 工具、插件市场与 MCP Server。
- [Agent Skill](./06-agent-skill.md)：Skill 创建、导入导出、部署、调试与权限管理。
- [发布、渠道与 OpenAPI](./07-publish-openapi.md)：分享、iframe、API Key、外部渠道和 OpenAPI。
- [模型与 AI 配置](./08-model-ai-config.md)：模型配置、模型渠道、模型测试、AI Proxy 与模型监控。
- [账号、团队与权限](./09-account-team-permission.md)：登录账号、团队组织、成员组、协作者与审计。
- [计费、用量与运营](./10-billing-operations.md)：套餐、积分、账单、发票、用量统计和运营数据。
- [自部署与系统管理](./11-self-host-admin.md)：部署、配置、升级、系统文件和后台初始化能力。

## 阅读建议

如果只是想快速了解 FastGPT 的功能范围，先看 [功能清单](./00-feature-list.md)。如果要做产品设计、权限改造或功能规划，再进入对应模块文档查看具体功能介绍。
