# 知识库功能介绍

知识库是 FastGPT RAG 能力的基础，负责文档接入、解析、分块、索引、检索、引用展示和权限管理。

## 知识库类型

FastGPT 支持多种知识库类型：

- 普通知识库：通过文件、文本、链接等方式导入资料并建立索引。
- Web 站点同步知识库：从网站链接同步内容，适合官网、文档站和帮助中心。
- 外部文件知识库：读取外部文件源，不一定由 FastGPT 托管原始文件。
- API 知识库：通过 API 管理和推送外部数据。
- 飞书知识库：接入飞书文档资料。
- 语雀知识库：接入语雀文档资料。
- 文件夹：用于组织知识库。

## 知识库管理

知识库支持创建、编辑、删除、列表、路径查看、详情查看、导出和文件夹组织。团队可以用知识库目录管理不同业务线的资料资产。

## 知识库权限

知识库支持协作者、读写管理权限、继承权限恢复和团队权限控制。权限能力用于保护企业资料，避免未经授权的成员查看或修改知识库。

## 集合管理

集合是知识库中的资料单元，类型包括文件、链接、外部文件、API 文件、图片集合、虚拟集合和文件夹。集合支持创建、更新、删除、同步、导出、路径查看和详情查看。

## 数据导入

知识库支持多种导入方式：

- 本地文件导入
- 文件链接导入
- 手动文本导入
- CSV / 模板导入
- 备份导入
- API 数据导入
- 图片导入
- 重新训练已有集合

## 文件解析

常见文件类型包括 txt、md、html、pdf、docx、pptx、csv、xlsx。系统还支持 PDF 增强解析、图片解析和文件预览分块。

## 数据处理模式

导入后可以选择不同处理模式：

- 分块模式：将文档拆成适合检索的 chunk。
- QA 模式：按问答对导入或生成问答结构。
- 图片解析模式：对图片内容进行理解并建立索引。
- 备份模式：用于备份数据恢复。
- 模板模式：按预设模板导入。

## 分块参数

知识库提供自动和自定义分块配置，包括 chunk 大小、最大 chunk 长度、强制分块、最小触发长度、自定义分隔符、段落深度和 LLM 段落拆分模式。

## 训练队列

资料导入后进入训练流程。系统支持查看训练队列、训练状态、训练详情、训练错误，更新训练数据、删除训练数据和重建向量索引。

## 检索模式

知识库支持向量检索、全文检索和混合检索。混合检索用于结合语义相似度和关键词匹配，提高不同类型问题下的召回效果。

## Rerank 重排

Rerank 用于对召回结果进行二次排序，提升最终送入模型的上下文质量，适合资料多、召回噪声高的场景。

## 知识库搜索测试

搜索测试允许在不进入完整应用的情况下验证知识库检索效果，包括召回内容、分数、引用来源和检索参数。

## 集合标签与元数据过滤

集合标签用于对资料进行业务分类。元数据过滤用于在检索时限定资料范围，例如按部门、产品、版本、地区或文档类型过滤。

## 引用与原文

知识库检索结果可以在聊天端展示引用，支持查看原文、查看集合引用、读取源内容和下载允许下载的来源文件。
