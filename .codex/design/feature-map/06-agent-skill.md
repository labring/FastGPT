# Agent Skill 功能介绍

Agent Skill 是 FastGPT 中用于封装专业能力、指令和运行资源的模块。它可以被 Agent 选择和调用，用于增强应用在特定任务上的表现。

## Skill 列表

Skill 列表支持查看、搜索和管理团队内 Skill。列表会展示 Skill 的基本信息、更新时间、创建者和关联应用数量。

## 创建 Skill

用户可以创建自定义 Skill，填写名称、简介、需求说明等信息。Skill 需求用于描述它应该解决什么任务、在什么条件下使用以及期望输出。

## 编辑 Skill

Skill 支持编辑基础信息和配置内容。编辑后可以保存并重新部署，使 Agent 使用最新能力。

## 复制与删除

Skill 支持复制和删除。复制用于基于已有 Skill 创建变体，删除需要考虑是否有关联应用，避免破坏线上应用。

## 文件夹管理

Skill 支持文件夹组织，便于团队按业务域、能力类型或维护人管理 Skill。

## 导入与导出

Skill 支持 zip 导入和导出，用于跨环境迁移、备份和团队间共享。

## 详情配置与预览

Skill 详情页包含配置和预览两个视图。配置用于维护 Skill 内容，预览用于检查 Skill 在实际运行中的表现。

## 版本管理

Skill 支持历史版本列表、版本更新和版本切换。版本能力用于控制 Skill 变更，避免直接影响线上应用。

## 保存部署

Skill 配置完成后需要保存部署，部署过程可能包含沙盒准备、文件上传、依赖下载、能力提取等步骤。

## 调试聊天

Skill 提供调试聊天能力，用户可以在不接入正式应用的情况下验证 Skill 的响应效果。

## 调试会话

调试会话支持列表、记录查看和删除，用于复盘 Skill 调试过程和清理无用数据。

## 权限管理

Skill 支持读、写、管理等权限配置，并可结合团队成员、成员组和协作者体系控制访问范围。

## 与应用关联

Skill 可以被应用或 Agent 选择使用。关联数量用于提示删除风险，也帮助团队理解某个 Skill 的影响范围。
