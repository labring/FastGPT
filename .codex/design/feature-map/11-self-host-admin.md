# 自部署与系统管理功能介绍

自部署与系统管理模块覆盖 FastGPT 在私有化环境中的部署、配置、升级、数据维护和系统级文件服务。

## Docker 部署

FastGPT 支持通过 Docker Compose 快速启动。部署配置包含主应用、MongoDB、PostgreSQL 或其他向量数据库、对象存储和相关服务。

## Sealos 部署

Sealos 部署用于在 Sealos Cloud 或 Kubernetes 环境中快速安装 FastGPT，适合云原生团队。

## 模型配置文件

自部署环境可通过配置文件维护模型列表、默认模型、模型供应商和价格信息，也可以通过界面可视化管理。

## 对象存储配置

对象存储用于保存上传文件、头像、知识库原始文件、沙盒文件和临时文件。系统支持通过配置接入不同对象存储服务。

## 环境变量与 config.json

环境变量和 config.json 控制数据库连接、服务地址、模型配置、对象存储、功能开关和系统初始化参数。

## 本地模型接入

文档覆盖 Ollama、Xinference、ChatGLM、M3E、BGE Rerank、MinerU、Marker 等本地模型或本地解析服务的接入方式。

## 数据库迁移与备份

系统提供 MongoDB、PostgreSQL 等数据迁移和备份文档，帮助用户进行环境迁移、灾备和版本升级前备份。

## 版本升级

版本升级文档记录不同版本的升级步骤、注意事项和初始化脚本。项目中也包含多个历史版本初始化 API。

## 初始化与数据修复

管理员接口包含初始化、反馈标记初始化、无效数据清理、应用登记创建等能力，用于版本迁移和数据维护。

## 系统文件服务

系统支持文件上传、下载、头像上传、临时文件上传、JWT 文件访问、图片代理和普通文件读取。

## 插件系统代理

系统插件代理用于将插件请求转发到插件服务或插件运行环境，支撑插件生态能力。

## 沙盒代理

沙盒代理负责代码沙箱的认证、CSP 密码代理和文件访问，支撑代码执行、文件生成和 HTML 预览。

## 埋点上报

系统提供 tracks push 接口，用于采集前端行为或运营事件，为产品分析提供数据来源。

## 系统初始化数据

初始化数据接口向前端提供系统配置、用户状态、功能开关、导航配置和其他启动所需信息。
