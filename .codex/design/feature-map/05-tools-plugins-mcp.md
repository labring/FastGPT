# 工具、插件与 MCP 功能介绍

工具、插件与 MCP 是 FastGPT 连接外部能力的扩展体系。它们让应用和 Agent 可以调用搜索、文件处理、HTTP API、MCP Server、系统插件和自定义工作流工具。

## 系统工具管理

系统工具由管理员维护，供团队或应用调用。管理能力包括工具创建、编辑、删除、详情查看、排序和标签管理。

## 工具市场

工具市场展示可安装和可使用的工具。团队可以从市场选择需要的插件或工具，并安装到团队空间中。

## 插件包管理

插件包支持上传、解析、确认安装和删除。通过插件包可以将一组工具、配置和元数据打包分发。

## URL 安装插件

管理员可以通过 URL 安装插件，适合从外部插件仓库或内部制品服务快速接入工具。

## 团队插件安装

团队可以安装或卸载插件。安装后，团队成员可在应用、工作流或 Agent 中选择这些工具。

## 自定义插件配置

自定义插件支持名称、简介、作者、头像、标签、用户指南、价格、Token 费用、关联插件和默认安装状态等配置。

## HTTP Tool

HTTP Tool 用于将外部 HTTP API 封装为工具。系统支持从 URL 获取 API Schema、创建工具、更新工具和运行工具。它适合对接 CRM、工单、搜索、订单、库存等业务系统。

## MCP Tool

MCP Tool 用于接入 MCP Server 暴露的工具。系统可以创建或更新 MCP 工具集、拉取工具列表、运行工具，并将 MCP 工具作为工作流或 Agent 的可调用能力。

## MCP Server 管理

MCP Server 管理能力包括创建、编辑、删除、列表、端点展示、JSON 配置、关联应用和工具调用说明。它用于把 FastGPT 应用或外部 MCP 服务组织成可复用的工具入口。

## 应用暴露为 MCP 服务

FastGPT 应用可以通过 MCP 端点暴露给外部 MCP Client 调用，使平台内应用能够成为其他系统中的工具。

## 工具调用成本提示

部分工具会产生调用成本，前端会通过成本提示帮助用户理解工具调用可能消耗的积分或费用。
