# 应用与工作台功能介绍

应用与工作台是 FastGPT 的核心生产区，用户在这里创建、配置、调试、发布和运营 AI 应用。

## 应用类型

FastGPT 支持多种应用形态：

- 简单聊天应用：面向单轮或多轮对话配置，适合快速搭建问答助手。
- Chat Agent / 规划 Agent：通过 Agent 规划、工具调用和多步推理完成复杂任务。
- 工作流应用：使用 Flow 节点编排实现确定性的业务流程。
- 工作流工具 / 插件应用：将工作流封装成可被其他应用或 Agent 调用的工具。
- HTTP 工具集：将外部 HTTP API 封装为可调用工具。
- MCP 工具集：将 MCP Server 暴露的工具接入应用。

## 应用创建

应用可以从空白配置、模板、复制现有应用或导入配置创建。创建时会根据类型初始化不同的配置结构和工作流模板。

## 应用列表与文件夹

工作台提供应用列表和文件夹管理能力。用户可以搜索、移动、复制、删除应用，也可以用文件夹组织不同业务线、部门或场景下的应用。

## 应用基础信息

应用支持编辑名称、头像、简介、标签等基础信息。这些信息会影响工作台展示、聊天入口展示、发布渠道展示和团队成员识别。

## 应用权限

应用权限覆盖协作者、读写管理权限、所有者和继承权限。它用于控制团队成员对应用的查看、编辑、发布和管理能力。

## 应用配置

应用配置包括 AI 模型参数、系统提示词、欢迎语、文件上传、问题引导、语音输入输出、自动执行、变量配置和定时触发等。

## 模板市场

模板市场提供可复用的应用模板，默认分类包括推荐、写作、图像生成、联网搜索、角色扮演和办公服务。用户可以从模板快速创建应用，也可以在商业后台维护模板分类。

## 应用版本

应用版本用于保存和发布稳定配置。版本能力包括发布、更新、查看历史版本、获取最新版本和回滚相关配置，适合生产应用的变更管理。

## 应用导入导出

导入导出用于跨环境迁移应用配置、分享模板或备份复杂工作流。导出时需要过滤不适合外部传播的运行数据和敏感配置。

## 应用调试

应用调试允许用户在发布前直接测试应用效果。对于工作流应用，调试会展示节点运行状态、输入输出和错误信息。

## 应用运行日志

运行日志用于追踪真实用户调用。日志支持字段配置、用户筛选、IP 类型筛选、错误筛选、反馈筛选、图表统计和导出。

## 应用评测

应用评测通过上传评测集、选择被测应用和评测模型，对应用回答进行批量评估。评测结果包含任务进度、问题、标准答案、应用回答、评分和导出能力。
