# 发布、渠道与 OpenAPI 功能介绍

发布能力负责将 FastGPT 应用提供给内部用户、外部用户和第三方系统。OpenAPI 则提供标准化程序调用入口。

## 分享链接

分享链接用于生成免登录访问入口。它适合对外演示、客户问答、临时体验和轻量级外部访问。

## iframe 嵌入

iframe 嵌入用于将应用对话窗口放入第三方网页、官网、文档站或内部系统页面。

## API Key 发布

API Key 发布用于通过接口调用应用。它可以配置访问限制，并用于服务端集成、自动化任务和第三方系统调用。

## Playground 发布

Playground 发布提供可配置的在线体验入口，适合应用展示、内部测试和产品演示。

## 飞书机器人

飞书机器人渠道用于将 FastGPT 应用接入飞书聊天，支持通过飞书消息触发应用回复。

## 钉钉机器人

钉钉机器人渠道用于将应用接入钉钉群或钉钉机器人场景，服务企业协同流程。

## 企业微信机器人

企业微信机器人渠道用于企业微信内的应用问答和流程触发。

## 微信客服

微信客服渠道支持扫码登录、状态检查、重新登录、退出登录等能力，用于接入微信客服消息场景。

## 公众号

公众号渠道用于把 FastGPT 应用接入微信公众号，处理公众号用户消息。

## 发布渠道管理

发布渠道支持创建、编辑、删除和列表查看。渠道配置包含名称、访问参数、密钥、限制条件和前端展示选项。

## 访问限制

发布渠道支持 QPM、Token 鉴权、IP 白名单等限制，用于保护应用免受滥用并控制调用成本。

## 展示控制

发布配置可控制是否展示节点运行状态、引用、Skill Reference、完整文本和来源下载能力。

## 自定义域名

自定义域名用于将分享或发布入口绑定到团队自己的域名。系统支持域名解析、DNS 记录、验证和删除。

## OpenAI 兼容 Chat Completions

FastGPT 提供 OpenAI 兼容的 Chat Completions 接口，包括 v1 和 v2，便于外部系统以标准协议调用应用。

## 音频转录接口

音频转录接口用于将音频转换为文本，支撑语音输入和外部语音处理场景。

## Chat 停止接口

停止接口用于终止正在运行的聊天任务，适合前端停止生成和外部系统取消任务。

## 应用 OpenAPI

应用 OpenAPI 支持应用的创建、查询、更新、删除等能力，方便外部系统自动化管理应用。

## 对话 OpenAPI

对话 OpenAPI 支持发起对话、读取历史、管理会话等能力，用于将 FastGPT 集成到外部产品中。

## 知识库 OpenAPI

知识库 OpenAPI 支持知识库、集合和数据的 CRUD，适合从外部数据源自动同步资料。

## 分享链接 OpenAPI

分享链接 OpenAPI 用于管理和调用分享渠道，支持外部应用围绕分享入口做集成。

## API Key 管理

API Key 支持创建、更新、删除、健康检查和列表查询，是外部调用鉴权和权限控制的基础。

## OpenAPI 文档生成

系统提供 OpenAPI JSON 文档，方便生成 SDK、接口文档和自动化测试。
