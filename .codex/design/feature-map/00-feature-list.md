# FastGPT 功能清单

## 1. 聊天与使用端

- 应用聊天窗口
- 团队应用入口 / 应用树
- 最近使用应用、收藏应用、快捷应用
- 聊天历史列表、分页、重命名、删除、批量删除、清空
- 对话恢复 / 继续运行
- 流式回复、停止生成
- 引用展示：知识库引用、集合引用、原文查看、下载分块
- 运行日志查看：节点输入输出、请求体、Token、错误信息、工具输出
- 用户反馈、管理员反馈处理、自定义反馈关闭
- 文件输入：文档、图片、音频、视频、自定义扩展名
- 语音输入、语音转文字、TTS 语音回复
- 输入引导词 / 模糊输入提示
- Markdown 渲染：图片、音频、视频、iframe、ECharts、Mermaid、代码块
- 沙盒文件预览和编辑入口
- Helper Bot 辅助聊天

## 2. 应用与工作台

- 创建应用
- 简单聊天应用
- Chat Agent / 规划 Agent
- 工作流应用
- 工作流工具 / 插件应用
- HTTP 工具集
- MCP 工具集
- 应用文件夹管理
- 应用列表、搜索、复制、删除、导入、导出
- 应用基础信息编辑：名称、头像、简介、标签
- 应用权限、协作者、继承权限恢复
- 应用版本管理：发布、更新、查看历史版本、latest 版本
- 模板创建应用
- 模板市场：推荐、写作、图像生成、联网搜索、角色扮演、办公服务等分类
- 应用调试 / Chat Test
- 应用运行日志、日志字段配置、用户筛选、错误筛选、反馈筛选、日志导出
- 应用评测：创建任务、上传评测集、选择应用和评测模型、查看进度、详情、导出

## 4. 知识库

- 知识库列表、详情、创建、编辑、删除

- 普通知识库

- API 知识库
- 飞书知识库
- 语雀知识库
- 知识库权限、协作者、继承权限恢复
- 知识库导出
- 集合管理：文件、链接、外部文件、API 文件、图片集合、虚拟集合、文件夹
- 集合创建、更新、删除、同步、导出、路径查看
- 本地文件导入、URL 导入、手动文本导入、CSV/模板导入、备份导入、图片导入
- 支持 txt、md、html、pdf、docx、pptx、csv、xlsx 等文档解析
- PDF 增强解析
- 图片解析 / 图片索引

- Rerank 重排
- 集合标签
- 元数据过滤
- 原文查看、引用数据查询
