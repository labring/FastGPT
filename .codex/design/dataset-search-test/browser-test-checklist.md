# 知识库详情页浏览器测试清单

日期：2026-05-14

## 测试说明

本清单用于验证知识库详情页本轮改动，主要包含：

- `projects/app/src/pageComponents/dataset/detail/Test/` 拆分后的知识库搜索测试面板功能。
- `projects/app/src/pageComponents/dataset/detail/components/InputDataModal/` 拆分后的数据编辑弹窗功能。

重点覆盖：

- 搜索测试基础渲染与输入体验。
- 搜索配置弹窗参数读写。
- 搜索请求、结果展示、历史记录交互。
- 图片入口隐藏后的请求行为。
- 知识库切换时的结果隔离。
- 数据编辑弹窗的加载、只读、保存、索引增删改体验。
- 桌面和窄屏布局稳定性。

测试前建议准备：

- 至少 1 个有可检索数据的知识库，并包含 1 个可编辑数据集。
- 可选：第 2 个知识库，用于验证切换隔离。
- 可选：准备一个只读权限账号或共享只读知识库，用于验证只读态。
- 打开浏览器 DevTools，观察 Console 与 Network。

## 浏览器测试清单

### 1. 基础渲染

- [ ] 进入知识库详情页，切换到“搜索测试”。
- [ ] 左侧显示“输入测试内容”、搜索配置按钮、输入框、测试按钮。
- [ ] 右侧默认显示空结果提示。
- [ ] 图片上传入口不可见。
- [ ] 输入框文字颜色为正常输入态，不是浅灰提示态。
- [ ] 输入框字号与普通输入体验一致。
- [ ] Textarea 内容区域没有额外上下 padding。

### 2. 提交状态

- [ ] 空输入时，“测试”按钮不可点击。
- [ ] 输入普通文本后，“测试”按钮可点击。
- [ ] 点击“测试”后按钮进入 loading。
- [ ] 搜索无结果时显示 warning toast。
- [ ] 搜索无结果时不新增历史记录。
- [ ] 搜索有结果时展示搜索参数、耗时和引用结果。

### 3. 搜索配置

- [ ] 点击“搜索配置”可以打开配置弹窗。
- [ ] 默认“结果重排”关闭。
- [ ] 调整召回模式后点击完成，配置生效。
- [ ] 调整引用 token 后点击完成，配置生效。
- [ ] 调整相似度后点击完成，配置生效。
- [ ] 再次打开配置弹窗，确认上次配置仍保留。
- [ ] 执行搜索后，结果区展示的搜索参数与配置一致。

### 4. 历史记录

- [ ] 成功搜索一次后，左侧历史新增一条。
- [ ] 历史记录显示输入文本和时间。
- [ ] 点击历史记录，右侧结果切换为该条历史结果。
- [ ] 删除未选中的历史记录，当前结果不变化。
- [ ] 删除当前选中的历史记录，右侧回到空结果态。
- [ ] 连续搜索多次，新记录显示在最上方。

### 5. 知识库切换

- [ ] 在知识库 A 搜索成功，确认右侧展示结果。
- [ ] 切换到知识库 B，确认不会展示知识库 A 的选中结果。
- [ ] 如果知识库 B 没有历史记录，右侧应为空结果态。
- [ ] 切回知识库 A，记录当前行为：当前实现会恢复之前选中的 A 结果。

### 6. 布局与响应式

- [ ] 桌面宽度下左侧宽度固定，右侧结果区占满剩余空间。
- [ ] 右侧结果较多时滚动正常。
- [ ] 左侧历史较多时滚动正常。
- [ ] 窄屏下页面不出现明显横向溢出。
- [ ] 长文本输入不会撑坏布局。
- [ ] 长历史文本能换行，不遮挡时间和删除按钮。

### 7. DevTools 回归确认

- [ ] Console 无新增报错。
- [ ] `/api/core/dataset/searchTest` 请求体中 `queryImageUrls` 为空数组。
- [ ] `/api/core/dataset/searchTest` 请求体中默认 `usingReRank` 为 `false`。
- [ ] 搜索配置变更后，请求体参数与弹窗配置一致。

### 8. 数据编辑弹窗基础渲染

- [ ] 进入知识库详情页，切换到“数据”列表。
- [ ] 点击已有数据卡片，打开数据编辑弹窗。
- [ ] 弹窗 loading 期间只显示加载态，不提前露出 body 内容。
- [ ] loading 完成后，弹窗标题显示来源名称；无来源名称时显示默认未知来源文案。
- [ ] 非图片集合显示 Chunk/QA tab，图片集合直接进入图片描述编辑态。
- [ ] 左侧数据输入区与右侧索引编辑区并排展示，右侧宽度保持固定 `sm` 体验。
- [ ] 弹窗内输入框字号为默认正常大小，不是 12px 小字。
- [ ] 索引卡片有边框，删除 loading 时卡片高度不明显缩小。

### 9. 数据内容编辑

- [ ] 修改 Chunk 内容后点击“生成索引”，按钮进入 loading。
- [ ] 更新成功后弹出成功提示，数据列表对应卡片内容同步更新。
- [ ] QA tab 中分别修改问题和答案后保存，重新打开弹窗确认内容保留。
- [ ] 图片集合中图片预览正常展示，图片描述修改后保存成功。
- [ ] 保存失败时弹出错误提示，弹窗内容不被错误覆盖或清空。
- [ ] 关闭弹窗再重新打开同一条数据，不残留上一次其他数据的编辑状态。

### 10. 自定义索引新增

- [ ] 点击右侧“新增”按钮后，新的自定义索引立即出现在列表顶部，不等待重新拉取数据。
- [ ] 第一个自定义索引新增后，输入框自动聚焦。
- [ ] 新增空白索引后直接失焦，空白草稿静默移除，不弹成功或错误提示。
- [ ] 输入索引文本后失焦，调用 `/api/core/dataset/data/index/create`。
- [ ] 新增成功后不闪烁、不消失重现，索引仍保持当前可见位置。
- [ ] 新增成功后重新打开弹窗，索引存在且拥有后端 `indexDataId`。
- [ ] 新增接口失败时，草稿内容仍保留在当前弹窗中，用户可以继续修改或重试。

### 11. 自定义索引编辑

- [ ] 点击已存在的自定义索引折叠遮罩，输入框展开并聚焦。
- [ ] 修改索引文本后失焦，调用 `/api/core/dataset/data/index/update`。
- [ ] 未修改内容直接失焦，不触发 update 请求。
- [ ] 快速连续修改同一个索引并多次失焦，最终后端保存的是最后一次输入内容。
- [ ] 保存中继续编辑同一个索引，保存完成后会继续补交最新内容，不出现 UI 与后端内容不一致。
- [ ] 保存失败时弹出错误提示，索引文本保留用户当前输入。
- [ ] 默认索引仅展示文本，不允许编辑、不允许单独保存、不显示删除按钮。

### 12. 自定义索引删除

- [ ] 已存在的自定义索引 hover 后显示删除按钮。
- [ ] 点击删除按钮后，整个索引卡片进入 loading 态。
- [ ] 删除成功后索引从列表移除，并弹出删除成功提示。
- [ ] 删除失败时索引仍保留，loading 结束并显示错误提示。
- [ ] 正在编辑索引时直接点击删除，不触发一次额外 update 后再重复出现索引。
- [ ] 保存请求仍在进行时点击删除，最终结果应为索引被删除，不出现 `Dataset data index not found`。
- [ ] 没有 `indexDataId` 的本地草稿索引不显示删除按钮。

### 13. 弹窗权限只读态

- [ ] 使用只读权限进入数据列表，点击数据卡片仍可打开弹窗。
- [ ] 左侧 Chunk/QA/图片描述输入框为只读态，不能修改内容。
- [ ] “生成索引”按钮禁用，不能提交更新。
- [ ] 右侧“新增”按钮禁用。
- [ ] 自定义索引输入框不可编辑，不会触发 create/update 请求。
- [ ] 自定义索引不显示删除按钮，不会触发 delete 请求。
- [ ] 只读态仍可查看索引内容和展开/折叠已有索引。

### 14. InputDataModal DevTools 回归确认

- [ ] Console 无新增报错。
- [ ] 新增索引请求体包含数据 ID、索引类型和 trim 后文本，不包含 `clientId`、`fold` 等前端字段。
- [ ] 更新索引请求体包含 `dataId` 和 `indexDataId`，`indexDataId` 不为空。
- [ ] 删除索引请求体包含 `dataId` 和 `indexDataId`，`indexDataId` 不为空。
- [ ] 快速连续操作索引时，Network 中没有使用空 `indexDataId` 调用 update/delete。
- [ ] 只读态操作弹窗时，不产生 create/update/delete 网络请求。
