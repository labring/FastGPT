1. # Open Microsoft Azure Portal and Log In

https://portal.azure.com/

2. # Create Bing Web Search Resource

https://portal.azure.com/#create/Microsoft.BingSearch

![](/imgs/bing_search_plugin2.png)

3. # Go to Resource Details and Click Manage Keys

![](/imgs/bing_search_plugin3.png)
