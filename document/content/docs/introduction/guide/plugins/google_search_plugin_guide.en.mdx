## 1. Create a Google Custom Search Engine

Visit https://programmablesearchengine.google.com/

Go to the Custom Search Engine control panel and create a new Search Engine.

![](/imgs/google_search_plugin1.png)

Get your search engine ID (cx):

![](/imgs/google_search_plugin2.png)

## 2. Get an API Key

Visit https://developers.google.com/custom-search/v1/overview?hl=zh-cn

![](/imgs/google_search_plugin3.png)

## 3. Fill in Plugin Parameters

Enter your search engine ID in the cx field and your API key in the key field:

![](/imgs/google_search_plugin4.png)
