## Features

- Can be added multiple times (prevents messy lines in complex workflows, looks cleaner)
- Manual input supported
- External input supported
- Outputs results to client

The specified reply module is typically used for special status replies. Reply content has two types:

1. Manually input fixed content.
2. Variable references.
