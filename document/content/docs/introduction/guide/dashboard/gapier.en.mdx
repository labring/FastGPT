## 3. Tool Calling Instructions

### Differences Between Models

Different models use different methods to call tools. Some models support toolChoice and functionCall for better results. Models that don't support these two methods use prompts for calling, but the results aren't great. To ensure smooth calling, FastGPT's built-in prompts only support calling one tool at a time.

Check the official website to see which models support functionCall (OneAPI support is also required), and adjust the corresponding fields in the model configuration file (see configuration field descriptions for details).

Online version users can see whether function calling is supported when selecting models.
