### 1. Start Deployment

Since databases need to be deployed, wait 2â€“4 minutes after deployment before accessing. The default configuration uses minimal resources, so the first access may be slow.

Follow the prompts to enter `root_password` and the `openai`/`oneapi` address and key.

![](/imgs/sealos1.png)

After clicking deploy, you'll be redirected to the app management page. Click the details button on the right side of the `fastgpt` main app (named fastgpt-xxxx), as shown below.

![](/imgs/sealos-deploy1.jpg)

After clicking details, you'll go to the fastgpt deployment management page. Click the link in the external access address to open the fastgpt service.

To bind a custom domain or modify deployment parameters, click **Change** in the top right corner and follow Sealos' instructions.

![](/imgs/sealos2.png)

### 2. Log In

Username: `root`

Password: the `root_password` you set during one-click deployment

### 3. Configure Models

You must configure at least one model set, or the system won't work properly.

## Using Sealos

### Overview

FastGPT commercial version includes 2 apps (fastgpt, fastgpt-plus) and 2 databases. When using multiple API keys, you need to install OneAPI (1 app and 1 database), totaling 3 apps and 3 databases.

![](/imgs/onSealos1.png)

Click details on the right to view detailed information for each app.

### Modifying Config Files and Environment Variables

In Sealos, open **App Launchpad** to see deployed FastGPT apps, and open **Database** to see corresponding databases.

In **App Launchpad**, select FastGPT, click **Change**, and you'll see the corresponding environment variables and config files.

![](/imgs/fastgptonsealos1.png)

### How to Update/Upgrade FastGPT

[Upgrade script documentation](./upgrading/) â€” read the docs first to see which version you need to upgrade to. Note: don't skip versions!

For example, if you're on version 4.5 and want to upgrade to 4.5.1, change the image version to v4.5.1, run the upgrade script, wait for completion, then continue upgrading. If the target version doesn't require initialization, you can skip it.

Upgrade steps:

1. Check the [update documentation](./upgrading/index/) to confirm the version you're upgrading to and avoid skipping versions.
2. Open Sealos app management
3. There are 2 apps: fastgpt, fastgpt-pro
4. Click the 3 dots on the right side of the corresponding app, then **Change**. Or click details, then **Change** in the top right.
5. Modify the image version number

![](/imgs/onsealos2.png)

6. Click **Change/Restart** to automatically pull the latest image and update
7. Run the initialization script for the corresponding version (if applicable)

### Modify Site Name and Favicon

Modify the app's environment variables and add:

```
SYSTEM_NAME=FastGPT
SYSTEM_DESCRIPTION=
SYSTEM_FAVICON=/favicon.ico
HOME_URL=/dashboard/agent
```

SYSTEM_FAVICON can be a URL.

![](/imgs/onsealos6.png)

### Mount a Logo

Currently, you can't replace the browser logo. Only SVG is supported. Full replacement will be available after visual customization is implemented.

Add a mounted file with filename: `/app/projects/app/public/icon/logo.svg`, and the value should be the SVG content.
