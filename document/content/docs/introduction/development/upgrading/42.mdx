99.9%用户不影响，升级 4.2 主要是修改了配置文件中 QAModel 的格式。从原先的数组改成对象：

```json
"QAModel": {
    "model": "gpt-3.5-turbo-16k",
    "name": "GPT35-16k",
    "maxToken": 16000,
    "price": 0
}
```

改动目的是，我们认为不需要留有选择余地，选择一个最合适的模型去进行任务即可。
