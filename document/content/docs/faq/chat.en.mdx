## I modified the app in the workspace, why doesn't the "Chat" update the configuration?

The app needs to be published before the chat will update.

## Browser doesn't support voice input

1. First, ensure that the browser and computer microphone permissions are enabled.
2. Confirm that the browser allows the site to use the microphone and select the correct microphone source.
3. Only sites with SSL certificates can use the microphone.
