In FastGPT, you can create multiple API keys for each app to access the app's API endpoints. Each key can only access one app. For complete endpoint documentation, [see the app chat endpoint](/docs/openapi/intro).

## Get API Key

Select App -> "API Access", then click "API Key" to create a key.

## Replace Variables in Third-Party Apps

```bash
OPENAI_API_BASE_URL: http://localhost:3000/api (change to your deployed domain)
OPENAI_API_KEY = the key obtained in the previous step
```
