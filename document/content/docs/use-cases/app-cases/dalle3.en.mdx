## OpenAI Dalle3 API

First, let's look at the official API parameters and response:

Body

```json
{
    "model": "dall-e-3",
    "prompt": "A cute baby sea otter",
    "n": 1,
    "size": "1024x1024"
}
```

Response 

```json
{
  "created": 1589478378,
  "data": [
    {
      "url": "https://..."
    },
    {
      "url": "https://..."
    }
  ]
}
```

## Workflow Design

1. Use AI to optimize the image generation prompt (this step is skipped—find your own prompt)
2. Use the `【HTTP Request】module` to call the Dalle3 API and get the image URL.
3. Use the `【Text Processing】module` to build the `Markdown` image format.
4. Use the `【Specified Reply】module` to directly output the image link.

### 1. Build HTTP Module

Copy the Dalle3 API request parameters directly and change prompt to a variable. Add a `Headers.Authorization`.

Body:

```json
{
    "model": "dall-e-3",
    "prompt": "{{prompt}}",
    "n": 1,
    "size": "1024x1024"
}
```

Headers:

`Authorization: Bearer sk-xxx`

Response:

The response needs to extract values from the Dalle3 API return. Since we're only generating 1 image, we only need the first image's URL. Add a custom output `data[0].url` to the HTTP module.

### 2. Text Processing - Build Image Link

In `Markdown` syntax, `![image description](image link)` inserts an image. The image link is output by the【HTTP Request】module.

Therefore, add an input to receive the image link output from the `【HTTP Request】module`, and reference the image link through a variable in the `【Text Processing】module - Text` to get a complete `Markdown` image format.

### 3. Specified Reply

Specified Reply can directly output the passed content to the client, so you can directly output the processed `Markdown` image format.
