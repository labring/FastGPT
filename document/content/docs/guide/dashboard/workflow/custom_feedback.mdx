## 介绍

自定义反馈模块，可以为你的对话增加一个反馈标记，从而方便在后台更好的分析对话的数据。

在调试模式下，不会记录反馈内容，而是直接提示: `自动反馈测试: 反馈内容`。

在对话模式（对话、分享窗口、带 chatId 的 API 调用）时，会将反馈内容记录到对话日志中。（会延迟60s记录）

## 作用

自定义反馈模块的功能类似于程序开发的`埋点`，便于你观测的对话中的数据。
